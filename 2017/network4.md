# 雲端運算三強：AWS、Azure、GCP 競賽

## 市場格局

2017 年，雲端運算市場形成了穩定的三強格局：Amazon Web Services（AWS）、Microsoft Azure 和 Google Cloud Platform（GCP）。根據 Synergy Research 的數據，2017 年雲端基礎設施服務市場規模超過 400 億美元，其中 AWS 佔約 40%，Azure 約 20%，GCP 約 10%。

## AWS 的持續主導

AWS 在 2017 年推出了數百項新服務和功能。最重要的包括：

- **AWS Fargate**：無伺服器容器服務，讓使用者不再需要管理基礎設施
- **S3 Select** 和 **Glacier Select**：從儲存中檢索子集資料
- **AWS Neptune**：完全受管的圖資料庫服務
- **Amazon Sumerian**：VR/AR 應用開發工具

## Azure 的快速追趕

Microsoft Azure 在 2017 年展現了最強的成長動能。Nadella 的「雲端優先」策略開始見效：

- **Azure Stack**：將 Azure 服務延伸到本地資料中心
- **Azure Functions**：無伺服器運算服務，與 AWS Lambda 競爭
- **SQL Database 的改進**：受管執行個體和 GPU 支援

## GCP 的差異化

Google Cloud 以資料分析和機器學習作為差異化優勢：

- **Cloud TPU**：專為 TensorFlow 優化的硬體加速器
- **BigQuery**：持續強化無伺服器資料倉儲
- **Kubernetes Engine（GKE）**：利用 Google 在容器編排的領導地位
