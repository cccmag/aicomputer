# iPhone X：Face ID 與全螢幕時代

## iPhone 十週年之作

2017 年 9 月 12 日，Apple 在全新的 Steve Jobs Theater 發表了 iPhone X——一款為 iPhone 十週年而設計的旗艦手機。iPhone X 在設計上與所有前代產品徹底不同：它取消了 Home 按鈕，採用幾乎無邊框的全螢幕設計，並以 Face ID 取代了 Touch ID。

iPhone X 配備 5.8 吋 Super Retina OLED 顯示器（由 Samsung 供應），解析度 2436×1125，支援 HDR 和 True Tone。它搭載了 A11 Bionic 晶片，內建神經網路引擎（Neural Engine），專門用於機器學習任務。

## Face ID：生物辨識的轉折

Face ID 是 iPhone X 最重要的創新。它使用 TrueDepth 攝影機系統——由紅外線鏡頭、泛光照射器和點陣投影器組成——向使用者臉部投射超過 30,000 個不可見的紅外線點，建立精確的臉部深度圖。

Face ID 的安全性設計：

- 資料只在裝置端的 Secure Enclave 中處理
- 使用神經網路進行臉部比對
- 要求使用者睜眼注視（可設定關閉）
- 適應外觀變化（鬍子、眼鏡、帽子）

## 影響

iPhone X 的 999 美元定價使 iPhone 首次突破千元門檻。其設計語言——瀏海（notch）、OLED 面板、不鏽鋼邊框、玻璃背板——在接下來數年被大量 Android 手機廠商模仿。

iPhone X 證明了生物辨識可以完全擺脫指紋，對整個產業的設計方向產生了深遠影響。
