# 歷史穿梭：2017 年的種子

## Transformer 的深遠影響

2017 年的 Transformer 論文徹底改變了 NLP 的發展軌跡：

**Transformer (2017)** → **BERT (2018)** → **GPT-2 (2019)** → **GPT-3 (2020)** → **ChatGPT (2022)**

Transformer 捨棄了遞迴和卷積，僅用注意力機制就達到了超越所有先前模型的效能。這個架構的巨大可擴展性（scalability）讓後續的語言模型可以透過增加參數量和訓練資料量來不斷提升能力。

## RISC 的現代遺產

Hennessy 和 Patterson 在 1980 年代奠定的 RISC 原則，在 2017 年仍然主導著處理器設計：

**MIPS (1980s)** → **ARM (1990s)** → **RISC-V (2010s)** → **Apple Silicon (2020)**

2017 年，ARM 處理器已經成為行動裝置的絕對主流，而開源的 RISC-V 架構開始嶄露頭角。Apple 在 2020 年從 Intel x86 轉向自研 ARM 晶片——這是 RISC 理念的終極勝利。

## 從 Ryzen 到 CPU 競爭的復活

AMD Ryzen 的成功推倒了骨牌效應：

**Ryzen (2017)** → **Threadripper** → **EPYC 伺服器處理器** → **AMD 市佔率持續成長**

Ryzen 迫使 Intel 從「每年小幅升級」轉向「核心戰爭」——推出了 Core i9、增加了核心數、降低了價格。這種競爭最終讓消費者獲得了更好的性價比。

## 從 Ethereum 到 DeFi

2017 年的 ICO 狂潮雖然在 2018 年泡沫破裂，但它建立的基礎設施為後續的 DeFi（去中心化金融）奠定了基礎：

**Ethereum 智能合約 (2017)** → **DeFi Summer (2020)** → **NFT 熱潮 (2021)** → **Layer 2 擴展**

## 2017 年的時代脈絡

- **AI 的硬體加速**：NVIDIA Volta 的 Tensor Core 將 GPU 從圖形處理器轉變為 AI 加速器
- **雲端原生標準化**：Kubernetes 成為基礎設施的通用層——「雲端作業系統」
- **主機遊戲的創新**：Nintendo Switch 證明了掌機與家用主機的混合設計可以成功
- **密碼貨幣泡沫**：比特幣在 2017 年底達到近 20000 美元的歷史高點，吸引了大量散戶和媒體關注

2017 年最關鍵的遺產是 Transformer——一篇論文定義了一個時代。
