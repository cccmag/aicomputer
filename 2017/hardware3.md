# Intel Core i9：高效能桌機的新標竿

## Skylake-X 與 HEDT 平台

2017 年 6 月，Intel 推出了 Core i9 系列處理器，作為其高效能桌上型（HEDT）平台 Skylake-X 的一部分。Core i9 的推出是為了回應 AMD Ryzen 的競爭壓力——Intel 需要一個比 Core i7 更高端的產品線來維護其在頂級市場的地位。

Core i9-7900X（10 核心 20 執行緒，739 美元）是首款 Core i9 型號。隨後 Intel 還推出了更高核心數的型號：i9-7920X（12 核）、i9-7940X（14 核）、i9-7960X（16 核）和頂級的 i9-7980XE（18 核 36 執行緒，1999 美元）。

## Turbo Boost Max 3.0

Core i9 引入了 Intel Turbo Boost Max 3.0 技術——能夠識別晶片上最快的兩個核心，並將單執行緒工作負載優先分配給它們。這項技術依賴於在製造過程中對每個核心進行速度分級。

## 競賽的復活

Core i9 的推出標誌著 CPU 戰爭的全面復活。2017 年之前，Intel 在高效能市場幾乎沒有競爭壓力，產品更新緩慢、價格高昂。Ryzen 的出現迫使 Intel 不僅加速了產品迭代，還在核心數和價格上展開了激烈競爭——這對消費者來說無疑是好事。
