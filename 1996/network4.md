# 第一個手機上網：行動網路萌芽

## Nokia 9000 Communicator

1996 年，Nokia 發表了 9000 Communicator——雖然它主要是一款硬體產品（已於 hardware1 詳述），但它的網路功能同樣具有里程碑意義。

Nokia 9000 是世界上第一款可以真正連接到網際網路的手機。它透過 GSM 網路的電路交換數據（CSD）技術，以 9.6 kbps 的速度撥接上網。

## 手機上網的體驗

在 Nokia 9000 上網的體驗與今天截然不同：

- **速度**：9.6 kbps——下載一個 30 KB 的網頁需要約 25 秒
- **螢幕**：單色 LCD，640x200 像素
- **瀏覽器**：只支援基本的 HTML，沒有 CSS 和 JavaScript
- **費用**：按連線時間計費，極其昂貴
- **應用**：收發郵件、瀏覽新聞頭條、查看股市行情

## WAP 之前的時代

Nokia 9000 使用的是簡化的 HTML 渲染，不需要專用的 WAP 協定。而真正的手機上網標準——WAP（Wireless Application Protocol）——要到 1999 年才推出。

雖然 Nokia 9000 的上網體驗對一般消費者來說並不實用，但它證明了幾個重要的概念：
- 手機可以成為上網裝置
- 行動工作者可以在路上收發郵件
- 網際網路不一定要透過桌上型電腦
