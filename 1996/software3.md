# IE 3.0：微軟正式加入瀏覽器戰爭

## 從無到有

1996 年 8 月 13 日，Microsoft 發布了 Internet Explorer 3.0。與 IE 1.0（1995 年 8 月）和 IE 2.0（1995 年 11 月）的簡陋不同，IE 3.0 是第一款真正具有競爭力的 IE 版本。

IE 1.0 只是一個從 Spyglass Mosaic 授權而來的修改版本，功能極為有限。IE 2.0 稍有改善但仍然無法與 Netscape Navigator 競爭。IE 3.0 則是一次徹底的重寫。

## 關鍵功能

IE 3.0 引入了多項重要功能：
- **CSS 支援**：對 CSS Level 1 的首次實作（雖然不完整）
- **JavaScript 支援**：微軟實作了自己的 JScript——與 Netscape 的 JavaScript 相容但略有差異
- **ActiveX 控制項**：允許開發者建立類似於 Netscape 外掛的瀏覽器擴展
- **VBScript 支援**：Visual Basic Scripting Edition，僅在 IE 中可用
- **Internet Mail and News**：內建的電子郵件和新聞群組用戶端
- **安全區域（Security Zones）**：不同等級的安全策略設定

## 瀏覽器戰爭的轉折點

IE 3.0 的市場策略包括兩個關鍵要素：
- **免費**：Netscape Navigator 3.0 售價 49 美元，IE 3.0 免費
- **預裝**：IE 3.0 與 Windows 95 的 OEM 版本捆綁銷售

免費 + 預裝的組合讓 IE 的市場佔有率從 1996 年初的個位數成長到 1997 年的 20% 以上。Netscape 在 1998 年開始失去市場主導地位。
