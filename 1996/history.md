# 歷史穿梭：1996 年的種子

## 從瀏覽器大戰到平台壟斷

1996 年瀏覽器戰爭正式開打。Netscape Navigator 2.0 在 1 月推出，內建 JavaScript、框架（frames）和 cookies。微軟在 8 月推出 IE 3.0，首次支援 CSS、ActiveX 控制項和 JavaScript（其 JScript 實作）。

**Netscape (1994-1998)** → **IE 3.0 (1996)** → **IE 4.0 (1997)** → **IE 6 (2001)** → **Chrome (2008)**

瀏覽器戰爭的結果是 Netscape 的失敗和微軟的壟斷，但也導致了美國司法部的反壟斷訴訟。IE 的壟斷地位後來又被 Chrome 打破——瀏覽器市場的競爭從未真正停止。

## Java 1.0 到企業級平台

Java 1.0 在 1996 年 1 月正式發布。Sun 提供了免費的 JDK 下載，吸引了大量開發者。Java applet 被認為是網路的未來——雖然 applet 最終未能流行，但 Java 在伺服器端找到了真正的歸宿。

**Java 1.0 (1996)** → **Java 2 (1998)** → **Java EE (1999)** → **J2EE 應用伺服器 (2000s)** → **Spring 框架 (2002)**

Java 的成功部分來自於跨平台承諾，部分來自於成熟的企業生態系統——Servlet、JSP、EJB、JMS 等規範將 Java 打造為企業軟體的標準。

## CSS：從結構到表現的分離

CSS Level 1 在 1996 年 12 月成為 W3C 推薦標準。這在當時沒有引起轟動——瀏覽器對 CSS 的支援極不完整，直到 2000 年代早期 CSS 才真正普及。

**CSS 1 (1996)** → **CSS 2 (1998)** → **CSS 3 模組化 (2000s-2010s)**

CSS 的理念——將內容（HTML）與表現（樣式）分離——成為現代網頁開發的基本原則。Flexbox 和 Grid 等後來加入的布局系統讓 CSS 成為一個真正的排版工具。

## 即時通訊的誕生

ICQ 在 1996 年 11 月由 Mirabilis 公司推出。「ICQ」是「I Seek You」的諧音。它引入了今天即時通訊軟體的所有核心功能：聯絡人列表、在線狀態、訊息通知。

**ICQ (1996)** → **AOL Instant Messenger (1997)** → **MSN Messenger (1999)** → **Skype (2003)** → **WhatsApp/微信 (2009-2011)**

ICQ 在 1998 年被 AOL 以 4.07 億美元收購，而當時 Mirabilis 只有 12 名員工——這是網路時代的第一個「小團隊大回報」故事。

## 從 CRT 到 LCD 的漫長轉換

1996 年 LCD 螢幕開始從筆記型電腦走向桌上型應用，但價格昂貴、色彩和視角有限，無法立即取代 CRT。這個轉換從 1996 年開始，到 2005 年左右基本完成——接近十年的過渡。

**CRT 顯示器 (1950s-2005)** → **LCD 螢幕 (1996-至今)** → **OLED (2010s)**

## 1996 年的時代脈絡

- **Dolly 羊誕生**：第一隻複製哺乳動物，生物科技進入新紀元
- **DVD 規格統一**：由 DVD 論壇確立統一標準，取代 VHS 和 LD
- **Nokia 9000 Communicator**：結合手機和 PDA，智慧型手機概念的雛形
- **小型 PC 潮流**：Toshiba Libretto 展示了行動運算的極致追求
