# CSS 1.0：網頁排版的分離革命

## 內容與表現的分離

1996 年 12 月 17 日，全球資訊網聯盟（W3C）發布了 CSS Level 1 推薦標準。CSS（Cascading Style Sheets）的核心目標是將網頁的內容（HTML 結構）與表現（字型、顏色、佈局）分離。

在 CSS 出現之前，網頁樣式是透過 HTML 標籤的屬性直接指定的：`<font color="red">`、`<table width="100%">`、`<center>`。這種做法導致 HTML 文件變得臃腫、難以維護，而且無法實現統一的樣式管理。

## CSS 1 的核心功能

CSS Level 1 提供的功能包括：
- **字型屬性**：font-family、font-size、font-weight、font-style
- **文字屬性**：color、text-align、text-decoration、line-height
- **背景屬性**：background-color、background-image
- **區塊屬性**：margin、padding、border、width、height
- **清單屬性**：list-style-type、list-style-image
- **類別與 ID 選擇器**：.class 和 #id 選擇器

CSS 1 沒有提供的功能（留給 CSS 2）：定位（positioning）、z-index、媒體類型、分頁媒體、表格佈局。

## 初期採用的困境

CSS 的早期推廣極為困難。Netscape Navigator 4 和 Internet Explorer 3/4 對 CSS 的支援都極不完整且各有偏差——同一個 CSS 在不同瀏覽器上的顯示效果完全不同。

這個問題直到 2000 年代早期才逐步改善。CSS 真正的普及是在 IE 6（2001）對 CSS 1 提供了相對完整的支援之後。
