# Java 1.0 正式版：跨平台語言的承諾

## 正式發布

1996 年 1 月，Sun Microsystems 正式發布了 Java 1.0。這是 Java 的第一個穩定版本——雖然 1995 年 5 月已經公開展示，但完整的開發工具鏈和標準類別庫在 1996 年才就緒。

Java 1.0 包含：
- **Java 編譯器**（javac）：將 Java 原始碼編譯為位元組碼
- **Java 執行環境**（java）：JVM 實現
- **Applet API**：讓 Java 程式可以在瀏覽器中執行
- **標準類別庫**：java.lang、java.io、java.util、java.net、java.awt

## HotJava 瀏覽器

Sun 同時推出了 HotJava 瀏覽器——第一款用 Java 編寫的 Web 瀏覽器。雖然 HotJava 從未成為主流瀏覽器，但它是展示 Java applet 能力的重要展示平台。

HotJava 展示了 Java 的動態類別載入（dynamic class loading）能力——瀏覽器可以在執行時從伺服器下載並執行新的 Java 類別。

## 開發者社群的爆發

Java 1.0 的發布引發了開發者社群的極大熱情。主要原因包括：

- **免費下載**：Sun 提供了免費的 Java 開發工具包（JDK）
- **跨平台承諾**：寫一次程式，在 Windows、Macintosh、Unix 上都能執行
- **Web 整合**：Java applet 讓網頁可以有動畫、遊戲和互動應用
- **自動記憶體管理**：比起 C++ 手動管理記憶體，Java 的 garbage collection 大幅降低了開發難度
