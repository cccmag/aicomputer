# 1996 年資訊科技年鑑：前言

1996 年是 1995 年技術突破的落地之年。如果說 1995 年是網路時代的宣告，那麼 1996 年就是基礎設施的建設年。

Java 在 1 月推出 1.0 正式版，Sun 同時發布 HotJava 瀏覽器展示 Java applet 的能力。程式設計師開始認真看待「跨平台」這個當年還很陌生的概念。

Netscape Navigator 2.0 在年初發布，內建對 JavaScript 的支援。IE 3.0 在 8 月推出，開始真正威脅 Netscape 的市場主導地位。瀏覽器戰爭從這一年正式開打。

W3C 在 12 月發布 CSS Level 1 規範，將網頁的內容與表現分離——這是全球資訊網從簡陋文件系統走向成熟應用平台的重要一步。

硬體方面，Nokia 9000 Communicator 以「手機 + PDA」的型態問世，是現代智慧型手機的鼻祖。Toshiba Libretto 以僅 840 克的重量成為當時最小的 PC。DVD 規格終於統一，為後續的光碟儲存革命鋪平道路。

ICQ 在 11 月由以色列公司 Mirabilis 推出，即時通訊從聊天室走向個人化。USB 1.0 規格正式制定，雖然普及還需要幾年，但連接介面的統一已見曙光。

1996 年沒有 1995 年那樣的轟動性發布，但每項進展都讓網際網路離一般人的生活更近了一步。
