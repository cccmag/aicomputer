# 歷史穿梭：1997 年的種子

## Deep Blue：AI 的里程碑與幻覺

1997 年 5 月 11 日，IBM 的 Deep Blue 在六局比賽中擊敗 Garry Kasparov。這場比賽引發了全球對人工智慧的關注和討論。

**Deep Blue (1997)** → **IBM Watson (2011)** → **AlphaGo (2016)** → **GPT-3 (2020)**

然而，Deep Blue 的成功更多來自於客製化硬體的暴力搜尋，而非真正的機器學習。它每秒能評估 2 億個棋局位置，使用的是專用的 VLSI 西洋棋晶片。這種 brute-force 方法在西洋棋中有效，但並不通用。

今天的 AlphaGo 和大型語言模型代表了完全不同——更通用的——AI 方法。但 Deep Blue 在公眾意識中打開了一扇門：「機器可以打敗人類大腦」這個觀念從科幻變成了事實。

## Steve Jobs 的回歸與 Apple 的逆轉

1997 年 8 月，Steve Jobs 在 Apple 瀕臨破產時回歸。他帶來了 NeXT 的作業系統（後來成為 Mac OS X 的基礎），並與 Microsoft 達成了一項 1.5 億美元的投資協議（附帶條件是 Apple 放棄對 Microsoft 的 GUI 專利訴訟）。

**Apple 衰落 (1995-1997)** → **Jobs 回歸 (1997)** → **iMac (1998)** → **iPod (2001)** → **iPhone (2007)**

Jobs 回歸的第一年沒有推出革命性產品，而是做了三件事：精簡產品線（從十多條砍到四條）、獲得微軟投資、推出 Apple Store 線上商店。這些看似保守的措施，讓 Apple 有了生存的空間。

## PageRank：從研究專案到搜尋霸權

1997 年，Larry Page 和 Sergey Brin 在史丹佛大學開始 BackRub 專案。PageRank 演算法的核心洞察是：將網頁之間的超連結視為「投票」——被越多高品質網頁連結的頁面，其重要性越高。

**BackRub (1996-1997)** → **Google 公司 (1998)** → **AdWords (2000)** → **Android (2005)** → **Chrome (2008)**

PageRank 的創新在於它不僅僅考慮關鍵詞匹配，還考慮了網頁的權威性。這個看似簡單的洞察，讓 Google 的搜尋結果品質遠超當時的 Altavista、Yahoo! 和 Lycos。

## XML：資料交換的通用語言

XML 1.0 在 1997 年成為 W3C 提案，1998 年正式成為推薦標準。XML 提供了一套嚴格的語法規則來描述結構化資料。

**SGML (1986)** → **HTML (1993)** → **XML (1997-1998)** → **JSON (2002-2006)** → **YAML/TOML**

XML 在 2000 年代成為企業軟體整合的核心技術——SOAP、WSDL、XSLT、RSS/Atom 都建立在 XML 之上。雖然 JSON 後來在 Web API 領域取代了 XML，但 XML 在文件導向的應用中仍然廣泛使用。

## 六度分隔與社群網路的科學基礎

Stanley Milgram 在 1967 年的「小世界實驗」提出了六度分隔假說。1997 年，這個理論通過電腦網路實驗得到新的驗證——特別是 Duncan Watts 和 Steven Strogatz 發表了小世界網路（small-world network）的數學模型。

**Milgram (1967)** → **Watts-Strogatz 模型 (1998)** → **六度分隔 Facebook 研究 (2016)**

這些成果為 Facebook、LinkedIn 等社群網路的成長提供了理論基礎：人與人之間的連結雖然看似稀疏，但透過少數中間人就可以抵達世界上任何一個人。

## 1997 年的時代脈絡

- **亞洲金融風暴**：7 月從泰國開始，席捲亞洲各國，影響全球科技投資
- **Telia ADSL 啟動**：全球第一個商業 ADSL 服務，寬頻時代開始
- **Mac OS 8 推陳出新**：Apple 在 Jobs 回歸前的自救嘗試，銷量超過 300 萬
- **DVD 播放器上市**：數位視訊光碟走入家庭消費市場
