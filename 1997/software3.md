# Internet Explorer 4.0：瀏覽器戰爭白熱化

## 微軟的追趕

1997 年 9 月，Microsoft 發布了 Internet Explorer 4.0。這是瀏覽器戰爭中的關鍵轉折點——IE 4.0 在功能上首次與 Netscape Navigator 4.0 不相上下，在某些方面甚至超越。

IE 4.0 的開發代號是「Naples」。它被整合進 Windows 95 的 OEM Service Release 2.5 和 Windows 98 中，透過捆綁策略大幅擴大了用戶基礎。

## 主要功能

IE 4.0 引入了多項重要技術：
- **Dynamic HTML (DHTML)**：允許 JavaScript 動態操作 HTML 元素和 CSS 樣式
- **Active Desktop**：將網頁內容直接放在 Windows 桌面上作為背景
- **Web 資料夾整合**：檔案總管中可以直接瀏覽 WebDAV 資料夾
- **ActiveX 控制項**：增強了瀏覽器的擴展能力
- **完整 CSS 支援**：對 CSS Level 1 的支援遠比 Netscape 4 完善
- **HTTP 1.1 支援**：持續連線、分塊傳輸編碼等

## Active Desktop

IE 4.0 最引人注目的功能是 Active Desktop——Windows 的桌面本身變成了一個 HTML 頁面。使用者可以將即時股價、天氣資訊、新聞頭條等網頁元件放在桌面上。

這個功能雖然很酷，但也是系統效能的一大負擔——1997 年的電腦跑 Active Desktop 經常非常緩慢。它後來在 Windows XP 中被大幅簡化，在 Windows Vista 中被移除。

## 市佔率的轉折

IE 4.0 的發布是瀏覽器戰爭的轉折點。結合免費策略和 Windows 捆綁，IE 的市場佔有率在 1998 年初超過了 30%，並在 1999 年超過 60%。Netscape 的市佔率從 1996 年的 80% 以上跌落到 1999 年的 30% 以下。
