# Western Union Telex 網路 — 全球電報交換系統

一九五三年，Western Union 的 Telex 網路已經成為全球最重要的電報交換系統。Telex（Telegraph Exchange）是一種使用電傳打字機（teleprinter）的自動化電報交換系統，與今天的電子郵件和即時通訊在功能上有驚人的相似之處。

Telex 網路的運作方式如下：用戶在自己的辦公室安裝一台電傳打字機，透過專用線路連接到 Western Union 的交換中心。要發送訊息時，用戶在電傳打字機上輸入收件方的 Telex 號碼，然後開始打字。系統會自動將訊息路由到目的地，並在對方的電傳打字機上即時列印出來。如果收件方不在線，系統會將訊息儲存在交換中心，待收件方上線後再傳送。

Telex 網路的全球化程度在當時令人矚目。一九五三年，Telex 已經連接了美國各大城市，並透過海底電纜和短波無線電與歐洲、亞洲和澳洲的主要城市相連。一個在紐約的用戶可以直接與倫敦、巴黎、東京或雪梨的用戶進行文字通訊——回應時間以分計而非以天計。

Telex 使用 5 位元的 Baudot 碼（而非後來的 ASCII 碼），傳輸速度為每秒五十個字元。雖然速度緩慢，但 Telex 的可靠性極高——它使用專用線路和專業的交換設備，確保訊息能夠準確送達。在當時，Telex 的地位類似於今日的電子郵件：它是企業通訊的標準工具。

Telex 的技術對未來的資料通訊產生了直接影響。Telex 的存儲轉發（store-and-forward）機制——訊息在中間節點暫存後再轉發——後來被電子郵件系統和網路路由協議廣泛採用。Telex 的全球位址編碼方案——國家碼 + 地區碼 + 用戶號碼——類似於今日的 IP 位址和網域名稱系統的層級結構。

一九五三年，Telex 網路的運作與計算機幾乎沒有直接關聯。但隨著計算機技術的發展，Telex 的線路和協議逐漸被用於計算機間的通訊。一九六○年代，許多大學和研究機構透過 Telex 網路連接各個計算機中心，形成了早期計算機網路的前身。

從 Telex 到傳真、從傳真到電子郵件、從電子郵件到即時通訊——這條通訊技術的演化路徑清晰地顯示：一九五三年的 Telex 網路已經確立了「即時文字通訊」的基本模式，只是載體從電信網路轉移到了網際網路。
