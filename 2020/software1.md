# OpenAI GPT-3：1750 億參數的語言模型

## 規模的飛躍

2020 年 6 月 11 日，OpenAI 發表了 GPT-3（Generative Pre-trained Transformer 3），一個擁有 1750 億參量的自回歸語言模型。與 2019 年的 GPT-2（15 億參數）相比，GPT-3 在參數量上提升了超過 100 倍，是當時世界上最大的神經網路模型。

GPT-3 的訓練使用了微軟 Azure AI 超級計算機，耗費了數千 GPU 運算天和數百萬美元的運算資源。訓練資料涵蓋了 Common Crawl、WebText2、Books1、Books2 和維基百科——總計約 570GB 的文本資料。

## 湧現的能力

GPT-3 最令人驚訝的不是其巨大的規模，而是在 Zero-shot 和 Few-shot 設定下展現的能力：

- **文本生成**：可以撰寫連貫的文章、詩歌、故事和新聞報導
- **程式碼生成**：理解自然語言描述並生成對應的程式碼
- **翻譯與摘要**：在多語言環境下表現出色
- **問答對話**：可以進行有一定深度的對話
- **簡單推理**：展現了初步的常識推理能力

GPT-3 可以在沒有針對性訓練的情況下完成許多任務——只需在提示（prompt）中提供幾個範例（few-shot）即可。

## 獨佔 API 模式

不同於 GPT-2 的開源釋出，OpenAI 對 GPT-3 採取了完全不同的策略：透過付費 API 提供服務，不對外公開模型權重。這引發了關於 AI 民主化和技術獨佔的討論。微軟在 2020 年 9 月獲得了 GPT-3 的獨家授權——這項合作加劇了外界對 OpenAI 偏離其非營利初衷的擔憂。

## 影響

GPT-3 證明了「規模就是一切」（scale is all you need）——當模型足夠大時，新的、未經訓練的能力會「湧現」。這一發現直接引導了後來 GPT-4、Claude、PaLM 和 Llama 等更大模型的開發方向。
