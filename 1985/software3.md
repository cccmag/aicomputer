# GNU Emacs：自由軟體的編輯器基石

## Emacs 的誕生

1985 年，Richard Stallman 發布了 GNU Emacs——GNU 專案的第一個重要軟體產出。Emacs（Editing MACroS）最初由 Stallman 在 1976 年在 MIT AI 實驗室開發，運行在 PDP-10 的 ITS 作業系統上。GNU Emacs 則是從頭重寫的 Unix 版本。

GNU Emacs 的開發是 Stallman 在 GNU 專案中的策略性選擇。他寫道：「我暫停編譯器開發，先做 Emacs——因為 Emacs 是我最熟悉的領域，而且 Unix 社群需要一個比 vi 更強大的編輯器。」

## 可擴展的編輯器

GNU Emacs 最關鍵的設計決策是內建了一個完整的 Lisp 直譯器——使用者可以用 Emacs Lisp（Elisp）來擴展編輯器的任何部分。這使得 Emacs 不僅是一個文字編輯器，更是一個可程式化的計算環境：

GNU Emacs 的早期使用者寫出了各種擴展：語法高亮、版本控制整合、郵件用戶端（Gnus）、新聞閱讀器、Shell 模擬器、甚至遊戲。Emacs 的座右銘「一個自帶編輯器的作業系統」雖然是玩笑話，但也反映了它的擴展性。

## 自由軟體的模範

GNU Emacs 是第一個在 GNU GPL 授權下發布的重要軟體。它的成功證明了自由軟體模式可以產出高品質的實用工具。GNU Emacs 至今仍在活躍開發，是歷史最悠久且仍在廣泛使用的自由軟體之一。
