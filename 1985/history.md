# 歷史穿梭：1985 年的種子

## 32 位元的起點

Intel 80386 在 1985 年的發表是 PC 產業最重要的硬體里程碑之一。它開創了一條長達數十年的架構傳承：

**Intel 80386 (1985) → 80486 (1989) → Pentium (1993) → Core 2 (2006) → Core i 系列**

80386 引入的 32 位元架構、記憶體分頁和虛擬 8086 模式，為 Windows 和 Linux 的發展提供了硬體基礎。如果沒有 80386，多工 PC 作業系統的普及可能要推遲數年。

## Windows 的第一次試水

Windows 1.0 在 1985 年的推出雖然市場反應冷淡，但它啟動了微軟的 GUI 戰略：

**Windows 1.0 (1985) → Windows 2.0 (1987) → Windows 3.0 (1990) → Windows 95 (1995) → 現代 Windows**

Windows 1.0 證明了：即使目前的產品不夠好，只要持續投資，最終可以佔據 GUI 市場的主導地位。

## C++ 與物件導向的產業化

C++ 在 1985 年的商業化是物件導向從學術概念轉變為產業標準的轉折點：

**Simula 67 (1967) → Smalltalk (1972) → C++ (1985) → Java (1995) → C# (2000)**

C++ 的成功證明了「漸進式改良」的語言設計策略——不拋棄 C 的生態系，而是在其上加入物件導向特性——是最有效的普及路徑。

## 桌上出版的誕生

Macintosh + PageMaker + LaserWriter 的三位一體在 1985 年創造了桌上出版產業：

**傳統排版 → 桌上出版 (1985) → PostScript 標準 → 數位印刷 → 出版民主化**

PageMaker 改變的不僅僅是排版——它改變了誰可以出版、出版流程如何運轉、以及什麼是可能的設計。

## NSFNET 與網際網路的公共化

NSFNET 的啟動（1985）代表了網際網路從軍事研究（ARPANET）轉向公共學術服務的關鍵一步：

**ARPANET (1969) → CSNET/BITNET (1981) → NSFNET (1985) → 商業 ISP (1990s) → 全球網際網路**

NSF 決定支援 TCP/IP 而非 OSI 協定，可能是這一年最重要的網路政策決定——它確保了網際網路將基於一個開放且已驗證的協定發展。
