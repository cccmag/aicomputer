# Microsoft Windows 1.0：GUI 的微軟時代

## 從 VisiOn 到 Windows

1985 年 11 月 20 日，Microsoft 正式推出 Windows 1.01，售價 99 美元。Windows 的開發始於 1983 年，當時 Bill Gates 在 COMDEX 上看到了 VisiCorp 的 Visi On——一個在 IBM PC 上執行的 GUI 環境。Gates 意識到 GUI 是不可阻擋的趨勢，決定 Microsoft 必須參與其中。

Windows 1.0 的開發過程一波三折。原定 1984 年上市，但因為技術挑戰和 Apple 的壓力——Apple 已經有了 Macintosh——發布日期一再延遲。

## 技術特色

Windows 1.0 運行在 MS-DOS 之上，不是一個獨立的作業系統，而是一個圖形化操作環境。它的核心元件是 MS-DOS Executive——一個簡單的檔案管理程式，顯示檔案列表，支援滑鼠點選執行。

Windows 1.0 內建了多個應用程式：MS-DOS Executive、Write（文書處理）、Paint（繪圖）、Notepad、Calendar、Calculator、Clock 和 Reversi（遊戲）。

## 限制與批評

Windows 1.0 受到許多批評：視窗不能重疊（只能並排平鋪）、顏色只有黑白的 16 種灰階、執行速度緩慢、應用程式極少。它看起來與 Macintosh 類似，但使用體驗遠不如 Mac。

儘管如此，Windows 1.0 的推出具有重要的戰略意義：它讓 Microsoft 進入了 GUI 市場。Windows 1.0 在市場上反應平淡，但它為 Windows 2.0（1987）和 Windows 3.0（1990）的改善建立了基礎。
