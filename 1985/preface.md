# 1985 年資訊科技年鑑：前言

1985 年是 PC 產業加速分化與整合的一年。Intel 發表了 80386 微處理器——第一個 32 位元 x86 晶片，內含 275,000 顆電晶體，為未來的 Windows 和 OS/2 提供了硬體基礎。但 386 的預產樣本要到 10 月才釋出，真正的市場效應在隔年才展現。

Microsoft 在 11 月 20 日正式推出 Windows 1.0，售價 99 美元。雖然它只是一個在 MS-DOS 上執行的圖形化外殼，視窗不能重疊，功能有限——但這是微軟進軍 GUI 的正式起手式。

C++ 在這一年從實驗室走進商業世界。Bjarne Stroustrup 出版了《The C++ Programming Language》，Cfront 編譯器開始銷售。物件導向程式設計從學術概念變成了實用工具。

網路方面，3 月 15 日 symbolics.com 成為第一個 .com 域名。NSF 啟動了 NSFNET 計畫，連結全美五個超級電腦中心，採用 TCP/IP 協定——這為後來取代 ARPANET 的美國學術網路骨幹奠定了基礎。

X Window System 在 MIT 誕生，為 Unix 工作站提供了工業級的視窗系統。GNU Emacs 由 Richard Stallman 發布，成為自由軟體陣營中最具影響力的編輯器。Thomas Knoll 開始編寫一個名為 Display 的影像處理程式——這就是後來 Photoshop 的前身。

1985 年，PC 生態系的權力正在轉移。IBM 的影響力依然巨大，但相容機廠商、獨立軟體開發商、新創公司正在悄悄改寫遊戲規則。
