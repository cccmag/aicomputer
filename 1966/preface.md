# 1966 年資訊科技年鑑：前言

1966 年是電腦科學從萌芽走向體系化的關鍵年份。這一年，ACM 設立了圖靈獎（Turing Award），表揚對計算領域有重大貢獻的學者，首位得主艾倫·佩利（Alan Perlis）以他在編譯器與高階程式語言的開創性工作獲此殊榮，象徵了計算機科學作為一門獨立學術領域的正式確立。

硬體方面，HP 推出了第一台電腦 HP 2116A，從儀器控制器意外開啓了這家測量設備巨頭的電腦事業；IBM 的 System/360 Model 91 導入領先時代的「順序外執行」（out-of-order execution）架構；RCA Spectra 70 系列成為首批大規模採用積體電路的商用電腦；而 Robert Dennard 在 IBM 發明的 DRAM（動態隨機存取記憶體）單元，則奠定了往後數十年記憶體技術的基石。

軟體領域同樣熱鬧非凡。MIT 的 Joseph Weizenbaum 寫出了 ELIZA，這個模擬心理治療師的程式讓大眾首次感受到人工智慧的魔力；Martin Richards 在劍橋大學設計了 BCPL，這門語言日後衍生出 B 語言和 C 語言，影響深遠；Lockheed 的 DIALOG 系統成為第一個大型互動式線上檢索服務；而 Neil Pappalardo 在麻州總醫院開發的 MUMPS 語言，則在醫療資訊系統中屹立不搖數十年。

網路方面，1966 年是網際網路構想的誕生之年。Bob Taylor 加入 ARPA 後，成功說服 Larry Roberts 從 MIT 林肯實驗室轉戰華盛頓，正式啟動 ARPANET 計劃。同時，Carterfone 判決為電話線路的開放使用鋪平了道路，間接促成了日後的數據機與網路存取革命。IBM System/360 Model 67 的虛擬記憶體能力，則為分時系統提供了不可或缺的硬體支援。

這一年，Ralph Baer 在一張紐約公車站的筆記本上寫下了用電視玩遊戲的想法，人類的電子遊戲產業就在那一刻悄悄誕生。

1966 年的每一項進展，都像一顆種子，在接下來的數十年中陸續長成大樹。
