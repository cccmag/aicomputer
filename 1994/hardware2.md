# 28.8 kbps 數據機：撥接速度翻倍

## 頻寬的關鍵跳躍

1994 年，數據機技術從 V.32bis（14.4 kbps）升級到 V.34（28.8 kbps）——這是一個對網際網路發展至關重要的頻寬跳躍。雖然這些速度在今天看來慢得可笑，但 28.8 kbps 讓真實的網頁瀏覽體驗第一次變得可以接受。

在 14.4 kbps 的時代，一個包含幾張圖片的網頁可能需要 30-60 秒才能載入。28.8 kbps 將這個時間縮短了一半，讓使用者在網頁導航時不至於因為等待時間過長而放棄。

## V.34 標準的技術成就

1994 年通過的 ITU V.34 標準是類比數據機工程的頂峰：

- 採用先進的訊號調變技術——格狀編碼調變（Trellis Coded Modulation）
- 使用二維迴聲消除技術
- 動態速率協商：自動偵測線路品質並調整傳輸速度
- 雙向不對稱傳輸：下載速度可與上傳速度不同

V.34 數據機需要強大的數位訊號處理器（DSP）晶片來實作其複雜的調變演算法。這推動了 DSP 技術的快速發展。

## 對 Web 多媒體的推動

28.8 kbps 不僅僅是速度的增加，它改變了人們對 Web 可能性的期望：

- 圖片：可以在合理時間內載入高解析度的 JPEG 圖片
- 音效：即時播放壓縮音訊片段成為可能
- 互動性：表單填寫和提交的等待時間減少

1994 年的 28.8 kbps 數據機和 Netscape Navigator 一起，為 Web 的多媒體革命提供了必要的技術基礎。雖然距離「寬頻」還有很長的路（寬頻在 2000 年代初期才真正普及），但 28.8 kbps 讓 Web 從「文字加慢速圖片」進入了「可以忍受的多媒體」階段。
