# W3C 成立：全球資訊網標準化

## 麻省理工學院的約定

1994 年 10 月 1 日，Tim Berners-Lee 在麻省理工學院（MIT）計算機科學實驗室創立了全球資訊網聯盟（World Wide Web Consortium, W3C）。W3C 的使命是「引導 Web 發揮其全部潛力」，方法是制定和維護 Web 的開放標準。

W3C 的成立源於 Berners-Lee 對 Web 碎片化的擔心。到 1994 年，Web 已經從 CERN 的小型研究專案成長為全球現象，數百家公司和組織正在開發 Web 相關產品。如果沒有統一的標準，Web 可能會分裂成多個不相容的系統。

## W3C 的治理結構

W3C 採用了獨特的組織形式：
- **主辦機構**：MIT（美國）、ERCIM（歐洲，法國國家資訊與自動化研究所 INRIA）、慶應義塾大學（日本）
- **會員制**：由會員組織（公司、政府、非營利組織）資助
- **工作小組**：由會員代表和專家組成，制定標準
- **免版稅承諾**：所有 W3C 標準都可以免費實作

Berners-Lee 堅持 W3C 必須是一個中立、開放、非營利的組織——這確保了 Web 的標準不會被單一商業公司所控制。

## 1994 年的 W3C 成就

在 1994 年的最後三個月中，W3C 開始了 HTML、HTTP 和 URL 標準化的工作。雖然這些技術的基礎已經由 Berners-Lee 在 CERN 定義，但 W3C 提供了正式的標準化程序。

W3C 的第一場會議於 1994 年 12 月 14 日舉行，討論了 HTML 2.0 標準的草案。1994 年加入 W3C 的初始成員包括 Netscape Communications——它在成立幾週內就加入了 W3C。

## 開放的遺產

W3C 的存在確保了 Web 始終保持開放和互通。如果沒有 W3C，Web 的歷史可能會完全不同——就像 1980 年代的家用電腦市場分裂為 Commodore、Atari 和 Apple 的不相容標準一樣。

1994 年 W3C 的成立——與 CERN 1993 年的公共領域決定相結合——確立了 Web 作為開放平台的長期基礎。這可能是 Berners-Lee 做出的最重要的制度性貢獻。
