# 歷史穿梭：1994 年的種子

## 從 Yahoo! 到搜尋引擎霸權

1994 年由兩位史丹佛研究生創立的人工編輯目錄，後來成長為網際網路第一個巨獸：

**Yahoo! 目錄 (1994)** → **Yahoo! 公司 (1995)** → **Google 成立 (1998)** → **Yahoo! 的衰落**

Yahoo! 的早期成功展示了「組織資訊」的巨大價值。但人工目錄無法跟上網路成長的速度——這為 Google 的自動化搜尋革命留下了空間。諷刺的是，兩位創辦人開始這個專案時，原本只是不想靠記憶記住他們最喜歡的網站。

## 從 Amazon 到電子商務帝國

1994 年貝佐斯在車庫中啟動的電子商務實驗，改變了全球零售業：

**Amazon 成立 (1994)** → **Amazon 上線 (1995)** → **Amazon 上市 (1997)** → **AWS (2006)** → **全球市值前十 (2010s)**

貝佐斯在 1994 年做出的關鍵決策——選擇書籍作為切入點——是典型的「最小可行產品」思維：選擇一個標準化、需求明確、且不受地域限制的商品，先求驗證電子商務的商業模式。

## 從 Netscape 到瀏覽器大戰

**Netscape Navigator 1.0 (1994)** → **網景上市 (1995)** → **Internet Explorer 3.0 (1996)** → **瀏覽器大戰** → **Mozilla Firefox (2004)** → **Chrome (2008)**

Netscape 在 1994 年的故事不僅是一家公司的崛起，更是一種商業模式的驗證：網路可以透過軟體發行（而非硬體銷售）來創造價值。Netscape 1995 年的 IPO 是 dot-com 時代的發令槍。

## 從 SSL 到網路安全產業

**SSL 1.0 (1994)** → **SSL 3.0 (1996)** → **TLS 1.0 (1999)** → **HTTPS 普及 (2010s)**

SSL 的發明解決了電子商務的最後障礙——信任。從 1994 年第一筆 12.48 美元的 CD 交易，到今天每年數兆美元的電子商務市場，SSL/TLS 是所有這些交易的基礎。

## 從專家系統到現代 AI

Feigenbaum 和 Reddy 在 1994 年獲得圖靈獎時，AI 正處於轉型期。但他們所倡導的「用知識增強系統」的理念，在今天以不同的形式復興：

**專家系統 (1980s)** → **AI 寒冬 (1990s)** → **統計機器學習 (2000s)** → **深度學習 (2012)** → **大型語言模型 (2023)**

Reddy 的語音辨識研究和 Feigenbaum 的知識工程，最終在 Siri、Alexa 和 ChatGPT 中找到了主流應用——雖然實現方式已經完全不同。

## 1994 年的時代脈絡

- **網際網路商業化元年**：這一年發生的事件——Netscape、SSL、Amazon、Yahoo!、W3C——共同構成了現代網際網路的基礎
- **冷戰後紅利**：美國經濟在 1990 年代中期經歷了持續繁榮，為 dot-com 投資提供了資本
- **電信自由化**：美國 1996 年電信法的前奏正在醞釀，推動了電信基礎設施的競爭與投資
- **個人電腦的成熟**：Windows 3.1、Intel Pentium、CD-ROM 和 28.8 kbps 數據機的組合，讓一般家庭也有了上網的技術條件

1994 年是「網路變成真實世界」的一年。1993 年 Mosaic 讓網際網路變得可視，而 1994 年讓網際網路變得有用——可以購物、可以搜尋、可以安全交易。1994 年奠定的這些基礎，至今仍在運作。
