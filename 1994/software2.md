# Java 語言的準備：Oak 改名與 HotJava

## 從機上盒到網際網路

1994 年，Sun Microsystems 的綠色專案（Green Project）經歷了第三次也是最重要的一次轉向。最初為消費電子裝置開發的 Oak 語言在 1993 年版機上盒市場的爭奪中失敗。1994 年初，First Person Inc.（Sun 為綠色專案成立的子公司）被解散，團隊成員面臨失業的命運。

但在這時，團隊中的 Patrick Naughton 和 Jonathan Payne 注意到了一個重要的新趨勢——World Wide Web 的爆炸性成長。他們意識到 Oak 的特性——平台無關、安全、小巧——正好完美地適合 Web 環境。

## HotJava 瀏覽器

1994 年夏天，Naughton 用一個週末寫了一個名為 WebRunner（後改名為 HotJava）的原型瀏覽器。HotJava 展示了 Oak 最革命性的功能：可以在瀏覽器中動態下載和執行應用程式（後來稱為 applet）。

1994 年秋天，Green 團隊向 Sun 高層演示了 HotJava，獲得了管理層的支持。Scott McNealy（Sun 的 CEO）認識到 Oak 的戰略價值——它可能是 Sun 挑戰 Microsoft 桌面統治的武器。

## Oak 改名為 Java

1994 年底，團隊發現「Oak」這個名稱已被另一家科技公司（Oak Technology）註冊為商標。在一場腦力激盪會議中，團隊提出了幾個替代名稱。最終選中了「Java」——這個詞代表了程式設計師對咖啡的熱愛。

Java 的正式公開要到 1995 年 5 月的 SunWorld 會議，但 1994 年的這項基礎工作——重新定位為 Web 語言、開發 HotJava 瀏覽器、建立 Java 平台的核心架構——為 Java 在 1995 年的轟動登場做好了所有準備。
