# Ada 語言：美國國防部的標準

## 從數百種語言到一種標準

1970 年代，美國國防部（DoD）面臨一個嚴重的問題：其嵌入式系統使用了超過 450 種不同的程式語言，維護成本極高。1975 年，DoD 啟動了 HOLWG（Higher Order Language Working Group）計劃，目標是建立一個統一的標準程式語言。

經過四年的需求分析、設計競賽和審查，1979 年 DoD 選定了以法國公司 CII Honeywell Bull 的設計為基礎——由 Jean Ichbiah 領導的團隊提出了「綠」方案（Green），擊敗了其他競爭方案（包括「紅」方案，後來的 Pascal 衍生）。

## Ada 的設計哲學

Ada 以詩人拜倫的女兒 Ada Lovelace——普遍被認為是世界上第一位程式設計師——命名。這門語言強調可靠性、可維護性和效率，特別適合大型嵌入式即時系統。

Ada 引入了許多當時先進的特性：強型別檢查、封裝（package）、例外處理、多工（tasking，內建支援並行處理）、泛型（generic）單元。它也是第一個在設計階段就將軟體工程方法論納入考量的語言。
