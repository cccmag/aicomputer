# Smalltalk-80：純粹的物件導向

## PARC 的夢想機器

1970 年代，Xerox PARC 的 Alan Kay 領導的學習研究小組（Learning Research Group）在開發 Smalltalk——一個完全以物件導向為核心的程式語言和開發環境。1979 年，這個計畫已發展到 Smalltalk-80 版本。

Smalltalk 將「一切皆物件」的理念貫徹到底：整數是物件、類別是物件、程式碼區塊也是物件。訊息傳遞（message passing）是唯一的計算方式。Smalltalk 環境本身也提供了圖形化介面、視窗系統、位元圖（bitmap）顯示器和滑鼠操作——這些概念比商用市場早了將近五年。

## 啟發了下一代

1979 年 12 月，Steve Jobs 在參觀 Xerox PARC 時見到了 Smalltalk 的展示——視窗重疊、下拉選單、所見即所得（WYSIWYG）的文字處理。這段歷史性的參觀直接影響了 Lisa 和 Macintosh 的設計。

Smalltalk-80 的影響遠遠超越了 Xerox：它催生了 Objective-C（NeXT 和後來 macOS/iOS 的基礎），影響了 Java 的虛擬機設計，其 MVC 架構至今仍是 Web 框架的標準模式。
