# 歷史穿梭：1979 年的種子

## 從 VisiCalc 到現代試算表

1979 年 VisiCalc 的電子試算表概念，在接下來的數十年中發展為一條清晰的路線：

**VisiCalc (1979)** → **Lotus 1-2-3 (1983)** → **Microsoft Multiplan (1982)** → **Microsoft Excel (1985, Mac)** → **Excel for Windows (1987)** → **Google Sheets (2006)**

VisiCalc 的 A1 標記法至今仍然是所有試算表的共同語言。它將程式設計中的「自動重算」概念帶給了商業使用者，是軟體作為「生產力工具」的典範。

## 從 Walkman 到 iPhone

Sony Walkman 開創的行動媒體模式，是個人行動裝置的始祖：

**Walkman TPS-L2 (1979)** → **CD Walkman (1984)** → **MP3 播放器 (1998)** → **iPod (2001)** → **iPhone (2007)**

Walkman 證明了人們願意為了行動體驗而攜帶專用裝置。iPhone 將這個概念推向了極致——將電話、音樂播放器、網路瀏覽器整合在一個口袋大小的裝置中。

## 68000 架構的統治

Motorola 68000 的優雅架構培育了一個時代的電腦設計：

**68000 (1979)** → **68020 (1984, 完整 32 位元)** → **68030 (1987)** → **68040 (1990)** → **68060 (1994)**

68k 架構驅動了 Macintosh、Amiga、Atari ST 等經典平台。雖然 Motorola 後來轉向 PowerPC，但 68k 的扁平記憶體模型和正交指令集設計理念，影響了後來的所有 RISC 處理器。

## 從 Usenet 到社交媒體

Usenet 的分散式討論系統是線上社群的鼻祖：

**Usenet (1979)** → **BBS (1980s)** → **網路論壇 (1990s)** → **Slashdot/Reddit (1997/2005)** → **社交媒體 (2000s)**

Usenet 不僅是技術基礎設施，更是網路文化的孵化器。表情符號、FAQ、spam、trolling、netiquette——這些詞彙都是在 Usenet 上形成的。Usenet 的階層式討論群組結構（comp.*、sci.*、rec.*）影響了後來所有的主題式討論空間。

## 1979 年的時代脈絡

- **第二次石油危機**：1979 年能源價格飆升，影響全球經濟，但同時促進了節能技術和日本消費電子產品的競爭力
- **Walkman 與日本經濟**：Walkman 的成功是日本消費電子產業的縮影——從 1960 年代的模仿者轉變為 1980 年代的創新領袖
- **商業微電腦革命**：VisiCalc 讓個人電腦從 hobbyist 玩具變成了辦公室必需品，直接促成了 IBM 決定進入 PC 市場
- **冷戰科技競爭**：Ada 語言的研製反映了美軍對軟體標準化的需求，Usenet 則是大學研究者在國防經費之外尋找通訊管道的創新
