# 歷史穿梭：2015 年的種子

## 從 Docker 到雲端原生

2015 年的容器革命開啟了一條清晰的產業路徑：

**Docker (2013)** → **Kubernetes 1.0 (2015)** → **CNCF 成立 (2015)** → **Kubernetes 畢業 (2018)** → **服務網格與無伺服器運算**

容器技術改變了應用程式的打包、部署和維運方式。Kubernetes 成為雲端運算的「Linux」——一個基礎設施層的開放標準。2015 年成立的 CNCF 建立了雲端原生軟體的生態治理模型。

## TensorFlow 與深度學習的民主化

TensorFlow 的開源（2015 年 11 月）是 AI 發展的關鍵轉折：

**TensorFlow (2015)** → **PyTorch (2017)** → **BERT (2018)** → **GPT (2018)** → **大型語言模型時代**

2015 年之前，深度學習框架大多是學術研究導向或公司內部專用。TensorFlow 的開源降低了 AI 開發的門檻，讓全球數百萬開發者可以參與機器學習的創新。

## 從 Swift 開源到伺服器端 Swift

Apple 的 Swift 開源代表了平台策略的重大轉變：

**Swift 開源 (2015)** → **Swift Package Manager** → **AWS Lambda Swift 支援** → **Swift on Server 生態**

Swift 不再只是 iOS 開發者的語言，它開始進入伺服器端、雲端函數、跨平台應用等領域。

## Windows 10 的服務化

Windows 10 的「Windows as a Service」模式改變了作業系統的生命週期管理：

**Windows 10 (2015)** → **Windows 11 (2021) 繼承相同模式**

傳統的每三年一次大版本更新被持續交付取代。這個模式後來被 macOS、Android 和 iOS 所借鑑。

## 物聯網的種子與 5G 時代

2015 年的 IoT 平台部署與 MQTT/CoAP 標準化為物聯網的規模化奠定了基礎：

**IoT 平台 (2015)** → **邊緣運算崛起** → **5G 標準完成 (2018)** → **百億連線裝置**

2015 年的物聯網還處於「連網」階段，但已經為後續的資料驅動智慧應用做好了基礎設施準備。

## 2015 年的時代脈絡

- **後 PC 時代的過渡**：2015 年全球 PC 出貨量持續下滑，智慧型手機和平板成為主要運算平台
- **雲端運算的價格戰**：AWS、Azure、Google Cloud 展開價格競爭，雲端服務價格大幅下降
- **中國科技崛起**：阿里巴巴、騰訊、華為在雲端服務和行動支付領域快速擴張
- **加密貨幣波動**：比特幣價格在 2015 年經歷低谷後開始回升，區塊鏈技術吸引更多關注
