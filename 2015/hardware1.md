# Intel Skylake 處理器：14 奈米的架構革新

## 第六代 Core

2015 年 8 月，Intel 正式發布 Skylake 微架構（第六代 Core 處理器），代號 Skylake-S。這是 Intel 在 14 奈米製程上的第二代產品（繼 Broadwell 之後），但 Skylake 代表了一次真正的架構革新而非單純的製程微縮。

Skylake 引入了全新的核心與顯示架構，支援 DDR4 與 DDR3L 記憶體，並配備了增強的整合式 GPU（Gen9 顯示架構）。在效能上，Skylake 比前一代 Haswell 提供了約 10-20% 的 CPU 效能提升，而 GPU 效能提升更為顯著。

## 平台變革

Skylake 採用了新的 LGA 1151 插槽，搭配 100 系列晶片組（Z170、H170 等）。這一代處理器首次將電壓調節模組移回主機板上（FIVR 設計的逆轉），簡化了主機板供電設計。

在功耗方面，Skylake 展示了驚人的效率——TDP 從 95W（Haswell）降至 91W（Skylake 高階型號）或更低（65W 主流型號），同時提供更高的運算效能。

## 影響與意義

Skylake 成為使用壽命最長的 Intel 架構之一，後續的 Kaby Lake、Coffee Lake 都基於 Skylake 的基礎進行優化而非全面翻新。直到 2018 年，Skylake 仍然是許多商用電腦的主流選擇。
