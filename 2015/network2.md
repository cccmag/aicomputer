# ASP.NET Core 1.0：微軟的跨平台革命

## .NET 的轉捩點

2015 年，Microsoft 在 Connect() 開發者大會上發布了 ASP.NET Core 1.0（當時稱為 ASP.NET 5），這標誌著 .NET 平台從 Windows-only 走向跨平台的重要轉折。

ASP.NET Core 從頭開始重新設計，是一個模組化、輕量級、高效能的 Web 框架。它可以在 Windows、macOS 和 Linux 上執行，這在 .NET 的歷史上是前所未有的。這項改變來自於微軟新的 CEO Satya Nadella 對「開發者優先」的擁抱，以及公司對開源的態度轉變。

## 設計特點

ASP.NET Core 引入了多項架構創新：

- **模組化中介軟體（Middleware）**：建構在 OWIN 規範之上的請求處理管線
- **內建依賴注入（DI）**：框架層級的 DI 容器
- **跨平台執行**：基於 .NET Core Runtime 而非完整 .NET Framework
- **高效能**：在 Techempower 基準測試中表現優異

## 開源生態

ASP.NET Core 是完全開源的（MIT 授權），在 GitHub 上開發。這吸引了大量社群貢獻者，並開啟了與 Node.js、Go、Python 等語言生態的直接競爭。

ASP.NET Core 1.0 的發布也包含了 Entity Framework Core——EF 的輕量級跨平台版本，以及 .NET Core CLI 工具鏈。
