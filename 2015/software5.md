# Swift 2.0 開源：Apple 語言的開放時刻

## 語言的開放承諾

2015 年 6 月，Apple 在 WWDC 上宣布 Swift 2.0 將在年底開源，兌現了 Tim Cook 讓 Apple 成為「開源社群成員」的承諾。2015 年 12 月 3 日，Swift 正式在 GitHub 上以 Apache 2.0 授權開源。

Swift 2.0 引入了多項關鍵語言特性：錯誤處理模型（do-try-catch）、guard 陳述式、可用性檢查、協定擴展等。這些特性使 Swift 不僅更強大，也更加安全。

## 開源版圖

Swift 的開源不僅僅是原始碼的釋出，還包括完整的社群治理結構。Apple 建立了 Swift.org 網站，發布了 Swift Evolution 流程——一個用於討論和決定 Swift 語言變更的開放社群程序。

開源版本同時支援 macOS 和 Linux（Ubuntu），這意味著 Swift 不再只是 iOS/macOS 的專屬語言，而可以用於伺服器端開發。這個決定催生了後續的 Swift Server 生態系統。

## 歷史定位

Swift 的開源在 Apple 的歷史上是極為罕見的一步。長久以來，Apple 對其開發工具和三方生態採取封閉控制的策略，Swift 開源代表了公司文化的一次重大轉變。

到 2015 年底，Swift 已經在 GitHub 上獲得了大量社群貢獻，並開始進入伺服器端開發領域。
