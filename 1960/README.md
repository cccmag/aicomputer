# 1960 年資訊科技年鑑

## 目錄

### 1. 前言
- [前言](preface.md) — 1960 年資訊科技總覽：PDP-1 交付、IBM 1401 問世、ALGOL 60 標準化、COBOL 完成、首顆平面積體電路、Quicksort 誕生

### 2. 硬體發展
- [hardware1: DEC PDP-1 交付](hardware1.md) — 第一台小型電腦誕生，售價 12 萬美元，全電晶體化，配備陰極射線管顯示器
- [hardware2: IBM 1401 發表](hardware2.md) — 電晶體化可變字長商業電腦，6 位元 BCD 碼，磁芯記憶體，成為史上最暢銷主機之一
- [hardware3: 首顆平面積體電路](hardware3.md) — Fairchild 的 Jay Last 團隊於 9 月 27 日製出第一顆單晶矽平面 IC
- [hardware4: NEAC 2203](hardware4.md) — 日本第一部電晶體化電腦，為近畿日本鐵道開發線上訂位系統
- [hardware5: LGP-30](hardware5.md) — Librascope 公司出品，售價 5 萬美元的桌上型電腦，鼓式記憶體，單用戶操作

### 3. 軟體發展
- [software1: ALGOL 60 公佈](software1.md) — 第一個國際程式語言標準，引入區塊結構與遞迴
- [software2: COBOL 規格完成](software2.md) — 通用商業語言，英語化的語法設計
- [software3: LISP 1.0](software3.md) — McCarthy 的列表處理語言，car/cdr 原語與條件表達式
- [software4: Quicksort 演算法](software4.md) — C.A.R. Hoare 提出遞迴分割排序法，成為最廣泛使用的排序方法
- [software5: IBM 1401 程式設計](software5.md) — SPS 組譯器與 RPG 報表產生器，服務商業用戶

### 4. 網路發展
- [network1: 第一個線上訂位系統](network1.md) — 日本 NEAC 2203 為近畿日本鐵道建立的即時訂票系統
- [network2: SAGE 系統部署](network2.md) — IBM 製造的 AN/FSQ-7，275 噸重電腦，23 個站點佈署
- [network3: Bell Labs Dataphone 開發](network3.md) — 首批用於電腦通訊的數據機
- [network4: 遠端終端運算](network4.md) — IBM 026 卡片穿孔機連接中央主機的運作模式
- [network5: 早期電信網路](network5.md) — 電傳打字機網路，TWX 電傳打字交換服務
