# 1962 年資訊科技年鑑

## 目錄

### 1. 前言
- [前言](preface.md) — 1962 年資訊科技總覽：Telstar 衛星、太空戰爭遊戲、IBM 1311 磁碟機、Atlas 虛擬記憶體、COBOL 標準化

### 2. 硬體發展
- [hardware1: IBM 1311 磁碟機](hardware1.md) — 可移除式磁碟組，200 萬字容量，支援 IBM 1400/7000 系列
- [hardware2: Telstar 通訊衛星](hardware2.md) — Bell Labs 建造，首次電視/電話中繼，引爆全球衛星通訊
- [hardware3: Atlas 電腦](hardware3.md) — 曼徹斯特大學開發，首部具備虛擬記憶體及分頁概念的電腦
- [hardware4: IBM 7094](hardware4.md) — 電晶體化大型主機，32K 字磁芯記憶體，500 KIPS 效能
- [hardware5: DEC PDP-1 擴展](hardware5.md) — 共售出 53 台，配備光筆、CRT 顯示器、鍵盤輸入

### 3. 軟體發展
- [software1: SpaceWar! 遊戲](software1.md) — 第一套電子遊戲，Steve Russell 開發，MIT 駭客文化，搖桿控制器
- [software2: COBOL-61 標準](software2.md) — 首個商業語言標準，美國政府強制採用
- [software3: LISP 1.5 手冊](software3.md) — John McCarthy 出版，AI 語言定義，垃圾回收機制規範
- [software4: ALGOL 60 編譯器](software4.md) — Burroughs B5000 以硬體實現 ALGOL 堆疊架構
- [software5: IBM 1401 程式](software5.md) — Autocoder 組譯器，磁帶批次處理

### 4. 網路發展
- [network1: Telstar 衛星通訊](network1.md) — 第一顆主動式通訊衛星，橫跨大西洋電視轉播
- [network2: SAGE 網路](network2.md) — 第一個全國性電腦網路，即時雷達整合
- [network3: IBM 遠程處理](network3.md) — 遠端終端與主機間的數據通訊
- [network4: Bell Labs Dataphone 服務](network4.md) — 經由電話線的數據傳輸，300–2400 bps
- [network5: 早期線上系統](network5.md) — American Airlines SABRE 訂位系統開發

### 5. 歷史穿梭
- [歷史穿梭](history.md) — 1962 的種子：SpaceWar! 催生電子遊戲產業，虛擬記憶體革新作業系統，衛星通訊改寫全球通訊史
