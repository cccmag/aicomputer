# 歷史穿梭：1991 年的種子

## 從 Linux 到開放原始碼運動

1991 年 Linus Torvalds 的 Linux 核心發布，是開放原始碼運動的關鍵節點：

**GNU 專案 (1983)** → **Linux 核心 (1991)** → **Apache (1995)** → **Mozilla (1998)** → **GitHub (2008)**

Linux 的成功證明了分散式協作開發可以有效產出高品質的作業系統。1991 年，Torvalds 還在猶豫是否要將 Linux 商業化；一年後，他選擇了 GPL 授權——這個決定讓 Linux 走上了與 BSD 截然不同的發展路徑。

如果 Torvalds 沒有在 1992 年將 Linux 轉為 GPL，自由軟體運動的歷史可能會完全不同。

## 從 Oak 到 Java 到網際網路

James Gosling 在 1991 年啟動的 Oak 專案（後來改名為 Java），走了十年才找到真正的舞台：

**Oak (1991)** → **HotJava 瀏覽器 (1994)** → **Java 1.0 (1996)** → **Java EE (1999)** → **Android (2008)**

Java 原本是為消費電子裝置設計的語言，卻因為 World Wide Web 的爆發找到了 killer app——applet。一個原本為機上盒設計的語言，最終驅動了網際網路應用伺服器。

## Python 的漫長崛起

1991 年的 Python 0.9 只是一個小眾的腳本語言，其成長曲線極為漫長：

**Python 0.9 (1991)** → **Python 1.0 (1994)** → **Python 2.0 (2000)** → **Python 3.0 (2008)** → **AI 時代的霸主 (2010s)**

Python 的設計哲學——可讀性、顯式優於隱式、簡潔優於複雜——讓它在 30 年後成為人工智慧和資料科學的主導語言。Guido van Rossum 在 1991 年做的「歐洲風格的替代方案」，最終征服了全球。

## 視覺化程式設計的傳承

Visual Basic 在 1991 年開啟的「視覺化開發」理念，形成了一條清晰的技術路線：

**Visual Basic (1991)** → **Delphi (1995)** → **Visual Studio 完整版 (1997)** → **.NET WinForms (2002)**

VB 的拖放式介面設計模式至今仍在影響著現代開發工具——只是目標平台從 Windows 桌面轉向了 Web 和行動裝置。

## Web 的指數爆炸

1991 年的 10 個 WWW 伺服器，到 1993 年成長為 500 個，到 1994 年突破 10,000 個：

**CERN httpd (1991)** → **NCSA httpd (1993)** → **Apache (1995)** → **nginx (2004)**

1991 年 Tim Berners-Lee 在 alt.hypertext 上寫下的那則訊息，是這場指數爆炸的起點。

## 1991 年的時代脈絡

- **蘇聯解體（1991 年 12 月）**：冷戰結束，美國國防部的網路研究經費轉向商業化應用
- **World Wide Web 的誕生**：完美地與網際網路從政府網路向商業網路過渡的時間點重合
- **波斯灣戰爭（1991 年 1-2 月）**：展示了精確打擊和資訊優勢的軍事價值，推動了數位化戰爭的概念
- **消費電子產品的多媒體化**：CD-ROM 和音效卡的普及，讓電腦從文字工具轉變為多媒體平台

1991 年最驚人的地方在於：Linux、Python、Java（Oak）、Visual Basic、World Wide Web——這些主宰下一個三十年的技術——竟然都出現在同一年。
