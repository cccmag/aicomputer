# 1991 年資訊科技年鑑：前言

1991 年是資訊科技史上最關鍵的轉折點之一。這一年，網際網路從學術研究的邊緣工具，開始向大眾世界邁出了第一步。World Wide Web 在 CERN 由 Tim Berners-Lee 公開釋出，雖然當時仍只是文字模式的簡陋原型，但它預示了人類通訊方式的根本變革。

軟體領域同樣波瀾壯闊。芬蘭學生 Linus Torvalds 在 8 月 25 日於新聞群組上發布了 Linux 核心的訊息，從此開啟了開放原始碼作業系統的傳奇。荷蘭程式設計師 Guido van Rossum 在 2 月發布了 Python 語言的 beta 版，追求程式碼可讀性與簡潔性的語言哲學從此萌芽。Microsoft 在 5 月推出了 Visual Basic，讓 Windows 應用程式開發變得前所未有的簡單。而在 Sun Microsystems，James Gosling 的 Oak 專案悄然啟動——這個後來改名為 Java 的語言，將在五年後徹底改變網際網路程式的面貌。

硬體方面，Intel 的 80486 處理器達到了成熟期，可攜式電腦開始成形，CD-ROM 驅動器從奢侈品走向標準配備。Apple 推出了備受讚譽的 PowerBook 系列，重新定義了筆記型電腦的設計語言。

同年，美國國會通過了高速計算法案（High Performance Computing Act of 1991），由當時的參議員 Al Gore 倡導，為後來的美國國家資訊基礎建設（NII）奠定了政策基礎。

ACM 圖靈獎頒給了英國計算機科學家 Robin Milner，表彰他在 LCF（邏輯可計算函數）、ML 語言和 CCS（通訊系統演算）三項獨立而完整的貢獻。
