# Java 語言的濫觴：Oak 專案的起點

## 綠色專案

1991 年 4 月，Sun Microsystems 發起了一個名為「綠色專案」（Green Project）的內部研發計畫。這個由 James Gosling、Patrick Naughton 和 Mike Sheridan 三人組成的小團隊，其目標是預測「下一波運算浪潮」，結論是消費性電子裝置的數位化整合。

這些裝置——電視機上盒、錄影機、PDA——需要使用多種不同的 CPU，且記憶體和儲存空間極度有限。團隊需要一種能夠跨平台運作且體積小巧的程式語言。

## Oak 的誕生

Gosling 最初嘗試用 C++ 來開發，但很快發現 C++ 的複雜性（特別是記憶體管理和多重繼承）不適合消費電子裝置的環境。他決定從頭建立一個新語言，名為 Oak——取自他辦公室外的一棵橡樹。

Oak 的設計目標：
- **簡潔且熟悉**：採用類似 C/C++ 的語法以降低學習曲線
- **平台中立**：編譯後的位元碼可在任何平台上執行
- **安全可靠**：內建垃圾回收，無指標算術
- **物件導向**：但比 C++ 更純粹
- **支援網路**：從核心層級支援 TCP/IP

## 未來的種子

1991 年的 Oak 還只是一個原型——Gosling 在當年夏天就讓 Oak 運行了他的第一個程式。團隊在 1992 年用 Oak 打造了一個名為 Star7 的 PDA/遙控器原型裝置。

Oak 後來因商標問題改名為 Java，並在 1995 年因為 World Wide Web 的爆炸性成長而找到了 killer app——嵌入網頁的 applet。雖然 1991 年的這項開始並未引起注意，但它種下了改變網際網路應用開發模式的種子。
