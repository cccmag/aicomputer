# Python 語言：Guido van Rossum 的第一個 Beta

## 來自荷蘭的替代方案

1991 年 2 月 20 日，荷蘭數學與電腦科學研究中心（CWI）的研究員 Guido van Rossum，在 alt.sources 新聞群組上發布了 Python 語言的 beta 版本——版本號 0.9 patchlevel 1。

Python 的設計目標是創造一種「結合非凡威力與非常清晰語法的擴展性直譯程式語言」。van Rossum 在 1980 年代末期就開始在 CWI 的 Amoeba 分散式作業系統專案中開發 Python，作為 shell 腳本和 C 程式之間的橋樑。

## 設計哲學

Python 從一開始就具備了一系列區別於其他語言的設計特點：

- **用空白縮排定義區塊**：捨棄了 C 家族的大括號，強制程式設計師以一致的縮排書寫程式碼
- **動態型別系統**：變數無需宣告型別，由直譯器自動推斷
- **內建高階資料結構**：list、dictionary、tuple 等讓資料處理更直接
- **直譯器互動模式**：可即時測試程式碼片段
- **擴展模組系統**：容易用 C 語言編寫擴展模組

van Rossum 從 ABC 語言（同樣來自 CWI）繼承了許多設計理念，但他也意識到 ABC 的封閉性是其主要弱點——因此 Python 從一開始就是開放且可擴展的。

## 第一印象

最初的 Python 0.9 支援了例外處理、函式、具有繼承能力的類別、以及與多種視窗系統（X11、Mac、GL）的介面。它可以在大多數 Unix 衍生系統、Macintosh 和 MS-DOS 上運行。

當時 Perl 是腳本語言的主流選擇，但 Python 以其出色的可讀性和一致性吸引了那些重視程式碼品質的開發者。從 1991 年的這個小 beta 開始，Python 將成長為全球最受歡迎的程式語言之一。
