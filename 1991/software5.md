# World Wide Web 軟體：Berners-Lee 的釋出

## 從 NeXT 到大眾

1991 年，Tim Berners-Lee 在 CERN 完成了 World Wide Web 基礎軟體的開發和發行工作。雖然他在 1990 年底就已在 NeXT 工作站上建立了第一個網頁伺服器和瀏覽器，但 1991 年才是 Web 真正對外開放的年份。

Berners-Lee 發布的 WWW 軟體包含三個核心元件：
- **Line-mode 瀏覽器**：由實習生 Nicola Pellow 編寫，可在任何終端機上運行
- **CERN httpd**：HTTP 伺服器軟體
- **WWW Common Library**：供開發者使用的函式庫

## 時間線

1991 年的 Web 釋出時間線：
- **3 月**：Line-mode 瀏覽器對 CERN 內部使用者釋出
- **8 月 6 日**：Berners-Lee 在 alt.hypertext 新聞群組上發布了 WWW 專案的說明
- **8 月到 9 月**：陸續在 comp.sys.next、comp.text.sgml 等群組發布
- **10 月**：建立 www-interest 和 www-talk 郵件列表
- **12 月**：第一個美國 Web 伺服器在 SLAC（史丹佛線性加速器中心）上線

## 最初的網頁

第一個網站 info.cern.ch 只包含關於 World Wide Web 專案本身的資訊——如何取得瀏覽器、如何設定伺服器、以及超文字技術的簡介。網頁純粹由 HTML 編寫，沒有圖片，只有文字和連結。

1991 年結束時，全球大約只有 10 個 Web 伺服器——但這個數量在接下來的兩年中將呈指數級成長。Berners-Lee 在 1991 年所做的，不僅是釋出了一些程式碼，而是釋出了一個改變世界的概念。
