# Web 2.0 概念的提出：網路成為平台

## O'Reilly Media 的洞察

2004 年 10 月，O'Reilly Media 的創辦人 Tim O'Reilly 和 MediaLive International 的 Dale Dougherty 在一次會議腦力激盪中提出了「Web 2.0」的概念。他們觀察到 dot-com 泡沫破滅後存活下來的網路公司有一些共同的模式——這些模式不同於 1990 年代的 Web 1.0。

O'Reilly 後來在 2005 年發表了經典文章「What Is Web 2.0」，明確界定了這個概念。Web 2.0 的核心原則包括：

- **網路作為平台**：軟體不再安裝在本地電腦，而是透過瀏覽器運作
- **集體智慧**：利用用戶的集體貢獻來創造價值（如 Wikipedia、Delicious）
- **資料是新的 Intel Inside**：擁有獨特資料集的公司在競爭中佔有優勢
- **軟體發布週期的終結**：軟體持續更新，不再有版本號
- **輕量級程式設計模型**：簡單的 API 和混搭（mashup）文化
- **豐富的使用者體驗**：AJAX 技術讓網頁應用具有桌面軟體般的互動性

## 關鍵案例

O'Reilly 引用多個網站來說明 Web 2.0 原則：

- **Google**：以 PageRank（集體智慧）為核心，廣告（AdSense）為商業模式
- **Amazon**：用戶評論、推薦引擎、第三方賣家平台
- **Wikipedia**：完全由用戶集體編輯的百科全書
- **Flickr**：照片分享與社群標籤（folksonomy）
- **Blog**：每個人都可以發布內容，雙向對話取代單向廣播

## 深遠影響

Web 2.0 是 2000 年代最重要的網路概念之一。它改變了人們對網路的理解——從資訊發布平台轉變為參與式架構。它催生了社交媒體、UGC（用戶生成內容）、SaaS（軟體即服務）等模式。雖然「Web 2.0」這個詞後來被過度使用成為行銷術語，但其核心洞察——網路是一個不斷演化的參與式平台——至今仍然深刻影響著科技產業。
