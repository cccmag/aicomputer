# 2004 年資訊科技年鑑：前言

2004 年是網路服務重新定義的一年。2 月 4 日，Mark Zuckerberg 在哈佛宿舍中啟動了 TheFacebook——一個僅限哈佛學生的線上名錄。沒有人想到它將成為改變世界數十億人社交方式的平台。

4 月 1 日，Google 發布 Gmail，提供 1GB 的免費儲存空間——比當時 Yahoo Mail（4MB）和 Hotmail（2MB）高出數百倍。許多人以為這是愚人節玩笑，但 Gmail 真的來了。11 月 9 日，Mozilla Firefox 1.0 正式發布，為封閉的瀏覽器市場重新帶來了自由競爭。

硬體方面，Intel 的 Prescott 處理器遭遇了嚴重的散熱問題——90 奈米製程的漏電流讓 Pentium 4 成為「電暖器」的代名詞。這促使 Intel 放棄 NetBurst 架構，走向 Core 的路線。在日本，Nintendo DS 上市，其雙螢幕和觸控筆設計開創了遊戲機的新類別。ARM 發表了 Cortex 架構，為未來十年的行動晶片奠定了基礎。

這一年，Stewart Butterfield 和 Caterina Fake 在溫哥華創立了 Flickr，照片分享從此改變。Tim O'Reilly 提出了 Web 2.0 的概念，將網路重新定義為一個參與式平台。Adam Curry 的 iPodder 和 Daily Source Code 開啟了 Podcasting 時代——每個人都可以成為廣播員。
