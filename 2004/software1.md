# Mozilla Firefox 1.0：瀏覽器之戰的挑戰者

## 從 Netscape 灰燼中崛起

2004 年 11 月 9 日，Mozilla 基金會發布了 Firefox 1.0——一個從 Netscape 通信公司的開放原始碼專案中誕生的獨立瀏覽器。Firefox 的開發始於 2002 年，當時由 Dave Hyatt 和 Blake Ross 領導的團隊決定從臃腫的 Mozilla Suite 中分離出一個輕量級的瀏覽器。

Firefox 的第一個版本代號為 Phoenix（鳳凰，暗示從 Netscape 的灰燼中重生），後因商標問題改名 Firebird，最後定名 Firefox。這個名字來自紅貓熊的英文別名。

## 為什麼 Firefox 能挑戰 IE

2004 年，Internet Explorer 佔據了超過 95% 的瀏覽器市場，但已經多年沒有重大更新。Firefox 提供了 IE 所缺乏的多項功能：

- **分頁瀏覽**：在同一個視窗中開啟多個網頁
- **彈出式視窗阻擋**：內建且預設啟用
- **擴充套件系統**：允許使用者自訂瀏覽器功能
- **即時書籤**：支援 RSS 訂閱
- **更好的網頁標準支援**：更正確地渲染 CSS 和 HTML

## 草根行銷的典範

Firefox 的成功不僅來自技術優勢，也來自其創新的行銷方式。Mozilla 發起「Spread Firefox」活動，鼓勵使用者向朋友推薦。2004 年，Firefox 支持者在《紐約時報》刊登了整版廣告，資金全部來自小額捐款。

Firefox 1.0 發布前的最後一個預覽版被下載了超過 800 萬次。發布後，Mozilla 瀏覽器的市場佔有率在數月內從 4% 上升到 7%。Firefox 證明了開放原始碼軟體可以在消費者市場中與壟斷巨頭競爭。
