# USB 3.0 規範：高速傳輸的新標準

## SuperSpeed USB

2008 年 11 月，USB 3.0 規範正式發布——由 Intel、HP、Microsoft、NEC、NXP Semiconductors 和 Texas Instruments 組成的 USB 3.0 Promoter Group 共同制定。USB 3.0 被命名為「SuperSpeed USB」，理論最大傳輸速度達到 5 Gbps——比 USB 2.0 的 480 Mbps 快了十倍。

USB 3.0 的技術核心是增加了一組新的差分信號對（SDP，SuperSpeed Differential Pairs）。USB 2.0 使用四條線（電源、接地、D+、D-），而 USB 3.0 在相同連接器上增加了五條額外線路——兩條用於傳送（SSTX+、SSTX-）、兩條用於接收（SSRX+、SSRX-）、一條接地。這讓 USB 3.0 可以實現全雙工通訊——同時傳送和接收資料。

## 向後相容的設計

USB 3.0 保持了與 USB 2.0 和 USB 1.1 的完全向後相容。Type-A 和 Type-B 連接器的外形保持不變，但內部增加了額外的接腳。使用者可以將 USB 2.0 裝置插入 USB 3.0 埠（或相反），以較低的速度運作。

向後相容策略對 USB 3.0 的普及至關重要。使用者不需要更換周邊設備就能受益於新的主機控制器——雖然只有 USB 3.0 裝置才能達到 5 Gbps 的速度。

## 對儲存與多媒體的影響

USB 3.0 的高頻寬對外接儲存裝置的影響最為顯著。USB 2.0 的 480 Mbps 瓶頸限制了外接硬碟的實際效能。USB 3.0 讓外接 SSD 可以達到接近內部 SATA 介面的速度，外接硬碟的備份時間從數小時縮短到數十分鐘。

USB 3.0 也為高解析度視訊擷取和傳輸提供了足夠的頻寬——1080p 視訊可以流暢地在外接裝置上播放和編輯。
