# 歷史穿梭：2008 年的種子

## 行動平台的雙元競爭

2008 年是 iOS 與 Android 競爭格局確立的起點：

**App Store (7月) + iPhone 3G (7月)** → **iOS 生態系統**
**Android SDK (9月) + HTC Dream (10月)** → **Android 生態系統**

這兩個平台的競爭定義了接下來十年的行動運算市場。iOS 以封閉整合、品質控制和用戶體驗為核心；Android 以開放靈活、多樣化和價格區間廣為武器。到了 2010 年代，兩者聯手消滅了 Symbian、Windows Mobile、BlackBerry OS 和 webOS 等競爭對手。

## 瀏覽器作為平台的確立

Chrome 的 V8 引擎讓 JavaScript 的速度產生了質變，為未來的 Web 應用鋪平了道路：

**V8 (2008)** → **Node.js (2009)** → **npm 生態** → **全端 JavaScript**

Chrome 的行程隔離模式也影響了後來的瀏覽器設計——Firefox、Safari 和 Edge 都採用了類似的多行程架構。

## 從本地到雲端

Dropbox 的商業化代表了個人雲端儲存的轉折點：

**Dropbox (2008)** → **iCloud (2011)** → **Google Drive (2012)** → **雲端同步成為標準**

雲端儲存不再只是企業伺服器的概念，而是每個人都可以在日常工作中使用的工具。

## 版本控制的典範轉移

2008 年是分散式版本控制取代集中式系統的轉捩點：

**SVN (2000)** → **Git (2005) + GitHub (2007)** → **Git 成為主流 (2008-2010)**

這個轉變不僅是技術上的，也是社會上的——它改變了開發者如何協作、如何貢獻開源專案、以及如何建立自己的程式設計履歷。

## 2008 年的時代脈絡

- **全球金融危機**：2008 年雷曼兄弟倒閉引發全球金融危機，科技產業受到一定衝擊，但 Apple、Google 等公司繼續快速成長
- **美國總統大選**：Obama 當選總統，他的競選活動大量使用了社交媒體和網路募款，展示了網路的政治力量
- **科技 IPO 寒冬**：2008 年的金融危機導致科技 IPO 市場幾乎停滯——直到 2011 年 LinkedIn 上市才重新開啟
- **亞洲製造的崛起**：HTC 代表台灣製造商在全球手機市場中嶄露頭角，中國的華為和中興也開始進入手機市場
