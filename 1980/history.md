# 歷史穿梭：1980 年的種子

## 從 MS-DOS 到 Windows

Microsoft 在 1980 年與 IBM 的簽約是軟體產業史上最重要的商業決策之一：

**QDOS (1980)** → **MS-DOS/PC-DOS (1981)** → **MS-DOS 2.0 (1983)** → **Windows 1.0 (1985)** → **Windows 3.0 (1990)** → **Windows 95 (1995)**

非獨家授權條款讓 Microsoft 得以將 MS-DOS 授權給所有 IBM PC 相容機製造商——Compaq、Dell、HP 等——這不僅讓 MS-DOS 成為事實標準，也讓 Microsoft 從一家小工具軟體公司成長為世界最大的軟體公司。

## 從 Ethernet 到全球網路

DIX Ethernet 標準在 1980 年的提出，開啟了區域網路的大規模部署：

**DIX Ethernet (1980)** → **IEEE 802.3 (1983)** → **10BASE-T (1990)** → **Fast Ethernet (1995)** → **Gigabit Ethernet (1999)** → **400 GbE (2017)**

Ethernet 超越了所有競爭技術（Token Ring、FDDI、ATM），成為有線網路的事實標準。它的成功來自於持續的升級演進——從 10 Mbps 到 400 Gbps，40,000 倍的性能提升。

## 從 Smalltalk 到 Java

Smalltalk-80 的公開是物件導向程式設計普及化的關鍵轉折：

**Simula 67 (1967)** → **Smalltalk-80 (1980)** → **C++ (1985)** → **Objective-C (1984)** → **Java (1995)** → **Python (物件導向)**

Smalltalk 的虛擬機概念——編譯為平台無關的位元組碼——在 15 年後被 Java 的 JVM 重新發現並商業化成功。

## 從 Sinclair 到廉價運算

Sinclair ZX80 以 99.95 英鎊的價格證明：低價電腦可以創造巨大的市場：

**ZX80 (1980)** → **ZX81 (1981)** → **ZX Spectrum (1982)** → **Commodore 64 (1982)** → **Raspberry Pi (2012)**

Sinclair 的哲學是「足夠好」——用最少零件實現最基本功能。這種思維在四十年後的 Raspberry Pi 上重新出現：一台 35 美元的電腦讓新世代的孩子們學習程式設計。

## 1980 年的時代脈絡

- **個人電腦產業化**：Apple 的 IPO 和 Microsoft 的 IBM 合約，標誌著個人電腦從 hobby 產業轉變為主流商業
- **冷戰高峰期**：Ada 語言的開發反映了軍事需求的驅動力，ARPANET 的國際擴展與北約的戰略需求相關
- **日本經濟崛起**：日本消費電子產業在 1980 年達到高峰，美國科技公司開始感受到競爭壓力
- **辦公室自動化**：WordStar 和 dBASE II 的流行反映了企業對辦公室電腦化的需求——打字機被文字處理軟體取代，檔案櫃被資料庫取代
