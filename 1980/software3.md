# Smalltalk-80 公開：物件導向的普及

## 從 PARC 到全世界

Xerox PARC 的 Smalltalk 系統在 1970 年代一直是公司內部的實驗性平台。1979 年 Steve Jobs 的參觀讓外界首次窺見了它的威力。1980 年，Xerox 做出了重要的決定：將 Smalltalk-80 的完整規格公開。

Adele Goldberg 和 David Robson 合著了「Smalltalk-80: The Language and Its Implementation」一書，詳細描述了 Smalltalk 的語法、語意和虛擬機設計。同時，Xerox 向大學和研究機構提供了 Smalltalk-80 的參考實作——稱為「Smalltalk-80 v1」。

## 虛擬機的概念

Smalltalk-80 系統包含一套虛擬機（Virtual Machine）——將 Smalltalk 源碼編譯為位元組碼（bytecode），再由虛擬機執行。這讓 Smalltalk 程式可以跨平台執行。這個概念後來被 Java 和 .NET 廣泛採用。

Smalltalk-80 的標準映像檔（image）包含了完整的開發環境：類別瀏覽器、除錯器、效能分析工具。這是史上第一個「整合開發環境」（IDE），遠超當時以編輯器-編譯器-連結器為基礎的開發流程。
