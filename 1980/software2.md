# Ada 語言標準化：美國國防部的決策

## 從競賽到標準

1979 年美國國防部（DoD）選定 Ada 作為統一程式語言後，1980 年進入了標準化階段。這一年，DoD 發布了 MIL-STD-1815——Ada 語言的軍事標準。標準的命名「1815」是 Ada Lovelace 的出生年份。

Ada 的設計由 CII Honeywell Bull 的法國團隊主導，首席設計師是 Jean Ichbiah。整個設計過程極為嚴謹，包括數百項需求審查、多輪設計評審和測試編譯器實作。

## Ada 的技術特色

Ada 的設計目標是「可靠、高效、可維護」——特別適合即時系統、航空電子、武器系統等安全關鍵應用。主要特性包括：

- **強型別**：所有型別轉換必須明確宣告，防止隱含錯誤
- **封裝**：package 機制支援模組化程式設計
- **例外處理**：結構化的錯誤處理機制
- **多工**：內建的 task 支援，用於並行程式設計
- **泛型**：類似 C++ template 的參數化型別
