# 貝爾實驗室數據機開發：數據通訊的開端

1958 年，貝爾實驗室（Bell Labs）推出了世界上第一款商用數據機——Bell 101 Data Set。這款設備的出現使得數位資料可以透過公共電話網路進行傳輸，為電腦之間的通訊打開了一扇全新的大門。

Bell 101 數據機採用頻率偏移調變（Frequency Shift Keying, FSK）技術，以 110 bps（位元每秒）的速率傳輸資料。它的運作方式是：將二進位的「1」和「0」分別轉換為兩種不同頻率的聲音信號，透過電話聽筒的麥克風送入電話線路，接收端則將接收到的聲音信號解調回數位資料。這種方式雖然原始，但在當時的技術條件下確實有效。

貝爾實驗室的數據機開發源於一個實際需求：SAGE 防空系統需要將雷達資料從偏遠的雷達站傳送到中心電腦。傳統的類比語音通訊方式無法滿足數位資料傳輸的需求，因此貝爾實驗室開始研究如何在撥號電話網路上進行可靠的資料傳輸。

這項研究面臨了多重技術挑戰。首先，電話網路是針對語音通訊設計的，其頻率範圍（300-3,400 Hz）對資料傳輸的限制很大。其次，長距離電話線路中的雜訊和衰減會導致資料錯誤。第三，數據機需要能夠與標準的電話交換系統相容，才能實現撥號連接。貝爾實驗室的研究團隊逐一解決了這些問題，設計出了實用的數據機設備。

Bell 101 的後繼者 Bell 103（1962 年推出）進一步將速率提升到 300 bps，並建立了全雙工通訊能力（同時發送和接收）。貝爾實驗室制定的數據機標準成為事實上的業界規範，其他製造商的數據機必須與其相容才能在電話網路上使用。

數據機的發明是電腦網路發展史上的關鍵技術突破。沒有數據機，電腦就無法透過現有的電話網路進行通訊，區域網路和廣域網路的發展也將無從談起。ARPANET 的節點之間正是使用改良後的數據機進行通訊。從 110 bps 的 Bell 101 到後來 56K bps 的 V.90 數據機，這項技術陪伴了電腦網路近四十年，直到寬頻時代才逐步退場。
