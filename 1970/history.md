# 歷史穿梭：1970 年的種子

## 從關聯式模型到現代資料庫

Codd 的關聯式模型（1970）並非無中生有。它的數學根源來自於 19 世紀的集合論和關係邏輯，直接的前驅包括 Childs（1968）的集合論資料結構研究。

**Childs (1968) → Codd 的關聯式模型 (1970) → System R (1974) → SQL 標準化 (1986) → Oracle、MySQL、PostgreSQL**

資料庫的發展史展示了純粹數學理論如何在數十年後轉化為千億美元的產業。Codd 的關聯式代數直接催生了 SQL——這個語言至今仍是世界上最重要的程式語言之一。每次你在手機上查詢資料，背後都是一次 Codd 式關係運算。

## PDP-11、UNIX 與 C 語言的黃金三角

1970 年的三件技術——PDP-11 硬體、UNIX 移植、B 語言——構成了現代計算的基礎：

**PDP-11 硬體架構 → UNIX 作業系統 → C 語言 → 現代所有作業系統**

x86 處理器的暫存器模型直接繼承自 PDP-11。Linux、macOS、Windows 的核心設計概念都來自 UNIX。C 語言的語法影響了 C++、Java、C#、JavaScript 和 Go。

## 從結構化程式設計到軟體工程

Wirth 的 Pascal（1970）和 Dijkstra 的結構化程式設計運動（Go To Statement Considered Harmful, 1968）共同推動了軟體工程的誕生：

**ALGOL 60 → Pascal (1970) → Modula-2 (1979) → Oberon (1988)**

Pascal 的型別系統影響了 Ada（美國國防部強制使用）、Java 和 Swift。Wirth 的「逐步求精」方法論是敏捷開發的雛形——先做對，再做快。

## DRAM 與摩爾定律的兌現

Intel 1103（1970）是摩爾定律在記憶體領域的第一次大規模驗證：

**磁芯記憶體 (1950s) → Intel 1103 DRAM (1970) → 4K DRAM (1973) → 16K DRAM (1974) → 現代 GB 級 DRAM**

1 Kb 的 1103 在 1970 年售價 60 美元，如今 16 GB 的 DDR5 售價不到 100 美元。容量增加了 1.28 億倍，價格下降了 99% 以上。

## Xerox PARC 與個人電腦的遠景

**SRI 的 NLS 系統 → Xerox PARC (1970) → Alto (1973) → Apple Macintosh (1984) → Windows → 今日的每個桌面**

PARC 成立於 1970 年，雖然它當時還沒有產出具體成果，但它的成立本身就是一個歷史轉折點。Xerox 給予了研究員們前所未有的自由和資源，而他們回報以改變世界的發明。

## 1970 年的時代脈絡

1970 年的技術發展與當時的社會脈動密不可分：
- **越戰與冷戰**：ARPANET 的經費來自國防部，反映了軍事對分散式通訊的需求
- **太空競賽的後續**：NASA 對計算能力的需求推動了半導體技術的商業化
- **企業自動化**：IBM System/370 的面世反映了大型企業對資訊系統的依賴日益加深
- **反文化與個人解放**：參與 ARPANET 的研究生們和 PARC 的研究員們代表了一種新的、非官僚的創新文化

1970 年最值得記住的是：這一年所有重要的技術——關聯式資料庫、Pascal、UNIX on PDP-11、DRAM——在誕生之初都沒有被認為即將改變世界。只有回顧歷史，我們才能看清這些種子的力量。
