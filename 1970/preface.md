# 1970 年資訊科技年鑑：前言

1970 年是電腦科學從理論走向實用，從實驗室走向產業的轉折之年。

硬體方面，DEC 推出了 PDP-11/20，這台 16 位元迷你電腦以其正交指令集和 Unibus 架構，成為史上最有影響力的迷你電腦之一。Intel 發表了 1103 DRAM 晶片，這是第一款商用動態隨機存取記憶體，開始取代昂貴且笨重的磁芯記憶體。IBM 在六月宣布了 System/370 系列，為大型主機帶來了新的效能標竿。

軟體領域同樣精彩。Niklaus Wirth 在蘇黎世聯邦理工學院完成了 Pascal 編譯器，為結構化程式設計奠定了語言基礎。Edgar Codd 在 IBM 聖荷西實驗室發表了「A Relational Model of Data for Large Shared Data Banks」，這篇論文將徹底改變資料庫的運作方式。在貝爾實驗室，Ken Thompson 將 UNIX 從 PDP-7 移植到 PDP-11，並開發了 B 語言。

網路方面，ARPANET 的 Network Control Program (NCP) 在 12 月 15 日完成標準化部署，第一套主機對主機的通訊協定正式上線。Xerox 在加州帕羅奧圖成立了 PARC 研究中心，日後這裡將誕生圖形使用者介面、雷射印表機、乙太網路等改變世界的發明。

1970 年最令人驚嘆的，或許是這些技術之間的交互影響：PDP-11 催生了 UNIX，UNIX 催生了 C 語言，關聯式資料庫改變了商業運作，而所有這些都將在未來的個人電腦革命中交匯。
