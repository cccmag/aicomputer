# 歷史穿梭：1972 年的爆發

## C 語言的傳承：從 B 到 Linux

1972 年 C 語言的誕生，啟動了電腦史上最成功的語言傳承路線：

**BCPL (1967)** → **B (1969)** → **C (1972)** → **C++ (1983)** → **Java (1995)** → **C# (2000)** → **Go (2009)** → **Rust (2015)**

C 的影響遠遠超出了這些直接繼承者。幾乎所有現代語言——Python、Ruby、Perl、PHP、JavaScript——的執行時期都是 C 寫的。作業系統核心（Linux、Windows、macOS）仍然以 C 為主體。嵌入式系統、遊戲引擎、瀏覽器核心——整個數位世界的底層都是由 C 撐起的。

Dennis Ritchie 在 1983 年與 Ken Thompson 共同獲得圖靈獎。如果沒有 1972 年那個在 PDP-11 上誕生的語言，整個計算機歷史將會完全不同。

## 遊戲產業的誕生

1972 年同時見證了電子遊戲的兩個起點：

**Magnavox Odyssey (家用主機)** → **Atari 2600 (1977)** → **NES (1983)** → **PlayStation (1994)** → **Xbox (2001)**

**Pong (街機)** → **Space Invaders (1978)** → **Pac-Man (1980)** → **整個街機黃金時代**

Ralph Baer 和 Nolan Bushnell 之間的專利訴訟，最終確立了 Baer 作為「電子遊戲之父」的法律地位。但更重要的是，他們共同開創了一個數十年後將超過 2000 億美元的全球產業。

1972 年也連結了一個有趣的巧合：Steve Jobs 在這年加入了 Atari，成為 Bushnell 的員工。在 Atari 的經歷——特別是參觀 Xerox PARC 時看到的 Alto——深刻影響了他後來對 Apple 的願景。

## 網路的三股力量匯流

1972 年網路世界發生了三個看似獨立的事件，但它們將在兩年後交會：

- **ALOHAnet 啟發**：隨機存取協定 → Ethernet (1973) → 區域網路
- **Cerf/Kahn 相遇**：網路互連設計 → TCP/IP (1974) → 網際網路
- **ICCC 展示**：ARPANET 公開亮相 → 國際連線 (1973) → 全球網路

這三條路線的匯流創造了我們今日所知的網際網路。ALOHAnet 的隨機重傳概念被 Ethernet 和 TCP 的擁塞控制繼承；Cerf 和 Kahn 的端到端原則成為網路架構的基石；ICCC 展示則讓全世界意識到分組網路的潛力。

## 從微處理器到個人電腦

Intel 8008 (1972) 並不是一顆成功的晶片——它太慢、需要太多支援晶片、市場反應平平。但是：

**Intel 4004 (1971)** → **Intel 8008 (1972)** → **Intel 8080 (1974)** → **Altair 8800 (1975)** → **Apple II (1977)** → **IBM PC (1981)**

8008 證明了 8 位元微處理器是可行的。它讓 Gary Kildall 開發了 PL/M 語言和 CP/M 作業系統——後者成為第一個微電腦作業系統標準。而 CP/M 的 BIOS 概念，直接影響了後來 PC 的韌體架構。

## 1972 年的時代脈絡

1972 年的科技創新並非發生在真空中：

- **越戰與抗議**：ILLIAC IV 從伊利諾大學遷移到 NASA 艾姆斯，反映了反戰抗議對校園研究的影響
- **冷戰通訊需求**：ARPANET 的成長和 TCP 的設計，背後是美軍對冗餘、可靠通訊系統的需求
- **消費主義崛起**：Odyssey 和 Pong 代表了電子產品從軍工和企業市場向大眾消費市場的轉變
- **反文化與個人解放**：Alan Kay 的 Dynabook 願景和 Papert 的教育理念，與強調個人創造力的反文化思潮共鳴

1972 年最令人驚嘆的是：這一年誕生的 C 語言、微處理器、網路協定和電子遊戲，共同勾勒了未來半個世紀數位世界的藍圖。
