# 1972 年資訊科技年鑑：前言

1972 年是電腦科學史上轉折性的一年。這一年，程式語言的歷史被徹底改寫——Dennis Ritchie 在貝爾實驗室創造了 C 語言，為 Unix 作業系統的移植性鋪平了道路，並奠定了此後半個世紀系統軟體開發的基石。同年，Ken Thompson 和 Dennis Ritchie 開始用 C 語言重寫 Unix，使得這個原本只用組合語言撰寫的作業系統首次具備了跨平台的能力。

消費電子領域同樣發生了革命性的變化。Magnavox Odyssey 作為第一台家用遊戲主機在聖誕季上市，而 Atari 的 Pong 街機則將電子遊戲從實驗室帶入了酒吧與遊樂場，創造了一個全新的娛樂產業。前者源自 Ralph Baer 在 1966 年公車站筆記本上的靈感——一個用電視玩遊戲的想法。

硬體方面，ILLIAC IV——這台備受爭議的超級電腦終於在伊利諾大學開始運作，雖然延宕多年且成本超支，但它的 SIMD（單指令多資料流）架構為日後的向量處理和 GPU 設計提供了重要的先例。HP-35 手持科學計算機的推出，讓工程師從笨重的桌上計算機中解放出來，改變了計算工具的型態。

網路技術也在這一年迎來關鍵突破。Vint Cerf 與 Bob Kahn 在夏威夷的 ALOHAnet 研討會上開始構思新一代的網路互連協議，最終催生了 TCP/IP 的設計。這項工作直接回應了 ARPANET 日益增長的異質網路互連需求。

1972 年證明了，最具影響力的創新往往來自於對既有問題的全新思考——無論是 C 語言的簡潔設計、Pong 的直覺遊戲性，還是 TCP/IP 的分層架構。
