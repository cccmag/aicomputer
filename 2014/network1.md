# Microsoft Azure 的大規模成長

## Satya Nadella 的雲端願景

2014 年是 Microsoft Azure 從追隨者成長為雲端市場有力競爭者的轉折點。2 月，Satya Nadella 被任命為 Microsoft 第三任 CEO。Nadella 在就職信中明確提出了「雲端優先」（Cloud First）的戰略方針。

Nadella 的願景與 Steve Ballmer 時代截然不同。Ballmer 將 Azure 定位為 Windows Server 的雲端延伸，而 Nadella 認為 Azure 應該是一個開放的雲端平台——支援 Linux、支援 Docker 容器、支援開源軟體。

## Azure 的關鍵發布

2014 年，Microsoft 推出了多項重要的 Azure 服務：Azure SQL Database 的效能大幅提升、Azure Machine Learning 進入預覽、Azure 的容器支援（與 Docker 合作）開始測試。Microsoft 也推出了 Azure 的「企業級」承諾——99.95% SLA、企業級安全合規認證、以及與既有 Microsoft 產品的無縫整合。

## 營運成長

Microsoft 在 2014 年開始公布 Azure 的營收數據。Azure 的季度營收在 2014 年實現了超過 100% 的年成長率。雖然相較於 AWS（先發優勢明顯）的市場規模仍有相當差距，但 Azure 的成長速度已讓業界注意到 Microsoft 在雲端領域的威脅。

Azure 的成功也得益於 Microsoft 龐大的企業客戶基礎。全球有數百萬個組織使用 Microsoft 的企業產品——Windows Server、SQL Server、Active Directory、Office——這些客戶自然地被引導到 Azure 平台。
