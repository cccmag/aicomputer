# 5G 願景與研究啟動

## 下一代行動網路的藍圖

2014 年，雖然 4G LTE 仍在全球大規模部署中，學術界和產業界已經開始討論下一代行動通訊技術——5G。ITU（國際電信聯盟）在 2014 年啟動了 IMT-2020 計畫，為 5G 設定性能目標和技術框架。

5G 的設計目標遠不止是「更快的 4G」。ITU 定義了三大應用場景：增強型行動寬頻（eMBB）、超可靠低延遲通訊（URLLC）、以及大規模機器類型通訊（mMTC）。這表示 5G 不僅要服務手機用戶，還要支援自動駕駛（低延遲）、工業自動化（高可靠）和物聯網（大量連接）。

## 技術路線的探索

2014 年，學術界和標準化組織開始探索 5G 的關鍵技術：
- **毫米波（mmWave）**：利用 30-300 GHz 的頻段提供超大頻寬
- **大規模 MIMO**：在基地台部署數百根天線，大幅提升頻譜效率
- **小型基地台（Small Cell）**：密集部署小型基地台以提升覆蓋和容量
- **網路功能虛擬化（NFV）和軟體定義網路（SDN）**：讓行動網路更加靈活和可程式化

## 歐洲與亞洲的競賽

2014 年，歐盟啟動了 5G PPP（Public-Private Partnership）研究計畫，投入 14 億歐元用於 5G 研發。中國也啟動了 IMT-2020（5G）推進組，由工信部、發改委和科技部聯合支持。南韓和日本在 5G 測試上也展現了積極態度。
