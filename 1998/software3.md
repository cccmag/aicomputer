# Java 2 Platform：企業級 Java 的誕生

## 三版分立

1998 年 12 月，Sun Microsystems 發表了 Java 2 Platform（版本 1.2），這是 Java 發展史上最重要的版本之一。Java 2 引入了一個重要的分類架構，將平台劃分為三個版本：

- **Java 2 Standard Edition (J2SE)**：標準版，針對桌面應用和 applet
- **Java 2 Enterprise Edition (J2EE)**：企業版，包含 EJB、Servlet、JSP、JMS 等企業級 API
- **Java 2 Micro Edition (J2ME)**：微型版，針對手機、PDA 和嵌入式裝置

這個分類架構一直沿用到 Java 5（2004 年），J2SE/J2EE/J2ME 的命名才被 Java SE/EE/ME 取代。

## Java 2 的技術創新

Java 2 引入了多項重要的語言和 API 改進：

- **Swing GUI 工具包**：取代了 AWT，提供了更豐富、更一致的外觀
- **Collections Framework**：List、Set、Map 等資料結構的標準 API
- **JIT（即時編譯器）**：將位元組碼即時編譯為機器碼，大幅提升執行效能
- **Java 2D API**：進階的 2D 繪圖和影像處理
- **拖放支援**：系統級的拖放功能
- **安全性改進**：更精細的安全權限控管

## 企業市場的奠基

Java 2 Enterprise Edition 的推出是 Java 征服企業市場的關鍵。J2EE 定義了一系列的標準 API：
- **Enterprise JavaBeans (EJB)**：分散式元件模型
- **Java Servlet**：伺服器端 HTTP 處理
- **JavaServer Pages (JSP)**：動態網頁模板
- **Java Message Service (JMS)**：非同步訊息傳遞
- **Java Naming and Directory Interface (JNDI)**：命名與目錄服務
