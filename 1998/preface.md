# 1998 年資訊科技年鑑：前言

1998 年是網路商業化從狂熱走向初步理性的一年，也是開放原始碼運動站上舞台中央的一年。

9 月 4 日，Larry Page 和 Sergey Brin 正式註冊 Google 公司。這個從史丹佛大學宿舍出發的搜尋引擎專案，即將改變人類獲取資訊的方式。

Apple 在 5 月發表 iMac——Steve Jobs 回歸後的第一款重要產品。iMac 以半透明的 Bondi Blue 外殼、整合 USB、捨棄軟碟機的大膽設計，不僅拯救了 Apple，也重新定義了電腦的工業設計。

Microsoft 推出 Windows 98，內建 USB 支援與 FAT32 檔案系統，鞏固了 PC 桌面市場。同時，美國司法部對 Microsoft 的反壟斷訴訟持續升溫。

Java 2 Platform 發布，將 Java 劃分為 Standard Edition、Enterprise Edition 與 Micro Edition，確立了 Java 在企業級開發中的地位。

開放原始碼運動在這一年爆發。Red Hat 在納斯達克上市，創下 Linux 公司 IPO 的紀錄。Linux 生態系統從駭客的玩具成長為企業可以信賴的平台。LAMP 架構（Linux、Apache、MySQL、PHP）開始成形。

Shawn Fanning 開始開發 Napster——第一個大規模點對點檔案分享系統。雖然 Napster 在 1999 年才正式上線，但它的雛形在 1998 年就已出現，預示了音樂產業即將面臨的顛覆。

1998 年是科技史上承先啟後的一年：Google 讓搜尋變聰明，iMac 讓設計變美，Linux 讓軟體變自由。
