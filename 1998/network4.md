# Linux 生態爆發：LAMP 架構成形

## 開放原始碼的拐點

1998 年是 Linux 和開放原始碼從邊緣走向主流的拐點。這一年發生了多件關鍵事件：

- **1 月**：Netscape 宣布開放 Navigator 原始碼（Mozilla 專案）
- **2 月**：OSDL（Open Source Development Lab）由多家公司聯合創立
- **3 月**：第一屆 O'Reilly Open Source Convention
- **4 月**：Corel 宣布移植 WordPerfect 到 Linux
- **8 月**：Red Hat IPO 成功
- **12 月**：Linux 核心 2.2.0 pre-release

## LAMP 架構的形成

LAMP 是 Linux、Apache、MySQL、PHP（或 Perl/Python）的縮寫。這個組合在 1998 年開始成為 Web 伺服器的事實標準：

- **Linux**：作業系統，免費、穩定、高效
- **Apache**：Web 伺服器，擁有超過 50% 的市場佔有率
- **MySQL**：開源資料庫，速度優先
- **PHP**：伺服器端腳本語言，與 Apache 緊密整合

LAMP 的競爭對手是微軟的 WAMP（Windows + IIS + SQL Server + ASP）。LAMP 的優勢是零軟體授權成本，這對初創公司和中小企業極具吸引力。

## 企業採用

1998 年，一些重要企業開始公開支援 Linux：

- **IBM**：宣布將 Linux 移植到 S/390 大型主機
- **Oracle**：正式宣布 Oracle 8 支援 Linux
- **Dell**：開始預裝 Red Hat Linux 銷售
- **Intel**：投資 Red Hat 並最佳化 Linux 對 Pentium 系列的支援
