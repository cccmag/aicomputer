# Windows 98：USB 與 FAT32 的時代

## 從 OSR2 到正式版本

1998 年 6 月 25 日，Microsoft 發布了 Windows 98。與 Windows 95 從 DOS 到 Windows 的跳躍性升級不同，Windows 98 被視為 Windows 95 的改良版本——但它引入了幾個重要的新功能。

Windows 98 的開發代號是「Memphis」。它的核心與 Windows 95 OSR2 相同，主要差別在於驅動程式支援和桌面功能整合。

## 主要新功能

**USB 支援**：Windows 98 是第一個原生支援 USB 的消費級 Windows 版本。它包含了 USB 控制器、USB 集線器和 USB 裝置（鍵盤、滑鼠、印表機、掃描器）的標準驅動程式。在此之前，USB 裝置在 Windows 95 上需要廠商自行提供驅動程式，導致相容性極差。

**FAT32 支援**：Windows 98 的安裝程式支援建立 FAT32 分割區。FAT32 允許單一分割區超過 2.1 GB——這個限制在 Windows 95 的原始版本中令人沮喪（雖然 OSR2 已有 FAT32 支援）。

**Windows Driver Model (WDM)**：新的驅動程式架構，讓同一個驅動程式可以在 Windows 98 和 Windows NT 5.0（後來的 Windows 2000）上執行。

**活動桌面 Active Desktop**：繼承自 IE 4.0 的桌面整合功能。

**系統工具改進**：磁碟清理、System Information、Windows Update（首次引入）。

## 司法部的陰影

Windows 98 的發布正值美國司法部對 Microsoft 的反壟斷訴訟升溫之時。政府律師試圖阻止 Windows 98 的發布，理由是 IE 與 Windows 的捆綁違反了 1995 年的同意判決。Microsoft 最終獲得了發布許可，但反壟斷訴訟一直持續到 2001 年。
