# Windows 11 發布與新硬體標準

## Windows 11 的誕生

2021 年 6 月 24 日，Microsoft 正式發表 Windows 11，並於 10 月 5 日開始推送——這是 Microsoft 六年來首個主要 Windows 版本。Windows 11 的設計重點是重新打造的介面、提高生產力、以及對混合工作模式的最佳化。

## 設計變革

Windows 11 採用了全新設計語言：
- **置中開始功能表和工具列**：類似 macOS 和 Chrome OS 的風格
- **圓角視窗**：整個系統的 UI 元素都採用了圓角設計
- **Widgets 面板**：由 AI 驅動的個人化資訊面板
- **Snap Layouts/Snap Groups**：視窗佈局管理的重大改進
- **虛擬桌面**：支援不同的桌布和設定

## 爭議性的硬體要求

Windows 11 引入了嚴格的硬體要求，引發了廣泛爭議：

- **TPM 2.0（可信平台模組）**：所有 Windows 11 裝置必須支援 TPM 2.0
- **第八代 Intel Core 或 AMD Ryzen 2000 以上**：排除了大量舊裝置
- **4GB RAM + 64GB 儲存**：比 Windows 10 的最低要求更高
- **Secure Boot**：必須啟用

這些要求讓許多 Windows 10 電腦無法升級。Microsoft 的理由是安全性和可靠性，但批評者認為這減少了使用者的選擇，並可能產生大量的電子廢棄物。

## Android 應用支援

Windows 11 最令人意外的功能是原生支援 Android 應用程式——透過 Intel Bridge 技術和 Amazon Appstore 實現。這項功能得到了開發社群的好評，但由於支援的應用數量有限和效能問題，實際使用率並不高。

## 市場反響

Windows 11 的初期採用率低於 Microsoft 的預期——截至 2021 年底，只有約 10-15% 的 Windows 使用者升級到 Windows 11。許多使用者選擇留在 Windows 10（支援至 2025 年），而硬體要求限制了舊裝置的升級。
