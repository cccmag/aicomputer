# 歷史穿梭：2021 年的種子

## 混合架構的紀元

Intel Alder Lake 的大小核設計與 Apple M1 Pro/Max 的統一記憶體架構代表了兩種不同的效能提升哲學：

**Intel Alder Lake**：在同一個晶片中混合不同類型的核心（P-cores + E-cores），由作業系統的 Thread Director 根據任務需求動態調度。

**Apple M1 Pro/Max**：使用同質高性能核心但擴展核心數量，依靠統一記憶體架構來消除 GPU 和 CPU 之間的資料傳輸瓶頸。

這兩條路線在 2022 年分別被 Intel 的 Raptor Lake 和 Apple M2 系列繼承。

## ARM 的全面擴張

2021 年是 ARM 架構從行動端全面擴張到伺服器端和 PC 端的關鍵年：

**行動端**：ARM 的主場——全球 95%+ 的手機使用 ARM 核心
**PC 端**：Apple M1 Pro/Max 證明了 ARM 在性能上可以超越 x86
**伺服器端**：AWS Graviton2/3 展示了 ARM 的性價比優勢
**超級電腦**：Fugaku（全球第一，2020-2022）使用 ARM 核心

## 從 Web 2.0 到元宇宙

Facebook 更名為 Meta 是社交媒體產業從「平面 2D 網路」向「3D 沉浸式網路」轉型的宣言：

**Web 1.0 (1990s)** → **Web 2.0 (2000s-2020)** → **元宇宙 (2021-)**

但元宇宙的實現比預期更加困難。VR 頭戴裝置的使用率、技術瓶頸和內容生態系統的缺乏，使得 2021 年被稱為「元宇宙元年」的同時，也被後續幾年的現實所考驗。

## Rust 的系統語言之路

Rust 的發展歷程是一條從 Mozilla 研究專案到 Linux 核心的漫長道路：

**Rust 首次編譯器 (2009)** → **Rust 1.0 (2015)** → **Mozilla 裁員影響 (2020)** → **Rust 基金會成立 (2021)** → **Linux 核心支援 (2021)**

Rust 的成功反映了軟體產業對記憶體安全問題的日益重視——C 和 C++ 的靈活性伴隨著巨大的安全風險，而 Rust 在不犧牲性能的前提下解決了這個問題。

## 從 FLoC 到 Topics 的隱私之殤

Google 的廣告技術演進說明了在隱私保護和商業模式之間尋找平衡的困難：

**第三方 Cookie (1994)** → **FLoC (2021)** → **Topics API (2022)**

每一次變革都面臨來自廣告商、隱私倡導者和監管機構的多方壓力。未來沒有人知道的確切藍圖。

## 2021 年的時代脈絡

- **COVID-19 疫苗普及**：mRNA 疫苗的全球部署證明了科技在解決全球危機中的力量
- **電動車爆發**：全球電動車銷量在 2021 年翻倍，Tesla 市值突破 1 兆美元
- **半導體短缺**：全球晶片荒影響了從汽車到遊戲主機的各個行業
- **加密貨幣波動**：比特幣在 2021 年 11 月達到 68,000 美元的歷史高點
- **太空旅遊**：Richard Branson 和 Jeff Bezos 先後進行了太空飛行
