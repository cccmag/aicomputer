# NFT 熱潮：非同質化代幣的瘋狂與爭議

## NFT 的爆發

2021 年，NFT（Non-Fungible Token，非同質化代幣）從區塊鏈小眾社群的玩具變成了全球性的文化現象。NFT 是一種基於區塊鏈（主要是以太坊）的數位憑證，用於證明對數位資產（圖片、影片、音樂、甚至一條推文）的所有權。

## 標誌性事件

2021 年 3 月，藝術家 Beeple（Mike Winkelmann）的數位藝術作品《Everydays: The First 5000 Days》在佳士得以 6930 萬美元拍出——這是 NFT 藝術進入主流視野的里程碑。

其他重要事件：

- **CryptoPunks**：24x24 像素的頭像系列——最高成交價超過 1100 萬美元
- **Bored Ape Yacht Club（BAYC）**：10,000 隻卡通猿猴——成為名人最愛的 NFT 系列
- **NBA Top Shot**：NBA 比賽精彩片段的 NFT——LeBron James 的精華片段以 20 萬美元售出
- **Jack Dorsey 的第一條推文**：以 290 萬美元拍出

## 技術基礎

最常見的 NFT 標準是 ERC-721（以太坊上的代幣標準）。每個 NFT 都有一個獨特的 token ID，儲存在以太坊區塊鏈上，包含指向實際數位資產（通常儲存在 IPFS 或中心化伺服器上）的後設資料。

2021 年，NFT 主要建立在以太坊上——但高昂的 Gas 費用和交易延遲成為瓶頸。這促進了其他區塊鏈（如 Solana、Polygon 和 Tezos）上 NFT 生態的發展。

## 經濟與社會爭議

NFT 引發了多方面的批評：

- **投機泡沫**：大多數 NFT 專案缺乏內在價值，價格主要由炒作驅動
- **抄襲與詐騙**：大量未經授權的 NFT（利用他人作品）充斥市場
- **環境影響**：以太坊的權益證明（PoW）共識機制耗能巨大——雖然 2022 年以太坊轉向權益證明（PoS）後這個問題大幅改善
- **洗售交易**：研究顯示大量 NFT 交易是同一人自買自賣以抬價
- **版權模糊**：購買 NFT 通常不包含數位資產的版權——你買到的只是一個指針

## 對創作者經濟的影響

儘管爭議不斷，NFT 在創作者經濟中開創了新的可能性——藝術家可以直接將作品賣給收藏者，保留版稅，並在二次交易中持續獲得收益。這在某種程度上顛覆了傳統藝術市場的中介模式。
