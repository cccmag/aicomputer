# 2021 年資訊科技年鑑：前言

2021 年是元宇宙、晶片競賽與 AI 規模化交織的一年。Facebook 改名為 Meta，將元宇宙從科幻概念變成科技產業的年度關鍵字。NFT 從區塊鏈小眾圈層衝入主流視野，藝術家和投機者共同掀起了一場數位資產狂歡。

在晶片領域，Intel 推出了革命性的 Alder Lake（大小核混合架構），這是 x86 處理器設計數十年來最重大的變革。Apple 則以 M1 Pro 和 M1 Max 證明了 M1 不是曇花一現——這些晶片將專業級工作站的性能帶入了輕薄筆記型電腦。AWS 的 Graviton 處理器說明了 ARM 在伺服器市場的勢頭正在加速。

軟體世界見證了 Python 超越 Java 成為 TIOBE 指數第二名，以及 Rust 正式進入 Linux 核心——這兩個事件標誌著系統程式語言版圖正在發生轉變。Google 從 FLoC 轉向 Topics API，反映了隱私保護與廣告商業模式之間的持續角力。

COVID-19 疫苗數位證明成為國際旅行的必需品，而 Windows 365 Cloud PC 則將作業系統本身變成了雲端服務。Tesla FSD Beta 的公開測試讓自動駕駛技術進入了大規模實際驗證階段。

硬體與軟體的界線持續模糊——每一個晶片都有其專屬的 AI 加速器，每一套軟體都在吞噬更多的硬體功能。
