# 歷史穿梭：1984 年的種子

## Macintosh 的 GUI 王朝

1984 年的 Macintosh 確立了圖形使用者介面作為個人電腦標準的方向。這條路線的傳承明確：

**Xerox Alto (1973) → Apple Lisa (1983) → Macintosh (1984) → Windows (1985) → 現代桌面與行動介面**

Macintosh 的設計理念——位元映射螢幕、滑鼠操作、一致的使用者體驗——不僅影響了後續的作業系統，更塑造了人機互動的學科。沒有 Macintosh，就沒有今日的圖形化計算環境。

## 自由軟體運動的起點

GNU 專案的啟動（1984）是軟體產業歷史上的根本性轉折。它將軟體從純粹的商品重新定義為知識與協作的載體：

**MIT AI Lab 的駭客文化 → GNU 專案 (1984) → GPL 授權 → Linux (1991) → 開源軟體產業**

Stallman 在 1984 年做出的選擇——辭職、寫程式、建立法律框架——改變了軟體世界的權力結構。如今，開源軟體已經成為網際網路基礎設施的骨幹。

## RISC 的革命從紙上走向矽

1984 年，RISC 架構從學術論文（Patterson 1980, Hennessy 1981）走向了商業化設計。MIPS R2000 的架構在這一年確定：

**IBM 801 (1975-1980) → Berkeley RISC (1980-1984) → Stanford MIPS (1981-1984) → MIPS R2000 (1986) → ARM (1985) → RISC-V (2010)**

今天世界上 99% 的處理器——從手機到物聯網——都是 RISC 架構。1984 年是這個趨勢從論文變成矽片的關鍵年份。

## 從光碟到多媒體

CD-ROM 在 1984 年的商業化預示了多媒體時代的來臨。雖然當時 CD-ROM 驅動器仍然昂貴且稀少，但它的容量優勢——相對於軟碟的千倍——遲早會改變軟體發行和內容出版的規則。

**CD 音響 (1982) → CD-ROM (1984) → 多媒體 PC (1990s) → DVD (1996) → 串流媒體 (2000s)**

## 1984 年的時代脈絡

1984 年 Apple 的「1984」廣告捕捉了時代精神：對抗大企業的壟斷、個人解放、科技作為文化力量。但同時，AT&T 的拆分和 UNIX 的商業化也說明了另一面：即使是最開放的文化產物，最終也會被市場力量所改變。Macintosh 讓 GUI 成為主流，但 IBM PC 相容機仍然掌握著大部分市場。1984 年告訴我們：真正的變革是需要理想、商業和文化同時發生的。
