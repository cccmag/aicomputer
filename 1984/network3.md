# CD-ROM 標準與多媒體革命

## 黃皮書的制定

1984 年，Philips 和 Sony 發布了 CD-ROM 的正式規格——黃皮書（Yellow Book）。這個標準定義了 CD-ROM 的實體格式（與音樂 CD 紅皮書相容）、資料組織方式和錯誤修正機制。

CD-ROM 的資料儲存採用與音訊 CD 相同的實體格式：12 公分光碟，650MB 容量，使用雷射讀取。不同之處在於，黃皮書為資料增加了兩層獨立錯誤修正：CIRC（Cross-Interleaved Reed-Solomon Code）和第三層 ECC（Error Correction Code），確保資料讀取的準確性。

## 多媒體的載體

CD-ROM 是 1980 年代「多媒體」概念的硬體基礎。它的大容量——相對於軟碟的幾百 KB——讓數位化的聲音、圖像和影片可以同時存在於同一張碟片上。

Grolier 電子百科全書在 1984 年推出 CD-ROM 版本，將 21 冊百科全書的內容——包括圖片、地圖和表格——放入一張碟片。搜尋功能讓使用者可以在幾秒鐘內找到需要的資訊，這是紙本百科無法比擬的。

## 產業鏈的形成

CD-ROM 的商業化催生了全新的產業：CD-ROM 驅動器製造商（Philips、Sony、松下）、光碟壓製廠、多媒體內容出版商、以及後來的 CD-ROM 遊戲產業。1990 年代初期，「多媒體 PC」（MPC）成為標準個人電腦的配置，CD-ROM 是其中不可或缺的元件。
