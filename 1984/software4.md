# Microsoft Word for Mac：跨平台文書處理

## 從 DOS 到 Mac

1984 年，Microsoft 展示了其跨平台策略的早期版本：在 Macintosh 上市前夕，Microsoft 宣布了 Word for Macintosh 的開發計畫。這是 Microsoft 首次在非 Microsoft 平台上推出旗艦產品——而且是在 Apple 的 GUI 平台上。

Word for Mac 的開發由 Richard Brodie 和 Randy Wigginton 領導，後者正是 MacWrite 的開發者之一。Microsoft 的目標非常明確：Word 要成為 Mac 上最好的文書處理器——甚至比 Apple 自己的 MacWrite 更好。

## 市場競爭

Word for Mac（1985 年 1 月上市）與 MacWrite 的競爭是文書處理器歷史上的重要戰役。Word for Mac 提供了 MacWrite 缺乏的功能：多文件視窗、搜尋取代、字數統計、更強大的格式控制。它還支援從 Word for DOS 匯入文件，提供了跨平台的檔案相容性。

Microsoft 的策略是「先在 Mac 上練兵，再將經驗帶回 Windows」——這成為 Microsoft 在 1980 年代後期的標準戰術。許多在 Word for Mac 中開發的功能和 UI 設計——包括工具列概念——後來被移植到 Windows 版。

## 意義

Microsoft Word for Mac 證明了跨平台軟體的可行性，也讓 Microsoft 在 Mac 生態系中佔據了關鍵位置。更重要的是，它為 Microsoft Office 在 1990 年代的崛起打下了基礎。
