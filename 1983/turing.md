# 1983 年圖靈獎：Ken Thompson & Dennis Ritchie

## UNIX 與 C 語言的創造者

1983 年，ACM 圖靈獎首次同時頒發給兩位獲獎者：Ken Thompson 和 Dennis Ritchie。授獎理由是「表彰他們在通用作業系統 UNIX 和程式語言 C 上的理論與實作貢獻」。

Thompson 和 Ritchie 在貝爾實驗室的合作始於 1960 年代。Thompson 在 1969 年寫出了第一個 UNIX 版本，執行在 PDP-7 上——最初只是為了執行他設計的「太空旅行」遊戲。Ritchie 則為了改寫 UNIX，設計了 C 語言。1973 年，UNIX 用 C 語言重寫完成——這是歷史上第一個用高階語言實現的作業系統。

## UNIX 的哲學

UNIX 的設計哲學——小就是美、每個程式只做一件事、透過文字管線連接程式——影響了整個軟體產業。UNIX 的可攜性讓它可以運行在從微控制器到超級電腦的各種硬體上。BSD、Linux、macOS 都是 UNIX 傳統的直接繼承者。

## C 語言的統治

C 語言在系統程式設計領域的地位至今無人能撼動。從作業系統、嵌入式系統到編譯器、資料庫——計算機產業的基礎設施多半用 C 寫成。C 的語法被 C++、Java、C#、JavaScript、Go 等語言繼承。

## 圖靈獎演講

Thompson 在圖靈獎演講「Reflections on Trusting Trust」中演示了一個經典的安全論證：編譯器可以在編譯自己時植入後門，使得後門在原始碼中完全不可見。這場演講成為資訊安全領域的經典教材。
