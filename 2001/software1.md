# Mac OS X 10.0 Cheetah：正式登場

## 十年的等待

2001 年 3 月 24 日，Apple 正式發布了 Mac OS X 10.0，代號 Cheetah（獵豹）。這是從 1996 年 Apple 收購 NeXT 以來，歷經五年開發後的正式版本——對 Mac 使用者來說，這是超過十年的等待（上一次重大系統更新還是 1991 年的 System 7）。

Mac OS X 10.0 在 3 月的 Macworld San Francisco 上由 Steve Jobs 親自展示。他在舞台上同時切換多個重度運算應用程式（iTunes 播放音樂、播放 QuickTime 影片、執行 Photoshop），展示基於 Unix 核心的穩定性——這在 Classic Mac OS 上幾乎是不可能的任務。

## 技術革新

**Darwin 核心**：基於 Mach 3.0 微核心和 FreeBSD 的 Unix-like 作業系統核心提供了真正的記憶體保護、搶佔式多工和對稱多處理支援。

**Aqua 使用者介面**：Public Beta 後進一步精緻化的介面，包括半透明效果、Dock 和新的視窗管理。

**Quartz 繪圖引擎**：使用 PDF 作為繪圖模型的 2D 圖形系統，提供了 WYSIWYG 的精確度和高品質的字體渲染。

**Carbon**：讓 Classic Mac OS 應用程式移植到 Mac OS X 的 API，簡化了開發者的過渡。

**Classic 環境**：可以在 Mac OS X 中運行舊的 Mac OS 9 應用程式。

## 初期爭議

Mac OS X 10.0 在發布後收到了褒貶不一的評價：

**優點**：穩定性極佳、Unix 核心強大優雅、Aqua 介面美麗

**缺點**：速度緩慢（特別是在 G3 機器上）、應用程式不足、缺乏 CD 燒錄和 DVD 播放等功能

許多評論家認為它是一款有巨大潛力但還不夠成熟的產品。但考慮到 Apple 從 1996 年就開始重寫作業系統，Mac OS X 10.0 的問世本身就被視為一項了不起的成就。
