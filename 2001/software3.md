# Linux 2.4 核心：企業級成熟

## 重要的版本躍進

2001 年 1 月 4 日，Linus Torvalds 釋出了 Linux 核心 2.4.0。這是從 1999 年的 2.2 系列以來的又一次重大升級，代表了 Linux 從「駭客的玩具」到「企業級伺服器作業系統」的關鍵轉變。

## 新功能亮點

**增強的 SMP 支援**：2.4 核心對多處理器系統（4 顆以上）的支援獲得了大幅改善，讓 Linux 可以在高階伺服器市場中與 Unix 和 Windows 競爭。

**36 位元記憶體定址**：支援超過 4 GB 的物理記憶體，這對大型資料庫和科學運算應用至關重要。

**USB 支援**：首次在核心中加入了完整的 USB 子系統，支援 USB 鍵盤、滑鼠、儲存裝置等周邊設備。

**IEEE 1394（FireWire）支援**：為高效能外部裝置（如 DV 攝影機和硬碟）提供了支援。

**改進的檔案系統**：對 ext2、ext3（日誌式檔案系統）和 ReiserFS 的支援不斷改善。

**Netfilter/IPtables**：全新的防火牆架構取代了舊的 ipchains，提供了更靈活的封包過濾和網路位址轉譯（NAT）功能。

## 企業採用

Linux 2.4 的穩定性、可擴展性和硬體支援讓它成為企業部署的可行選擇：

- IBM 在 2001 年宣布投入 10 億美元支援 Linux
- Oracle 資料庫開始全面支援 Linux 平台
- 戴爾、HP、IBM 開始在伺服器產品中預裝 Linux
- 網際網路基礎設施（Web 伺服器、郵件伺服器、DNS）大量轉向 Linux

## Linux 發行版

2001 年也是 Linux 發行版百花齊放的一年：

- Red Hat Linux 7.x 系列（基於 2.4 核心）
- Debian 2.2 「Potato」和 3.0 「Woody」
- Mandrake Linux 8.0
- SuSE Linux 7.x
