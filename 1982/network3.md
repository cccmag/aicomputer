# 區域網路的百家爭鸣

## 不同的技術路線

1982 年，區域網路（LAN）市場處於激烈競爭的狀態，至少有四種主要技術路線在爭奪市場：

1. **Ethernet** (Xerox/DEC/Intel, 10 Mbps)：最終勝出的技術
2. **Token Ring** (IBM, 4/16 Mbps)：IBM 主導的技術，後來 IEEE 802.5
3. **ARCNET** (Datapoint, 2.5 Mbps)：使用主動式集線器的令牌匯流排網路
4. **Omninet** (Corvus, 125 Kbps)：低價方案，適合小型辦公室

每種技術各有優缺點：Ethernet 速度快但布線成本高；Token Ring 在 IBM 的支援下在大型企業中受歡迎；ARCNET 以可靠性著稱；Omninet 以低成本吸引小型用戶。

## 標準化的努力

IEEE 802 計畫委員會在 1980 年成立，負責區域網路的標準化。802 系列標準涵蓋了多種技術：
- 802.3: CSMA/CD (Ethernet)
- 802.4: Token Bus
- 802.5: Token Ring

標準化的目的是確保不同廠商的產品可以互通。1982 年，802.3 標準草案已經接近完成（1983 年正式發布）。這場「LAN 戰爭」歷經整個 1980 年代，最終由 Ethernet 憑藉成本優勢和速度演進能力勝出。
