# DEC Alpha 21064：效能之王的巔峰

## 速度的追求

1993 年，DEC Alpha 21064 仍然是世界上最快的處理器。這顆於 1992 年推出的晶片，在 1993 年達到了量產巔峰——150 MHz 到 200 MHz 的時脈速度使其在 SPEC 基準測試中遙遙領先於 Intel Pentium（60-66 MHz）和任何其他競爭對手。

Alpha 21064 的設計目標可以用 DEC 工程師的一句話概括：「如果你想要兩倍的效能，就設計兩倍快的處理器。」他們不依賴複雜的超純量技術，而是追求更高的時脈速度和更深的管線化。

## 技術細節

Alpha 21064 的技術規格在 1993 年仍然令人驚嘆：
- 完全 64 位元 RISC 架構（無 32 位元相容模式）
- 七級整數管線，十級浮點管線
- 三個獨立的功能單元：整數、浮點和位址
- 16 KB 指令快取 + 16 KB 資料快取（晶片內建）
- 支援最多 128 GB 實體記憶體
- 採用 0.675 μm CMOS 製程，168 萬個電晶體

## Alpha 的生態系統

DEC 為 Alpha 打造了完整的生態系統。1993 年時，Alpha 工作站可以執行：
- DEC OSF/1（基於 Mach 核心的 Unix）
- OpenVMS（DEC 的傳統企業作業系統）
- Windows NT（Microsoft 為 Alpha 移植了 Windows NT 的專用版本）

## 為什麼 Alpha 消失了

Alpha 的消亡並非技術問題，而是商業現實。DEC 在 1990 年代中期陷入財務困境，1998 年被 Compaq 收購。Compaq 在 2001 年與 HP 合併，HP 決定終止 Alpha 產品線並轉向 Intel Itanium。

Alpha 的技術遺產——深管線化、高時脈設計、64 位元架構——影響了後來所有高效能處理器的設計。從 Intel Core 系列到 Apple 的 M 系列晶片，Alpha 的幽靈仍然存在。
