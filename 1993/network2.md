# 第一個網路廣播電台：Internet Talk Radio

## 聲音在網路上流動

1993 年，Carl Malamud 創辦了 Internet Talk Radio——世界上第一個網際網路廣播電台。這是一個使用 Mbone 技術和 FTP 下載模式發行的「廣播」節目，展示了網路作為新媒體平台的潛力。

Internet Talk Radio 的節目「Geek of the Week」每週採訪一位電腦科學和網路領域的重要人物。節目長約 30-60 分鐘，以 16 KB/s 的位元率錄製成音訊檔案。聽眾通過 FTP 或 Mbone 即時串流收聽。

## 技術現實

1993 年的網路廣播面臨著巨大的技術限制：

- 典型的家用數據機速度為 14.4 kbps——不足以串流 16 KB/s 的音訊
- 多數聽眾需要先下載完整的音訊檔案再收聽——這需要數小時的下載時間
- Mbone 雖然支援即時串流，但只有大學和研究機構能夠存取

儘管如此，Internet Talk Radio 證明了「網路可以廣播」這件事。它啟發了後來更為著名的 RealAudio（1995 年），後者實現了真正意義上的即時串流。

## 媒體的新規則

Internet Talk Radio 展示了舊媒體規則在新平台上的不適用。傳統廣播受困於頻譜限制——一個頻率在同一時間只能播放一個節目。網路廣播打破了這個限制：任何人在任何時候都可以點播任何節目。

Malamud 在 1993 年的實驗，預示了 podcast、Spotify 和串流音樂的未來。雖然 1993 年的聽眾可能只有幾千人，但這個模式——用網路取代空氣作為媒體的載體——是傳播史上的一次根本性轉變。
