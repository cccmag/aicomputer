# 歷史穿梭：1993 年的種子

## 從 Mosaic 到網際網路文明

1993 年是 World Wide Web 成為大眾現象的轉折年：

**Mosaic (1993)** → **Netscape (1994)** → **Internet Explorer (1995)** → **Google (1998)** → **Web 2.0 (2005)**

Mosaic 在 1993 年證明了 Web 瀏覽器應該是圖形化的、多平台的、容易安裝的。這個理念被 Netscape 繼承並商業化，被 Microsoft 的 Internet Explorer 複製，最終成為全球數十億人日常使用的工具。

## 從 Pentium 到 x86 霸權

**Intel Pentium (1993)** → **Pentium Pro (1995)** → **Pentium II (1997)** → **Pentium III (1999)** → **Pentium 4 (2000)** → **Core 2 (2006)**

Pentium 不僅是一款晶片，它是一個品牌、一個時代的象徵。在 Pentium 時代，Intel 從一家半導體公司轉變為全球最大的消費品牌之一——「Intel Inside」行銷攻勢讓處理器的品牌名稱進入了每個消費者的認知。

## 計算複雜度與密碼學

Hartmanis 和 Stearns 在 1965 年建立的複雜度理論，在 1990 年代找到了最重要的應用——網際網路安全：

**複雜度理論 (1965)** → **NP 完備性 (1971)** → **RSA 加密 (1977)** → **SSL (1994)** → **HTTPS (1994)**

如果沒有複雜度理論的基礎，就無法證明某些數學問題的難度——而 RSA 和 SSL 的安全性正是依賴於「大數分解很難」這個假設。

## 從 Doom 到電競產業

**Wolfenstein 3D (1992)** → **Doom (1993)** → **Quake (1996)** → **Counter-Strike (1999)** → **電競產業 (2010s)**

Doom 在 1993 年建立的多人 deathmatch 模式、共享軟體發行方式和 mod 社群文化，影響了整個遊戲產業。Doom 的程式碼架構（遊戲引擎與遊戲內容分離）也讓玩家可以製作自訂關卡——這是使用者生成內容（UGC）的早期範例。

## 1993 年的時代脈絡

- **網際網路的公共化**：CERN 將 WWW 放入公共領域，開創了免費開放標準的先例
- **冷戰後的數位經濟**：商業 ISP 的出現讓網際網路從政府計畫轉變為市場經濟的一部分
- **多媒體的爆炸**：Mosaic 的圖片功能、Doom 的 3D 圖形、Internet Talk Radio 的網路廣播——1993 年證明了網路不只是文字
- **IBM 的衰落與 Microsoft 的崛起**：IBM 在 1993 年報告了 81 億美元的虧損（美國公司史上最大虧損），象徵了大型主機時代的結束和個人電腦時代的全面到來

1993 年是「網路真正開始」的一年——不是技術上的誕生（那發生在 1989-1991），而是文化上的誕生。這一年，網際網路從科學家的書房走進了報紙頭版和一般人的對話中。
