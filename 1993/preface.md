# 1993 年資訊科技年鑑：前言

1993 年是 World Wide Web 從學術工具轉變為大眾現象的關鍵年份。這一年發生的兩件事改變了網路的歷史：NCSA Mosaic 瀏覽器的發布讓網際網路變得圖形化且易於使用；CERN 在 4 月 30 日宣布將 WWW 技術免費開放給所有人使用，確保了 Web 的自由發展。

硬體領域，Intel Pentium 處理器在 3 月 22 日正式出貨，以 60 和 66 MHz 的時脈速度和高達 3.1M 個電晶體，為個人電腦帶來了工作站級的運算能力。DEC 的 Alpha 21064 繼續保持著世界最快處理器的地位。PCI 匯流排的推出標準化了週邊連接介面，並沿用至今。

軟體方面，Microsoft 推出了 Windows NT 3.1——一個全新、32 位元、可移植的作業系統架構，象徵著 Microsoft 對企業市場的正式進軍。id Software 發布了 Doom，以共享軟體模式發行，定義了多人連線遊戲並推動了網路遊戲的發展。Slackware Linux 的發布讓 Linux 系統的安裝不再是專家才能完成的任務。

同年，ACM 圖靈獎頒給了 Juris Hartmanis 和 Richard Stearns，表彰他們在 1965 年共同發表的開創性論文——《計算機演算法的計算複雜度》，這篇論文建立了計算複雜度理論的基礎，定義了複雜度類別的存在性。
