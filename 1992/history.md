# 歷史穿梭：1992 年的種子

## 從 ThinkPad 到筆記型電腦霸權

1992 年 IBM ThinkPad 的推出，開啟了一條持續至今的產品線：

**ThinkPad 700C (1992)** → **ThinkPad 701C 蝴蝶鍵盤 (1995)** → **ThinkPad T 系列** → **Lenovo 時代 (2005)**

ThinkPad 的小紅點（TrackPoint）和黑色簡約設計至今仍是筆記型電腦設計的標杆。從 IBM 到 Lenovo，ThinkPad 品牌維繫了超過三十年的忠實用戶群。

## Windows 3.1 到 Windows 95 之路

1992 年 Windows 3.1 的巨大成功，為 Microsoft 的桌面霸權奠定了基礎：

**Windows 3.0 (1990)** → **Windows 3.1 (1992)** → **Windows for Workgroups (1992)** → **Windows NT (1993)** → **Windows 95 (1995)**

Windows 3.1 的 TrueType 字型系統和 OLE 技術，讓 Windows 在文件處理和多媒體領域首次超越了 Macintosh。同時，Windows 3.1 也讓 Microsoft 在與 IBM 的 OS/2 戰爭中取得了決定性優勢。

## 從 Mosaic 到網際網路商業化

1992 年開始開發的 Mosaic 瀏覽器，引爆了整個網際網路的商業化：

**Mosaic 開發 (1992)** → **Mosaic 發布 (1993)** → **Netscape (1994)** → **Internet Explorer (1995)** → **Google (1998)**

Mosaic 的兩位作者——Marc Andreessen 和 Eric Bina——在 1993 年改變了世界。但這一切的根源可以追溯到 1992 年底 NCSA 的決定：用一小筆研究經費支持兩位實習生開發一個圖形化瀏覽器。

## 從 Alpha 到 64 位元運算

1992 年 DEC Alpha 的推出，是 64 位元運算的重要里程碑：

**DEC Alpha (1992)** → **MIPS R4000 (1992)** → **AMD64 (2003)** → **ARMv8-A (2011)**

Alpha 雖然在商業上最終消失，但它證明了 64 位元架構的可行性。從伺服器到個人電腦再到智慧型手機，64 位元已經成為所有運算裝置的標準。

## GSM 到行動網路

1992 年第一個 GSM 通話，奠定了現代行動通訊的基礎：

**GSM (1992)** → **SMS 服務 (1993)** → **GPRS (2000)** → **3G (2001)** → **iPhone (2007)** → **4G/5G**

GSM 不僅是一項通訊技術，更是一種商業模式——它建立了頻譜授權、基礎設施共享和國際漫遊的框架，這些框架至今仍在運作。

## 1992 年的時代脈絡

- **後冷戰的數位化**：蘇聯解體後的第一年，網際網路開始從軍事研究網路轉向全球基礎設施
- **歐盟單一市場的建設**：GSM 作為歐洲統一的通訊標準，是歐洲技術主權的展現
- **克林頓當選美國總統**：高爾（Al Gore）成為副總統，持續推動資訊高速公路政策
- **多媒體產業的形成**：MPC 標準、CD-ROM、音效卡的普及，讓電腦成為多媒體機器

1992 年是「個人運算」真正成熟的年份——個人電腦不再只是文字處理和試算表的工具，開始成為通訊、遊戲、多媒體和工作的綜合平台。
