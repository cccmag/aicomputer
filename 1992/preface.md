# 1992 年資訊科技年鑑：前言

1992 年是個人電腦成熟、行動通訊起步的一年。Microsoft Windows 3.1 在 4 月發布，第一個月就銷售超過 100 萬套，將圖形使用者介面帶入主流市場。IBM 推出了 ThinkPad 700C——第一款配備小紅點（TrackPoint）的筆記型電腦，從此改變了行動運算的工業設計。

硬體領域見證了兩個重要的處理器誕生：DEC 的 Alpha 21064 是世界上最快的 64 位元 RISC 處理器，展示了精簡指令集的極致效能；IBM、Motorola 和 Apple 的 PowerPC 601 則代表了三巨頭聯手挑戰 Intel 的開端。JPEG 影像壓縮標準正式定案，為數位影像的儲存與傳輸建立了通用格式。

軟體方面，386BSD 的發布是開放原始碼作業系統發展的另一個里程碑。Wolfenstein 3D 由 id Software 發布，定義了第一人稱射擊遊戲類型。OS/2 2.0 展示了 IBM 在桌面作業系統領域的技術能力，卻未能撼動 Windows 的市場地位。

網路領域最重大的消息是 Mosaic 瀏覽器在 NCSA 開始開發——Marc Andreessen 和 Eric Bina 這兩位學生將在 1993 年用它引爆全球網路熱潮。同時，第一個 GSM 通話在芬蘭接通，預示了數位行動通訊的革命。

ACM 圖靈獎頒給了 Butler Lampson，表彰他對個人分散式運算環境的開創性貢獻——包括 Alto 電腦、乙太網路和雷射印表機技術。
