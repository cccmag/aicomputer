# Windows 3.1：Microsoft 的里程碑

## 數百萬人的第一個視窗系統

1992 年 4 月 6 日，Microsoft 發布了 Windows 3.1。雖然 Windows 3.0（1990 年）已經取得了巨大成功，但 3.1 才是真正讓 Windows 成為主流的版本——它在上市後的頭兩個月就銷售了超過 100 萬套。

Windows 3.1 不僅是 3.0 的維護更新，它包含了大量新功能和改進：
- **TrueType 字型系統**：內建可縮放字型，不再需要 Adobe Type Manager
- **OLE（物件連結與嵌入）**：讓不同應用程式之間的資料共享成為可能
- **多媒體擴充**：原生支援音效卡、CD Audio 和 MIDI
- **Super VGA 支援**：最高 800x600 解析度
- **CTRL-ALT-DEL 攔截**：避免意外重啟
- **檔案管理器（File Manager）和程式管理器（Program Manager）的改進**

## 零售與技術意義

Windows 3.1 的零售價為 149 美元。它在技術上最重要的決定是徹底移除了「真實模式」（Real Mode）——這個模式是為了相容 8086 處理器而保留的。移除真實模式意味著 Windows 3.1 需要 80286 或更高的處理器（至少 1 MB RAM），但也讓系統更穩定、更快。

Windows 3.1 也在法律上引發了爭議——Apple 控告 Microsoft 抄襲 Macintosh 的「外觀與感覺」（look and feel），但 Microsoft 最終贏得了這場訴訟。

## 作業系統市場的重新定義

Windows 3.1 的成功徹底改變了 PC 軟體生態。軟體開發商開始優先開發 Windows 應用程式而非 DOS 應用程式。從 1992 年開始，「Windows 相容」成了 PC 軟體的預設要求。Windows 3.1 的市場主導地位一直持續到 Windows 95 的發布（1995 年 8 月）。
