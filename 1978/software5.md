# 程式語言的實用化發展

## 從學術到商業

1978 年是程式語言從學術研究走向商業應用的關鍵年。Pascal 在大學中已經站穩腳步，但業界主流仍然是 FORTRAN（科學計算）、COBOL（商業資料處理）和組合語言（系統程式）。

## C 語言的萌芽

Brian Kernighan 和 Dennis Ritchie 的「The C Programming Language」（K&R C）雖然要到 1978 年底才出版，但在此之前，C 語言在貝爾實驗室內部已經使用多年。1978 年是 C 開始從實驗室走向外部世界的年份——Unix 作業系統的移植需求正在推動 C 語言的發展。

## BASIC 的成熟

在微電腦世界，Microsoft BASIC 已經成為事實標準。但 1978 年也出現了多種 BASIC 的商業分支——Commodore BASIC、Applesoft BASIC、TRS-80 Level II BASIC——每一種都針對各自的硬體平台做了客製化。

## 作業系統的程式設計

1978 年，電腦科學領域最重要的出版事件之一是「Algorithms + Data Structures = Programs」（Niklaus Wirth, 1976）在教學界的廣泛應用。同時，Kleinrock 的「Queueing Systems」持續影響著網路和作業系統的設計。
