## 關於本年鑑

1975 年是微電腦革命的元年。Altair 8800 的問市讓個人擁有電腦成為可能，MOS 6502 將處理器價格降至人人都負擔得起的程度，Microsoft 和 Homebrew Computer Club 在同一年誕生——這一年奠定了未來十年個人電腦產業的基礎。
