# 歷史穿梭：1975 年的轉折

## 微電腦革命的一年

1975 年是個人電腦歷史的轉捩點。三件事在同一年發生，共同創造了完美的風暴：

**Altair 8800** 在 1 月提供了大眾可以負擔的電腦硬體。**Altair BASIC** 和 **Microsoft** 提供了軟體和商業模式。**Homebrew Computer Club** 提供了社群和知識分享的平台。

這三者的結合——硬體、軟體、社群——創造了一個自我強化的生態系。用經濟學的術語來說，這是典型的「平台效應」：

```
Altair 8800 → 吸引業餘愛好者 → 需要軟體 → BASIC → 需要更多硬體 → 
Altair 相容機 → 更多使用者 → CP/M → 應用軟體 → 
Homebrew 知識分享 → 加速創新 → Apple、Commodore、Tandy
```

## 從迷你到微小的典範轉移

1975 年的微電腦革命代表了一種典範轉移（paradigm shift），Thomas Kuhn 的術語在此完全適用。大型機和迷你電腦的運算模式是集中的、昂貴的、專業的。微電腦則是分散的、便宜的、人人可用的：

**大型機典範**：
- 一台電腦服務數百人
- 由專業操作員管理
- 價格以百萬美元計
- 使用打孔卡片和終端機

**微電腦典範**：
- 一個人擁有一台電腦
- 由使用者自己管理
- 價格以數百美元計
- 使用鍵盤和螢幕

這個轉變不是技術上的漸進改良，而是運算理念的根本變革。

## 1975 年的時代脈絡

1975 年的科技發展與時代背景的緊密關聯：

- **美國建國兩百週年前夕**：1976 年是美國獨立 200 週年，科技創新被視為美國精神的體現
- **越南戰爭結束**：1975 年 4 月越戰結束，大量軍方技術人才流入民間
- **矽谷的成形**：Santa Clara Valley 從「杏梅谷」轉變為「矽谷」的過程正在加速
- **車庫創業文化**：Hewlett-Packard 的車庫創業故事（1939 年）成為矽谷的典範，Altair 和 Microsoft 的故事為這個神話增添了新的篇章

## 跨越年代的連結

1975 年的三大遺產：

**Microsoft** 從 Altair BASIC 起步，到 1981 年的 MS-DOS（與 IBM PC 捆綁），到 1985 年的 Windows，到 1990 年代的 Office 壟斷——成為世界上最大的軟體公司。

**6502 的價格革命** 直接通往 Apple II（1977）、Commodore 64（1982）和 Nintendo Entertainment System（1983）。這些機器將電腦帶入了數百萬家庭。

**Homebrew Computer Club 的開放精神** 通往自由軟體運動、Linux、開源軟體——一種重新定義了軟體生產和分配方式的運動。

1975 年是如此豐富的一年，以至於後世回顧時會說：現代電腦世界的藍圖，有一大半是在這一年畫下的。
