# 歷史穿梭：1977 的「三一體」

## 個人電腦的典範確立

1977 年三台個人電腦的共同特徵，定義了未來三十年的 PC 工業：

**Apple II** → 開放式擴充槽 + 彩色圖形 + 極佳的使用者體驗  
**Commodore PET** → 一體化設計 + 低價策略 + 垂直整合  
**TRS-80** → 零售通路優勢 + Z80 + CP/M 相容

這三條路線後來匯聚成 IBM PC（1981）的設計：開放架構 + Intel 處理器 + Microsoft 作業系統。

## 從 1977 三一體到今日

**Apple II (1977)** → **Macintosh (1984)** → **iMac (1998)** → **iPhone (2007)** → **Apple Silicon Mac (2020)**

**Commodore PET (1977)** → **VIC-20 (1980)** → **Commodore 64 (1982)** → **Amiga (1985)**

**TRS-80 (1977)** → **Tandy 1000 (1984)** → **IBM PC 相容機浪潮**

## TCP/IP 的漫長旅程

1977 年的三網路連線雖然被視為網際網路的誕生，但 TCP/IP 距離廣泛採用還有很長的路要走：

**TCP 測試 (1977)** → **TCP/IP 規範 (1981, RFC 791/793)** → **ARPANET 全面切換 (1983 年 1 月 1 日)** → **NSFNET (1986)** → **商業網際網路 (1995)**

1983 年 1 月 1 日的「旗標日」（Flag Day）是 ARPANET 從 NCP 協議全面切換到 TCP/IP 的日子——那時已經距離首次三網路測試整整五年。

## 從 BNF 到函數式程式設計

John Backus 的職業生涯貫穿了程式語言發展的兩次大轉折：

**FORTRAN (1957, 高階語言)** → **BNF/ALGOL (1960, 形式語法)** → **FP (1977, 函數式程式設計)**

Backus 1977 年的圖靈獎演講影響了後來 Haskell 和 ML 等純函數式語言的設計。他在演講中呼籲程式設計師擺脫「馮紐曼風格的束縛」——這個呼籲在三十年後隨著函數式程式設計的復興才被廣泛接受。

## VAX/VMS 的傳承

VAX-11/780 和 VAX/VMS 的遺產：

**VAX/VMS (1978)** → **VMS 設計理念** → **Windows NT (1993, Dave Cutler)** → **Windows 2000/XP/Vista/7/10/11**

VMS 的微核心設計、優先權排程、叢集支援等概念被 David Cutler 帶到了 Microsoft。如果你熟悉 VMS 的命令列（DCL），你會發現 Windows NT 的 cmd.exe 有些許相似之處。

## 1977 年的時代脈絡

- **微處理器革命**：6502 和 Z80 的價格已經低到可以放進家用產品
- **零售模式轉變**：電腦從郵購和專賣店進入一般零售通路
- **日本電子的崛起**：Atari 2600 裡的晶片、VHS 錄影機都來自日本
- **卡特時代**：能源危機和經濟停滯反而刺激了自動化辦公的需求

1977 年最深刻的變革是：電腦從一個需要焊接和程式設計技巧的愛好品，變成了一台插電就能用的機器。這個轉變讓電腦走出了車庫和實驗室，進入了辦公室和客廳。
