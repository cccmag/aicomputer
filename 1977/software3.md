# Microsoft BASIC 的版圖擴張

## 從 Altair 到全球

1975 年，比爾蓋茲和保羅艾倫為 Altair 8800 寫出了第一個微電腦 BASIC 直譯器。到了 1977 年，Microsoft 的 BASIC 已經成為微電腦產業的標準軟體。

## 每台機器都需要 BASIC

1977 年「三一體」的每一台電腦都以 BASIC 為核心賣點：

- **Apple II** 使用 Wozniak 開發的 Integer BASIC（後來也採用 Microsoft 的 Applesoft BASIC）
- **Commodore PET** 內建 Commodore BASIC（由 Microsoft 授權並修改）
- **TRS-80** 使用 Level I BASIC（Microsoft 授權，Radio Shack 自訂版本）

Microsoft 的商業模式是授權 OEM 廠商將 BASIC 內建在 ROM 中——每台電腦出貨時都附帶 BASIC，這讓 Microsoft 的收入隨著微電腦市場一起爆炸性成長。

## BASIC-80 的影響

Microsoft 的 BASIC-80（針對 8080/Z80 平台）是當時最完整的 BASIC 實作之一，支援浮點數運算、字串處理、陣列、以及檔案 I/O。它的程式碼經過高度最佳化，可以在 4 KB 的記憶體中運行——這在 1977 年的硬體限制下是極大的挑戰。
