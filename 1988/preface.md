# 1988 年資訊科技年鑑：前言

1988 年是網路安全意識覺醒的一年。11 月 2 日，康乃爾大學研究生 Robert Tappan Morris 從 MIT 釋出了一個自我複製的程式——Morris Worm，感染了當時約 60,000 台網際網路主機中的 6,000 台。雖然程式本身沒有破壞檔案的意圖，但它的快速蔓延導致大量系統癱瘓，成為第一個引起主流媒體關注的網路攻擊事件。

硬體方面，Steve Jobs 在被逐出 Apple 後創立的 NeXT 公司推出了 NeXT Computer。這台使用 Motorola 68030 處理器、配備光碟機和 Megapixel 顯示器的機器，雖然商業上不成功，但其 NeXTSTEP 作業系統後來成為 Mac OS X 的基礎。

軟體領域，ANSI C 標準歷經多年討論終於完成（C89），為 C 語言的跨平台可攜性奠定了基礎。Stephen Wolfram 發表了 Mathematica，開啟了電腦代數系統的新紀元。Intel 的 80386 處理器持續推動 PC 進入 32 位元時代。

通訊基礎建設方面，第一條跨大西洋光纖電纜 TAT-8 啟用，大幅提升了歐美之間的資料傳輸容量。同年，歐洲與北美之間建立了第一條直接的 IP 連線，網際網路真正開始全球化。

音樂產業也有一個重要里程碑：CD 的銷售量首次超越了黑膠唱片，數位音樂時代正式來臨。
