# CD 銷售超越黑膠：數位音樂的轉折

## 一個時代的終結

1988 年，一個重要的文化里程碑發生了：CD（Compact Disc）的銷售量首次超越了黑膠唱片。雖然 CD 在 1982 年就已經問世（由 Philips 和 Sony 共同開發），但直到 1988 年，它才真正在市場上取代了類比音樂的統治地位。

這個轉變的驅動力來自幾個方面：CD 播放器的價格持續下降、唱片公司加速發行 CD 格式的目錄、以及消費者對「完美音質永不衰退」的數位音響的嚮往。

## CD 的技術優勢

CD 使用 16 位元、44.1 KHz 的脈衝編碼調變（PCM）來儲存音訊，理論上能夠涵蓋人耳聽覺範圍內的所有頻率。與黑膠唱片相比，CD：

- 沒有磨損問題（雷射讀取非接觸式）
- 沒有表面噪音（爆音、炒豆聲）
- 可以隨機選曲
- 體積更小、更方便儲存

## 對電腦產業的意義

CD 在音樂市場的成功為 CD-ROM 在電腦市場的普及鋪平了道路。CD-ROM 本質上就是電腦用的 CD，使用相同的碟片尺寸和光學讀取技術。音樂消費者對 CD 的接受間接培養了光碟製造的產能，降低了成本，使得 CD-ROM 光碟機在 1990 年代初期能夠降價到可負擔的水準。

1988 年的這個轉折點不僅改變了音樂產業，也為多媒體電腦時代奠定了硬體基礎。
