# 歷史穿梭：1988 年的種子

## 從 Morris Worm 到網路安全產業

1988 年的 Morris Worm 催生了整個網路安全產業：

**Morris Worm (1988)** → **CERT 成立 (1988)** → **第一個防毒軟體 (1980s 末)** → **防火牆 (1990s 初)** → **IDS/IPS (1990s 末)** → **現代網路安全產業**

在 Morris Worm 之前，網路安全是學術研究的一個冷門分支，多數網路管理者對安全問題漠不關心。Morris Worm 之後，網路安全成為每個組織必須面對的現實問題。CERT（Computer Emergency Response Team）在 1988 年 11 月底成立，成為全球電腦安全事件應對的協調中心。

有趣的是，Robert Tappan Morris 後來成為了 MIT 的電腦科學教授，指導了多位在分散式系統和安全領域有傑出貢獻的學生。他的父親 Robert Morris 是 NSA 的知名密碼學家。

## 從 NeXT 到 Mac OS X

Steve Jobs 被逐出 Apple 後創立的 NeXT，雖然商業失敗，但其技術最終回歸 Apple：

**NeXT Computer (1988)** → **NeXTSTEP (1989)** → **Apple 收購 NeXT (1997)** → **Mac OS X (2001)** → **iOS (2007)** → **iPadOS (2019)**

這是一條充滿諷刺的傳承路線：Jobs 因為 Apple 的政治鬥爭而離開，十二年後帶著 NeXT 的技術回到 Apple，而 NeXTSTEP 最終成為了現代 Apple 所有作業系統的基礎。

Tim Berners-Lee 在 NeXT 電腦上開發了 WWW，這讓 NeXT 不僅僅是「Mac OS 的前身」——它還是全球資訊網的出生地。

## ANSI C 到 Java 和 C#

ANSI C 的標準化（1988-89）對程式語言的深遠影響：

**K&R C (1978)** → **ANSI C (1989)** → **C++ (1985-)** → **Java (1995)** → **C# (2000)**

C 語言的標準化讓程式碼可以在不同平台之間移植，這直接影響了網際網路的發展（伺服器軟體可以輕鬆地在各種 Unix 平台之間移植），也為後來的 Java「一次編寫，到處執行」的願景提供了語義基礎。

## IRC 到即時通訊

Jarkko Oikarinen 的 IRC（1988）開啟了即時通訊的時代：

**IRC (1988)** → **ICQ (1996)** → **AOL Instant Messenger (1997)** → **MSN Messenger (1999)** → **WhatsApp (2009)** → **Slack/Discord (2010s)**

IRC 的設計——頻道、暱稱、操作者、分散式伺服器——至今仍然影響著現代的通訊軟體。Slack 和 Discord 的頻道結構與 IRC 驚人地相似。

## CD 從音樂到資料

CD 在音樂市場超越黑膠（1988）是儲存媒體轉型的關鍵：

**CD Audio (1982)** → **CD-ROM (1985)** → **CD 超越黑膠 (1988)** → **CD-R/RW (1990s)** → **DVD (1995)** → **Blu-ray (2006)**

音樂市場對 CD 的接受間接促進了 CD-ROM 在電腦市場的普及。沒有音樂 CD 的龐大安裝基礎，CD-ROM 光碟機的生產成本就無法迅速下降。

## 1988 年的時代脈絡

- **冷戰結束前夜**：戈巴契夫在聯合國宣布裁軍，東歐的社會主義陣營開始動搖
- **第一次大規模網路攻擊**：Morris Worm 讓人們第一次意識到網際網路的脆弱性
- **光纖時代的開啟**：TAT-8 的啟用標誌著全球通訊從衛星到海底光纖的轉移
- **蘋果的低谷**：Steve Jobs 離開後，Apple 缺乏創新方向，Macintosh 銷售下滑
