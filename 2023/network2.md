# Reddit API 抗議：社群網路的風暴

## API 收費的導火索

2023 年 4 月，Reddit 宣布將對第三方應用程式的 API 存取大幅提高收費——從原來的免費改為每 1000 次請求收費 0.24 美元。對於 Apollo、Reddit is Fun、Sync 等第三方 Reddit 用戶端來說，這意味著每年數百萬美元的成本。

Apollo 的開發者 Christian Selig 計算，按照他的應用程式的使用量，每年需要支付 2000 萬美元的 API 費用——遠超過 Apollo 的收入。

## 大規模抗議

2023 年 6 月 12 日，超過 8000 個 Reddit 子版（subreddit）**集體關閉 48 小時**以示抗議。許多版主將版面設為私人狀態，使 Reddit 的流量大幅下降。這是 Reddit 歷史上最大規模的集體行動。

抗議的核心問題不只是價格，更是 Reddit 對社群貢獻者的態度——Reddit 的內容由使用者與版主志願者創造，但公司卻在沒有與社群協商的情況下單方面改變遊戲規則。

## 結果與影響

Reddit CEO Steve Huffman 態度強硬，拒絕妥協。他表示第三方開發者「應該付費才能使用 Reddit 的資料」，並暗示願意讓 Reddit 在短期內變得不那麼友善來推動 IPO。

最終，多個知名第三方用戶端（包括 Apollo、Reddit is Fun）在 6 月 30 日停止營運。部分版主被 Reddit 強制撤換，以結束抗議。

## 延伸的討論

Reddit 的 API 抗議引發了關於「平台與社群關係」的深層討論：

- 社群產生的資料歸誰所有？
- 平台公司應該如何對待志願貢獻者？
- 在 AI 訓練時代，API 與資料的定價問題？
- 類似爭議在 X（Twitter）與 Mastodon 之間也持續發酵
