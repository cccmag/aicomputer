# macOS Catalina 與 Windows 10 的重大更新

## macOS Catalina：iPad 應用進入 Mac

2019 年 10 月，Apple 發布 macOS 10.15 Catalina。最重要的變化是取代 iTunes 的獨立應用程式——Music、Podcasts 和 TV——結束了 iTunes 長達 18 年的歷史。

Catalina 最受關注的新功能是 Sidecar，允許 iPad 作為 Mac 的第二個顯示器或繪圖板使用。這項功能利用了 Apple 自主設計的晶片和 Metal 圖形框架，延遲極低且支援 Apple Pencil。

然而，Catalina 也是 macOS 過渡到 64 位元的最後一步——它不再支援 32 位元應用程式，這導致許多老舊軟體無法使用。這項變革雖然在短期內引起不便，但對系統安全性和性能有長遠益處。

## Windows 10 2019 年 5 月更新

Microsoft 在 2019 年發布了 Windows 10 的兩個重要更新：2019 年 5 月更新（版本 1903）和 2019 年 11 月更新（版本 1909）。

版本 1903 引入了「Windows 沙盒」（Windows Sandbox），允許使用者安全地在隔離環境中執行可疑程式。它還新增了輕量級主題模式和改善了更新機制——使用者可以更好地控制更新的時機。

版本 1909 則是一次較小的更新，專注於穩定性和性能改進，包括檔案總管的效能優化和 Windows 搜尋的改進。

## 桌面生態的雙軌發展

2019 年的這兩大桌面作業系統代表了不同的生態哲學：Apple 持續強化其硬體軟體整合的生態系統，為 ARM 轉換做準備；Microsoft 則將 Windows 定位為開放平台，積極擁抱 Linux 和容器技術。
