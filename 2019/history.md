# 歷史穿梭：2019 年的種子

## 預訓練模型的崛起：從 BERT 到 GPT 系列

2019 年是 NLP 預訓練模型從學術研究轉向工業應用的關鍵一年：

**ELMo (2018)** → **BERT (2018)** → **GPT-2 (2019)** → **RoBERTa (2019)** → **T5 (2019)** → **GPT-3 (2020)**

BERT 的雙向編碼器方法和 GPT-2 的自回歸解碼器方法代表了兩種不同的技術路線。這兩條路線在 2023 年分別演化為編碼器模型（如 DeBERTa）和解碼器模型（GPT-3/4、Llama）。BERT 的開源策略與 GPT-2 的分階段釋出策略也反映了兩種不同的研究文化。

## 從 x86 到 ARM 的轉折點

2019 年蘋果宣布轉向 ARM，是個人電腦架構史上的一個分水嶺：

**PowerPC (1994-2006)** → **Intel x86 (2006-2020)** → **Apple Silicon ARM (2020-)**

這條路徑反映了摩爾定律放緩後，垂直整合（Apple 設計自己的晶片）相對於水平分工（Intel 提供標準晶片）的優勢。Apple M1 在 2020 年的成功證明了這個決策的正確性。

## 隱私革命的連鎖效應

2019 年的隱私趨勢可以追溯到更早的事件：

**Snowden 揭露 (2013)** → **GDPR 通過 (2016)** → **GDPR 生效 (2018)** → **GDPR 執法 (2019)** → **CCPA (2020)**

從 1.1.1.1 的推出到 Apple 在 iOS 中的隱私標籤，隱私已從「合規要求」變成了「產品功能」和「市場優勢」。

## 雲端遊戲的嘗試與失敗

雲端遊戲的歷史顯示了技術與市場之間的反覆：

**OnLive (2009)** → **Gaikai (2011)** → **PlayStation Now (2014)** → **Google Stadia (2019)** → **Xbox Cloud Gaming (2021)**

Stadia 的失敗證明：即使擁有頂尖的技術基礎設施，缺乏內容生態系統和明確的商業模式也無法成功。

## 雲端原生時代的到來

2019 年的容器生態系統：

**Docker (2013)** → **Kubernetes (2014)** → **OpenShift (Red Hat, 2019)** → **IBM 收購 Red Hat (2019)**

IBM 以 340 億美元收購 Red Hat 的事實表明：開源軟體已從「免費的替代品」變成了「企業 IT 的核心戰略資產」。

## 2019 年的時代脈絡

- **中美科技脫鉤**：華為被列入實體清單（2019 年 5 月），科技供應鏈開始重組
- **氣候行動**：全球氣候罷工運動（9 月），Greta Thunberg 喚醒了公眾對氣候危機的關注
- **貿易戰**：中美關稅戰升級，影響了半導體供應鏈和科技產品的定價
- **5G 部署**：韓國和美國率先啟動 5G 商用服務（4 月），為行動網路的新時代揭開序幕
