# 2019 年資訊科技年鑑：前言

2019 年是半導體製程與人工智慧交織的一年。AMD 以 7nm Ryzen 3000 系列震撼了 x86 處理器市場，迫使 Intel 加速其 10nm 的推進。這一年，蘋果宣布與英特爾分手，轉向自主設計的 ARM 架構處理器——一個將在 2020 年徹底改變個人電腦格局的決定。

人工智慧領域，OpenAI 在 2 月發表了 GPT-2，但因其對假訊息生成的潛在風險而延後完整釋出，引發了 AI 安全與透明度的廣泛討論。Google 的 BERT 則全面改變了自然語言處理的技術路線，讓預訓練模型成為 NLP 的標準範式。

軟體開發方面，TypeScript 持續侵蝕 JavaScript 的領地，Deno 的開發預示著伺服器端 JavaScript 的新方向。Microsoft 推出 WSL 2，首次將真正的 Linux 核心整合到 Windows 中——這在十年前是難以想像的。

雲端遊戲成為熱點，Google Stadia 與 Apple Arcade 分別代表了串流與訂閱兩種不同的遊戲未來。IBM 以 340 億美元完成對 Red Hat 的收購，彰顯了開源軟體在企業市場中的核心地位。

2019 年也是隱私保護的關鍵年——GDPR 的效應開始在全球擴散，多國跟進制定類似的資料保護法規。Cloudflare 推出了 1.1.1.1 隱私 DNS 服務，將隱私權定位為網路基礎設施的核心功能。
