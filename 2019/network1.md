# Cloudflare 1.1.1.1：隱私優先的 DNS 服務

## 網路的電話簿

DNS（Domain Name System）是網際網路的基礎設施，相當於網路世界的電話簿——將人們易記的網域名稱轉換為機器可讀的 IP 位址。然而，傳統的 DNS 查詢在未加密的情況下傳輸，ISP 和網路管理員可以輕易監視使用者的瀏覽記錄。

## 1.1.1.1 的誕生

2018 年 4 月 1 日，Cloudflare 推出了 1.1.1.1 DNS 服務，但在 2019 年才獲得廣泛的關注和採用。1.1.1.1 的核心承諾是：

- **隱私優先**：Cloudflare 承諾不記錄使用者的 IP 位址，24 小時後清除所有 DNS 查詢日誌
- **速度**：結合全球 CDN 邊緣網路，提供極低的查詢延遲
- **安全**：支援 DNS-over-TLS（DoT）和 DNS-over-HTTPS（DoH）加密協議

## 與 Google 8.8.8.8 的競爭

在此之前，Google Public DNS（8.8.8.8）是最受歡迎的公共 DNS 服務。但 Google 以其廣告業務聞名，許多使用者對 Google 收集 DNS 資料的意圖持懷疑態度。1.1.1.1 的推出為注重隱私的使用者提供了另一個選擇，並由獨立的四大會計師事務所每年審計其隱私承諾。

## 更深遠的意義

1.1.1.1 的成功反映了一個更廣泛的趨勢：網路使用者對隱私的關注正在從「少數人的關切」變成「主流需求」。它推動了整個 DNS 產業的進步——Google、Quad9 和 Cisco OpenDNS 都在隨後加強了對加密 DNS 的支援。
