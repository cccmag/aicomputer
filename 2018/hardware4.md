# Raspberry Pi 3 Model B+

## 經典升級

2018 年 3 月 14 日，Raspberry Pi 基金會發布了 Raspberry Pi 3 Model B+——這是 Pi 3 系列的強化版本。雖然外觀和價格（35 美元）保持不變，但內部規格獲得了顯著提升：

- **CPU**：BCM2837B0，1.4GHz 四核心 Cortex-A53（比前代快 200MHz）
- **Wi-Fi**：802.11ac 雙頻（2.4/5GHz）
- **藍牙**：4.2 BLE
- **乙太網路**：Gigabit Ethernet（透過 USB 2.0）
- **散熱**：改良的散熱設計，在高負載下可保持更高的時脈

## 單板電腦的生態

Raspberry Pi 系列一直是單板電腦市場的標竿。Pi 3 Model B+ 的發布強化了它在教育、業餘愛好者和嵌入開發者社群中的地位。

2018 年，Raspberry Pi 的應用場景繼續擴展——從電腦科學教育到物聯網閘道器，從媒體中心到工業控制，從家用 NAS 到無線路由器。

## 競爭者

2018 年，Raspberry Pi 面臨越來越多的競爭對手——特別是 ASUS Tinker Board（Rockchip RK3288）和 Odroid-C2（Amlogic S905）。這些產品在計算效能或 I/O 能力上各有優勢，但 Raspberry Pi 憑藉龐大的社群支援和豐富的文件資源，仍然是首選。
