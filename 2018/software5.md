# Flutter 1.0 正式版

## Google 的行動 UI 框架

2018 年 12 月 4 日，Google 在 Flutter Live 2018 活動上正式發布了 Flutter 1.0——這代表 Flutter 從測試階段進入穩定生產階段。Flutter 是一個由 Google 開發的開源 UI 軟體開發套件，用於從單一程式碼庫為行動裝置、網頁和桌面建立高效能應用程式。

Flutter 使用 Dart 語言進行開發，其最大的特點是自繪引擎——它不使用平台原生的 UI 元件，而是使用 Skia 圖形引擎直接繪製所有 UI 元素。

## 技術優勢

Flutter 1.0 的穩定發布帶來了成熟的生態：

- **Hot Reload**：在亞秒級內看到程式碼修改的效果
- **豐富的 Material Design 和 Cupertino 元件**
- **自繪引擎**：跨平台外觀一致性，無需橋接
- **高效能**：AOT 編譯為原生 ARM 機器碼

## 與 React Native 的競爭

Flutter 1.0 的發布加劇了跨平台開發框架的競爭。Flutter 和 React Native 代表了兩種不同的哲學：

| | Flutter | React Native |
|---|---|---|
| 語言 | Dart | JavaScript |
| 渲染方式 | Skia 自繪 | 原生元件橋接 |
| 效能 | AOT 編譯 | JSBridge |
| 啟動時間 | 較快 | 較慢 |

Flutter 雖然在 2018 年底才發布 1.0，但它在 Google 內部和外部都已經積累了足夠的驗證——Google Ads、Alibaba 和許多初創公司都在生產環境中使用了 Flutter。
