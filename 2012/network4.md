# 大數據（Big Data）成為流行語

## 資料驅動的時代

2012 年，「大數據」從技術圈小眾術語一躍成為主流的商業 buzzword。這一年，《紐約時報》刊登了題為「大數據時代」的專題報導，世界經濟論壇將大數據列為「新經濟資產」，白宮宣布投入 2 億美元啟動「大數據研究與發展倡議」。

大數據通常以「3V」定義：**Volume**（大量）、**Velocity**（高速）、**Variety**（多樣）。但 2012 年的熱潮更多來自於企業對資料價值的認識——Google 用搜尋趨勢預測流感，Amazon 用推薦系統提升銷售，Netflix 用觀看資料決定自製內容。

## 技術基礎的成熟

2012 年，處理大數據的技術棧已經相當成熟。Apache Hadoop 生態系統（HDFS、MapReduce、Hive、Pig）成為事實標準。Apache Spark 在 2012 年從 UC Berkeley AMPLab 孵化出來，提供了比 MapReduce 快 100 倍的記憶體內計算能力。

NoSQL 資料庫在 2012 年持續擴張。MongoDB、Cassandra、Redis、Couchbase 各有擅長，滿足不同的大數據場景需求。

## 數據科學家的誕生

2012 年，《哈佛商業評論》發表了著名的文章「數據科學家：21 世紀最性感的職業」。這個稱號迅速在媒體間傳播，引發了商業分析、機器學習和資料工程領域的人才爭奪戰。Coursera、Udacity 等 MOOC 平台上的資料科學課程在 2012 年成為最受歡迎的課程類別之一。
