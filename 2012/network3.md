# Mozilla Firefox OS 與開放 Web

## Web 作為作業系統

2012 年 7 月，Mozilla 宣布了 Firefox OS（開發代號 Boot to Gecko）——一款完全基於開放 Web 技術的行動作業系統。Firefox OS 的核心理念是：作業系統本身應該只是一個極簡的 Linux 核心和硬體抽象層，所有使用者介面和應用程式都使用 HTML5、CSS 和 JavaScript 開發。

這個理念與 iOS 和 Android 形成了鮮明對比。Apple 和 Google 雖然也支援 Web 應用，但都將原生應用作為平台的核心。Mozilla 認為封閉的原生生態違背了網際網路的開放精神。

## 技術架構

Firefox OS 的底層是 Gonk——一個基於 Linux 核心和硬體抽象層的系統。上層是 Gecko——Mozilla 的瀏覽器引擎，負責渲染所有 UI。最上層是 Gaia——使用 Web 技術開發的使用者介面。

Firefox OS 的應用程式就是網頁。開發者可以用同樣的 HTML/CSS/JavaScript 技能開發應用，不需要學習 Objective-C 或 Java。這些應用可以離線運行，可以存取裝置功能（如相機、GPS），還可以像原生應用一樣分發。

## 市場命運

Firefox OS 在 2013 年首先在低價手機市場推出，目標是新興市場。但它在 Android 的價格不斷下降和 iOS 的生態壁壘之間找不到生存空間。2015 年，Mozilla 宣布停止開發 Firefox OS 手機。
