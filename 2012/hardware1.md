# Raspberry Pi 的出貨：低價電腦的革命

## 教育理念

2012 年 2 月 29 日，Raspberry Pi 基金會開始出貨第一款型號——Raspberry Pi Model B。這是一款只有信用卡大小的單板電腦，售價僅 35 美元。它的目標不是取代桌上型電腦，而是讓孩子們有一個可以自由探索的程式設計平台。

Raspberry Pi 的誕生源於 Eben Upton 在劍橋大學擔任招生官時的觀察：2000 年代後期，申請電腦科學系的學生越來越少，且學生的程式設計能力普遍下降。Upton 認為原因之一是孩子們失去了能像 1980 年代的 BBC Micro 那樣可以自由探索的程式設計環境——家庭電腦變得越來越封閉和昂貴。

## 技術規格

Raspberry Pi Model B 搭載 BCM2835 系統晶片（SoC），包含一顆 ARM11 架構的 700MHz 處理器和 Broadcom VideoCore IV GPU。記憶體為 256MB（後期升級至 512MB）。儲存使用 SD 卡。雖然規格在 2012 年看來極其簡陋，但足夠運行 Linux 作業系統和學習程式設計。

## 超乎預期的成功

Raspberry Pi 的市場反應遠遠超過了基金會的預期。首批 10 萬台在數小時內售罄，生產線完全無法滿足需求。到年底時，Raspberry Pi 已售出超過 100 萬台——不是因為孩子們的需求，而是因為業餘愛好者、嵌入式開發者、以及工業用戶的大量採購。

Raspberry Pi 啟發了大量後繼產品（如 BeagleBone、Banana Pi、Orange Pi），也催生了圍繞單板電腦的整個產業生態。它是硬體開源運動和自造者（Maker）運動的重要催化劑。
