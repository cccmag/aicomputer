# Sublime Text 2 與開發者工具

## 程式編輯器的黃金時代

2012 年是程式編輯器市場競爭白熱化的一年。Sublime Text 2 在 2012 年從 beta 走向穩定版，成為許多開發者首選的程式碼編輯器。

Sublime Text 由 Google 工程師 Jon Skinner 於 2008 年開始開發，最初只是他個人的 side project。Sublime Text 2 的核心特色包括小巧快速的啟動速度、多重選取（multiple selections）功能、極簡的使用者介面、以及強大的命令面板（Command Palette）。

## 套件生態系統

Sublime Text 2 的成功在很大程度上歸功於它的套件生態系統。由社區開發的 Package Control 套件管理器讓使用者可以一鍵安裝數千個擴充功能——語法高亮、程式碼片段、主題、linter、建置系統等。

Sublime Text 2 的崛起也催生了編輯器戰爭的新局面：傳統的 Vim 和 Emacs 社群仍然活躍，但圖形化編輯器市場從 TextMate（Mac 專屬）擴展到跨平台的 Sublime Text。日後 Atom（2014）和 Visual Studio Code（2015）的開發理念——以套件生態為核心的輕量級編輯器——都受到 Sublime Text 的深刻影響。
