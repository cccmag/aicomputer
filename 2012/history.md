# 歷史穿梭：2012 年的種子

## 從零知識證明到區塊鏈

Goldwasser 和 Micali 的零知識證明在 2012 年獲得圖靈獎時，區塊鏈技術才剛剛開始受到關注。比特幣誕生於 2009 年，2012 年 Bitcoin 正處於早期發展階段，價格約在 10 美元以下。

**零知識證明 (1985)** → **數位簽章理論** → **比特幣 (2009)** → **zk-SNARKs (2013)** → **Zcash (2016)** → **以太坊 Layer 2 擴容**

零知識證明後來成為區塊鏈隱私保護（如 Zcash）和擴容（如 zk-Rollup）的關鍵技術。Micali 創立的 Algorand 更直接將他的密碼學研究成果應用於區塊鏈共識設計。

## 從 Raspberry Pi 到自造者運動

Raspberry Pi 在 2012 年的出貨不僅是硬體事件，更是一場運動的催化劑：

**Raspberry Pi (2012)** → **自造者空間普及** → **STEM 教育改革** → **邊緣運算 (2010s末)**

Raspberry Pi 的成功證明了「夠好就好」的硬體哲學——它不以最高規格取勝，而是以極低的價格和開放的生態系統，創造了一個前所未有的硬體創新平台。從智慧型機器人到氣象站，從家庭媒體中心到校園伺服器，Raspberry Pi 催生了無數的創作和創新。

## MOOC 的承諾與現實

2012 年 MOOC 的狂熱預言了高等教育的即到崩潰。十年後回顧：

**MOOC 狂熱 (2012)** → **完成率危機** → **B2B 轉型** → **微證書與職業教育** → **Coursera IPO (2021)**

MOOC 沒有消滅大學——但它改變了學習的方式。Coursera、Udacity 和 edX 都從「取代大學」的宏大願景轉向了「補充大學」的務實策略。微碩士（MicroMasters）、Google 職業證書等新形式的線上教育繼續發展。

## 從 Kepler 到深度學習的 GPU 運算

Nvidia Kepler 在 2012 年的發表正逢一個歷史性時刻——當年 Alex Krizhevsky、Ilya Sutskever 和 Geoffrey Hinton 的 AlexNet 在 ImageNet 競賽中以 GPU 訓練的深度卷積神經網路大幅超越傳統方法，拉開了深度學習革命的序幕。

**Fermi GPU (2010)** → **Kepler GPU (2012)** → **AlexNet (2012)** → **深度學習爆炸** → **Nvidia 成為 AI 晶片龍頭**

Kepler 架構提供了充足的 CUDA 核心來訓練深層神經網路。如果沒有 GPU 運算的進步，深度學習的發展可能會延遲多年。

## 2012 年的時代脈絡

- **歐債危機**：歐洲主權債務危機持續，科技產業相對不受影響
- **美國總統大選**：Obama 連任，社群媒體在選舉中的角色受到矚目
- **行動廣告突破**：Facebook 行動廣告營收在 2012 年首次突破，證明行動廣告的商業模式可行
- **App 經濟成熟**：App Store 在 2012 年達到 50 萬應用程式，Google Play 達到 70 萬
