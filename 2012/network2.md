# 物聯網概念的興起

## 從概念到產業

2012 年是物聯網（Internet of Things, IoT）從學術研究走向主流市場的關鍵年。雖然 Kevin Ashton 早在 1999 年就創造了「物聯網」一詞，但直到 2012 年，相關的技術條件——低成本感測器、無線通訊、雲端運算——才成熟到足以支撐大規模部署。

2012 年，Google 收購了 Nest Labs（智慧恆溫器製造商），這被視為科技巨頭進軍物聯網的信號。Cisco 預測到 2020 年將有 500 億個裝置連接到網路。IBM 推出了物聯網解決方案。Amazon 在雲端服務中加入了 IoT 平台。

## M2M 通訊的演進

物聯網的前身是 M2M（Machine-to-Machine）通訊——工業領域的自動化資料交換。M2M 通常使用專用網路和協定，而 IoT 將物聯裝置連接到公共網際網路，並利用標準的 IP 協定和雲端平台。

2012 年，針對物聯網的輕量級通訊協定開始成形。MQTT（Message Queuing Telemetry Transport）和 CoAP（Constrained Application Protocol）等專為低頻寬、低功耗裝置設計的協定開始受到關注。

## 挑戰與疑慮

物聯網在 2012 年也面臨不少挑戰：安全性不足（許多裝置缺乏基本的更新機制）、標準紛亂（各家廠商推出不相容的協定和平台）、以及隱私問題（大量感測器採集的個人資料如何保護）。
