# GitHub Copilot 正式版：AI 程式設計的收費時代

## 從預覽到正式發布

2022 年 6 月 21 日，GitHub Copilot 正式從「技術預覽」階段畢業，轉為付費服務——每月 10 美元（每年 100 美元），對學生和開源維護者免費。這是 GitHub 歷史上最重要的商業化里程碑之一。

Copilot 是基於 OpenAI Codex 模型的 AI 程式碼補全工具。它在 Visual Studio Code、JetBrains 和 Neovim 等主流編輯器中以插件形式運行。正式發布時，Copilot 支援超過 15 種程式語言——其中 Python、JavaScript、TypeScript、Ruby 和 Go 的表現最佳。

## 實際效果

GitHub 公布的數據顯示：
- 開發者接受 Copilot 建議的比率約為 25-30%
- 在某些語言中（如 Python），接受率可達 40%
- 使用 Copilot 的開發者報告生產力提升了 55%（根據 GitHub 的研究）
- 在重複性程式設計任務中尤為高效——單元測試、正則表達式和樣板程式碼

## 爭議：集體訴訟

2022 年 11 月，一群開發者對 GitHub、Microsoft 和 OpenAI 提起了集體訴訟——指控 Copilot 侵犯了開源程式碼的授權條款。

訴訟的核心論點是：
- Copilot 在公開的 GitHub 儲存庫上訓練——其中許多使用 GPL、MIT 和 Apache 等開源授權
- Copilot 有時會生成與訓練資料中「高度相似」或「逐字相同」的程式碼
- 使用者被要求接受 Copilot 生成的程式碼時——不知道它是否來自受版權保護的開源專案
- 根據開源授權條款——使用 GPL 程式碼的衍生作品必須採用相同的授權

GitHub 的回應是：Copilot 的輸出是全新的、轉換性的代碼——類似於人類開發者閱讀開源程式碼後學習到的寫法。

## 開放替代方案

2022 年，隨著 Copilot 轉為收費，開源社群加速了替代方案的開發：

- **Tabnine**：基於 GPT 模型的 AI 補全（提供免費層級）
- **CodeWhisperer**：AWS 在 2022 年 12 月的 re:Invent 上推出了自己的 AI 程式設計助手——對 AWS 使用者免費
- **StarCoder**：Hugging Face 和 ServiceNow 合作開發的開源程式碼語言模型
- **Code Llama**：Meta 在 2023 年推出的開源程式碼生成模型

## 長期影響

Copilot 的正式發布標誌著 AI 輔助程式設計從實驗性的玩具變成了專業開發者工具箱中的標準配置。到 2022 年底，Copilot 的付費使用者在所有主要程式語言中每天都產生超過數百萬行的 AI 生成程式碼。
