# Windows 1.04 與國際化

## 持續改進

1986 年，Microsoft 持續對 Windows 1.0 進行改進和更新。這一年發布了三個子版本：1.02（歐洲版，5 月）、1.03（國際版，8 月）和 1.04（4 月 1987，但準備工作在 1986 年進行）。

**Windows 1.02** 是第一個非英語版本，針對歐洲市場推出。它引入了對歐洲鍵盤配置和字元集（如西歐語系的重音符號）的支援——這是 Microsoft 跨國軟體本地化策略的起點。

**Windows 1.03** 增加了更多國際語言版本、新的螢幕和印表機驅動程式，並修復了多項錯誤。它也統一了美國版和歐洲版的功能差異。

## 國際化的戰略意義

Windows 1.0 的國際化工作對 Microsoft 具有長期的戰略價值：

**全球市場**：Microsoft 是最早意識到軟體國際化重要性的公司之一。Windows 的非英語版本讓 Microsoft 可以進入歐洲和亞洲市場。

**本地化流程**：Microsoft 在這一年建立了將 Windows 翻譯為多種語言的系統化流程——包括字串資源管理、鍵盤配置驅動程式和字型支援。

**競爭優勢**：當競爭對手（如 Digital Research 的 GEM）僅有英語版本時，Windows 的非英語版本提供了顯著的市場優勢。

## 市場反應

Windows 在 1986 年仍然不是一個成功的產品——它慢、功能有限、應用程式極少。但 Microsoft 對它的持續投資——包括國際化、新驅動程式和新功能——顯示了公司高層對 GUI 未來的堅定信念。
