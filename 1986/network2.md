# Cisco AGS 路由器：網路設備的里程碑

## 史丹佛的夫妻創業

1984 年，史丹佛大學的 Leonard Bosack 和 Sandy Lerner 創立了 Cisco Systems。Bosack 是史丹佛電腦科學系的網路工程師，Lerner 是商學院的 IT 經理。他們遇到一個實際問題：史丹佛校園內各種不相容的網路——TCP/IP、DECnet、Xerox XNS——無法互通。

Bosack 的解決方案是設計一台可以連接不同網路協定的閘道器（gateway），後來被稱為路由器（router）。1986 年，Cisco 推出了 AGS（Advanced Gateway Server）——第一台企業級的多協定路由器。

## AGS 的技術創新

AGS 的核心創新在於：

**多協定支援**：AGS 可以同時處理 TCP/IP、DECnet、AppleTalk、IPX 和 XNS——讓不同類型的網路可以互通。

**動態路由**：AGS 實作了 IGP（Interior Gateway Protocol）和 EGP（Exterior Gateway Protocol），允許路由器自動學習網路拓撲。

**專用硬體**：不同於當時用迷你電腦充當路由器的做法（如 Fuzzball），AGS 是專為路由功能設計的硬體。

## 網際網路的骨幹

Cisco AGS 的成功在於它抓住了網路互連的市場需求。ARPANET、NSFNET 和各種區域網路正在聚合為一個統一的網際網路，而連接這些網路需要可靠的路由器。

到 1980 年代末期，Cisco 已經成為網路設備市場的主導者——其路由器運行著全球網際網路骨幹的絕大部分。
