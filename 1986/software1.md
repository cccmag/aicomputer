# PageMaker 1.0：桌上出版產業的誕生

## 排版革命的正式啟動

1986 年，Aldus PageMaker 1.0 正式在 Macintosh 上推出（1985 年已有早期版本）。PageMaker 與 Macintosh 的 GUI、LaserWriter 的 PostScript 雷射印刷結合，創造了「桌上出版」（Desktop Publishing）這個全新產業。

PageMaker 1.0 提供了在個人電腦上前所未見的排版能力：可拖曳的文字區塊、精確的尺規和參考線、多欄排版、字型與段落格式控制。使用者可以在螢幕上看到完全等同於列印結果的版面——這在 1986 年是革命性的。

## 出版流程的民主化

在 PageMaker 之前，出版一份印刷品需要：作者寫稿 → 編輯校對 → 專業排版技師在昂貴的排版系統上作業 → 製版 → 印刷。整個流程需要專業的印刷廠和排版公司。

PageMaker 將排版工作從專業排版系統（如 Linotype、Compugraphic）轉移到了個人電腦上。編輯、設計師和作者可以在自己的桌上完成排版，直接輸出到雷射印表機。

## 產業漣漪效應

PageMaker 的成功帶動了整個桌上出版生態系的成長：

**硬體**：Apple LaserWriter 和高解析度顯示器需求增加。

**字型**：Adobe 的 PostScript 字型成為產業標準。

**設計**：平面設計師開始從傳統工具轉向數位工具。

**印刷**：小型印刷廠開始接受數位檔案而非實體排版。

Photoshop（1990）和 Illustrator（1987）的誕生也在這個生態系中找到市場——圖形設計師需要更多工具來準備在 PageMaker 中使用的素材。
