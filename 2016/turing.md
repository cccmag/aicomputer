# 2016 年圖靈獎：Tim Berners-Lee

## Web 的發明者

2016 年 ACM 圖靈獎頒給了 Tim Berners-Lee，表彰他發明了 World Wide Web——包括第一套 Web 瀏覽器、HTML、HTTP 和 URL。這項始於 1989 年 CERN 的研究項目，從根本上改變了人類的通訊、商業、教育和社會互動方式。

Tim Berners-Lee 在 1989 年提出了「資訊管理系統」的提案，隨後在 1990 年開發了第一個 Web 瀏覽器（WorldWideWeb，後來更名為 Nexus）、第一個 Web 伺服器（info.cern.ch）、以及 HTML、HTTP 和 URL 的初始規格。

## 三個核心標準

Berners-Lee 發明的三個核心技術至今仍是 Web 的基石：

- **HTML（超文本標記語言）**：用於建立和結構化網頁的標記語言
- **HTTP（超文本傳輸協定）**：用於在客戶端和伺服器之間傳輸資料的協定
- **URL（統一資源定位符）**：用於識別和定位網路資源的標準化位址

## 開放 vs 封閉的選擇

Berners-Lee 最偉大的決定之一是他堅持不為 Web 技術申請專利，也不從中收取授權費。他成功說服 CERN 將 Web 的原始碼和技術規範釋放到公有領域。這個決定確保了 Web 可以成為一個開放、自由、每個人都可以參與的平台。

如果 Berners-Lee 選擇將 Web 商業化，網際網路的歷史可能會完全不同——今天的世界可能是一個由少數公司控制的封閉網路生態。

## 圖靈獎演講

Berners-Lee 在獲獎演講中談到了 Web 的未來挑戰：隱私保護、假資訊、集中化趨勢、數位權利。他呼籲科技社群共同努力，確保 Web 繼續服務於人類的最佳利益。
