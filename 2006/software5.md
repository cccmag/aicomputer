# Firefox 2.0：IE 衰退的加速器

## 瀏覽器市場的轉變

2006 年，Internet Explorer 的市場佔有率首次開始明顯下滑。IE 在 2002-2003 年的市佔率曾超過 95%，但到 2006 年底已降至約 80%。Firefox 在 2006 年 10 月 24 日發布了 2.0 版，市佔率穩定在 15% 左右。這看似微小的數字變化，卻代表了 Web 生態系統的重大轉變。

## Firefox 2.0 的新功能

Firefox 2.0（代號 Bon Echo）在 Firefox 1.5（2005 年 11 月）的基礎上引入了多項改進：

- **分頁瀏覽的改進**：分頁關閉按鈕、崩潰恢復（Session Restore）
- **拼字檢查**：在網頁表單輸入時自動核對拼字
- **內建釣魚網站防護**：Google 提供的黑名單比對
- **搜尋外掛管理**：輕鬆新增和切換搜尋引擎
- **微格式支援**：自動偵測頁面中的 vCard、事件等結構化資料
- **Feed 預覽**：在瀏覽器中直接預覽 RSS/Atom Feed 內容
- **JWAL（JavaScript Windows 自動化限制）**：防止網站強制開啟彈出視窗

## IE7 的回應

Microsoft 在 2006 年 10 月 18 日發布了 Internet Explorer 7——IE6（2001 年）以來首次重大更新。IE7 引入了分頁瀏覽、釣魚網站篩選器、CSS 2.1 支援的改善。但對於許多 Web 開發者來說，IE7 的改進來得太慢、太晚了。

## 影響

Firefox 的存在不僅僅是作為 IE 的替代品——它改變了微軟的開發節奏。IE6 長達五年的停滯期說明了壟斷對瀏覽器創新的傷害。Firefox 的崛起打破了這個僵局，推動了整個瀏覽器市場的競爭。

Firefox 2.0 和 IE7 的發布讓 2006 年成為「瀏覽器大戰 2.0」的正式開端。這個競爭在 2008 年 Chrome 加入後更加激烈，最終讓全球網友享受到了更快、更安全、更標準的 Web 瀏覽體驗。
