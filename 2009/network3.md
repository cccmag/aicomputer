# 社群媒體整合與即時串流

## Twitter 的爆發

2009 年是 Twitter 從一個小眾的微部落格服務成長為全球性社群媒體平台的關鍵年份。Twitter 在 2009 年達到了數千萬使用者——伊朗選舉抗議（綠色革命）中，Twitter 成為了重要的訊息傳播管道，讓全球看到了社群媒體在政治動員中的力量。

Twitter 在 2009 年推出了多項重要的功能改進：搜尋功能、清單（Lists）、以及重新設計的介面。即時串流（real-time streaming）成為了 Twitter 的核心差異化能力——任何事件發生後的幾分鐘內，全球的 Twitter 使用者就會開始討論。

## 社群媒體的 API 生態

2009 年，Facebook 和 Twitter 的 API 成為了網路基礎設施的一部分。成千上萬的第三方應用透過這些 API 來讀取和發布社群媒體內容：

**社交登入**：Facebook Connect 讓使用者可以用 Facebook 帳號登入第三方網站——簡化了註冊流程，同時為 Facebook 提供了更多的使用者行為數據。

**社交分享**：網站的分享按鈕（Share to Facebook、Tweet this）在 2009 年變得無所不在——每個新聞網站和部落格都整合了社交分享功能。

**即時搜尋**：Google 在 2009 年加入了即時搜尋功能，將 Twitter 的即時串流整合到搜尋結果中。這讓 Twitter 的即時性優勢與 Google 的搜尋能力結合了起來。
