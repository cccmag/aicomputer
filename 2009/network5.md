# Windows Mobile 衰退與行動市場重整

## 帝國的反應遲緩

2009 年，Windows Mobile——曾經的智慧型手機市場領導者——正在經歷急劇的衰退。Microsoft 在 2009 年 2 月的 Mobile World Congress 上發表了 Windows Mobile 6.5，但這個版本只是在 6.1 的基礎上進行的小幅改進，完全無法與 iPhone OS 3.0 和 Android 1.5 Cupcake 競爭。

Windows Mobile 的問題根深蒂固：

**使用者介面過時**：Windows Mobile 的介面仍然是為觸控筆設計的——小按鈕、複雜的功能表層次、不支援流暢的指尖觸控。

**開發者生態落後**：Windows Mobile 的開發工具和使用者體驗遠不如 iPhone 和 Android。

**硬體碎片化**：不同的製造商使用不同的螢幕尺寸、解析度和硬體配置，導致應用程式相容性問題。

## Symbian 的困境

Symbian——Nokia 的智慧型手機作業系統——在 2009 年仍然是出貨量最大的手機作業系統，但也在快速失去開發者和消費者的青睞。2008 年 Nokia 收購了 Symbian 的全部股權，並在 2009 年宣布將其開源——但這些努力來得太晚了。

## 新市場格局的形成

2009 年底的智慧型手機市場格局已經清晰：iOS 和 Android 代表了未來，Windows Mobile 和 Symbian 代表了過去。Apple 的封閉生態和 Google 的開放策略代表了兩種截然不同的哲學——但兩者都比舊世代的行動作業系統更優秀。

這個市場重整對整個科技產業產生了深遠影響：Nokia 的市值在接下來的五年內縮水了 90% 以上，最終將其手機業務出售給 Microsoft。
