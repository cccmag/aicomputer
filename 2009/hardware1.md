# Windows 7：從 Vista 的陰影中走出

## Vista 的教訓

Windows Vista（2007 年 1 月發布）是 Microsoft 歷史上最具爭議的產品之一。它的硬體需求過高、驅動程式相容性問題嚴重、UAC（使用者帳戶控制）頻繁打斷使用者操作、效能表現不如 Windows XP。許多企業和個人使用者選擇跳過 Vista，繼續使用 XP。

Microsoft 從 Vista 的失敗中吸取了教訓。Windows 7 的開發代號為「Blackcomb」和「Vienna」，但最終的目標非常明確：保留 Vista 的底層改進（新的核心、驅動程式模型、安全性架構），同時大幅改善使用者體驗和效能。

## 2009 年 10 月 22 日的發布

Windows 7 於 2009 年 10 月 22 日正式上市。它採用了與 Vista 相同的 NT 6.1 核心，但在使用者體驗、效能和資源消耗方面進行了全面優化。Windows 7 可以在 Vista 無法流暢執行的硬體上運作，且開機速度大幅提升。

Windows 7 引入了多項重要功能：

**工作列預覽**：將滑鼠懸停在工作列圖示上時，會顯示應用程式的即時縮圖預覽。Aero Peek 讓使用者可以暫時隱藏所有視窗，查看桌面。

**跳躍清單**：在工作列圖示上按右鍵，會顯示最近使用的文件和常用操作——這是一個微小但效率提升巨大的設計。

**視窗管理**：Aero Snap（拖曳視窗到螢幕邊緣自動調整大小）、Aero Shake（搖動視窗讓其他視窗最小化）——簡單而直觀的視窗管理手勢。

**HomeGroup**：簡化了家庭網路中的檔案和印表機共享。

## 市場成功

Windows 7 在發布後獲得了廣泛的好評。它在第一年的銷售量超過了 2.4 億套，成為歷史上銷售最快的作業系統。企業使用者終於有了一個可以從 XP 遷移的現代化作業系統。Windows 7 的成功為 Microsoft 挽回了 Vista 帶來的聲譽損失。
