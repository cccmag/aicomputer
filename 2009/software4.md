# Google Chrome OS 宣布

## 以瀏覽器為中心的作業系統

2009 年 7 月 7 日，Google 正式宣布了 Chrome OS——一個基於 Linux 的開源作業系統，核心是 Chrome 瀏覽器。Google 的策略非常清楚：未來的運算將全部發生在雲端，作業系統的角色只是運行一個瀏覽器。

Chrome OS 的設計理念：

**速度**：開機時間目標為 7 秒。系統啟動後直接進入 Chrome 瀏覽器——沒有桌面、沒有檔案總管、沒有開始功能表。所有應用程式都是網頁應用。

**安全**：Chrome OS 採用多層次安全架構——驗證開機、唯讀系統分割區、沙盒瀏覽器、自動更新。使用者不需要安裝防毒軟體或管理系統更新。

**簡潔**：作業系統本身不可見——它只是一個載入 Chrome 瀏覽器的載入器。所有的資料儲存在雲端，本地只是快取。

## 對於傳統 PC 的挑戰

Chrome OS 的宣布對整個 PC 產業產生了衝擊——特別是對 Microsoft。如果 Google 可以讓使用者完全在瀏覽器中完成所有工作，Windows 的存在價值就會大打折扣。

但 Chrome OS 也面臨著一些質疑：沒有穩定的網路連線怎麼辦？需要執行桌面級應用程式怎麼辦？網頁應用在功能深度上能否滿足專業需求？這些問題在 2009 年還不明確，需要在後續的 Chromebook 產品中逐步驗證。

## 開源策略

Chrome OS 的原始碼被發布在 Chromium OS 專案中——類似於 Chrome 瀏覽器與 Chromium 專案的關係。任何人都可以查看原始碼、自行編譯和修改。Google 也允許 OEM 廠商設計和生產 Chromebook。
