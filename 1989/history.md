# 歷史穿梭：1989 年的種子

## 從提案到全球資訊網

1989 年 3 月的 WWW 提案是人類通訊史上最重要的文件之一：

**Memex (1945, Bush)** → **NLS (1968, Engelbart)** → **HyperCard (1987)** → **WWW 提案 (1989)** → **第一個瀏覽器 (1990)** → **第一個網站 (1991)** → **Mosaic 瀏覽器 (1993)** → **全球普及 (1995-)**

Berners-Lee 在提案中寫道：「我們應該找到一個方法來組織和連結 CERN 的所有資訊——包括文件、圖片、程式和資料——讓任何人都可以方便地存取。」他的主管評語「模糊，但令人興奮」現在看來是史上最準確的預測。

1989 年的 WWW 概念在當年幾乎沒有引起注意。如果 CERN 的管理層當時拒絕了這個提案，全球資訊網可能要在幾年後才會被其他人重新發明。

## 從 Intel 486 到 x86 王朝

Intel 486（1989）的架構創新開創了 x86 的高效能時代：

**Intel 8086 (1978)** → **80286 (1982)** → **80386 (1985)** → **80486 (1989)** → **Pentium (1993)** → **Pentium Pro (1995)** → **Pentium 4 (2000)** → **Core 系列 (2006-)**

486 的 FPU 整合、L1 快取和管線化設計是 x86 處理器從「夠用」走向「高效能」的關鍵一步。486 處理器後來被廣泛用於嵌入式系統，直到 2000 年代仍然在生產——總生產壽命超過 15 年。

## 從遊戲機到隨身遊戲

Nintendo Game Boy（1989）改變了人們對遊戲的認知：

**Game & Watch (1980)** → **Game Boy (1989)** → **Game Boy Color (1998)** → **Game Boy Advance (2001)** → **Nintendo DS (2004)** → **Nintendo Switch (2017)**

Game Boy 的成功核心——硬體保守但軟體優秀——至今仍影響著 Nintendo 的產品策略。當競爭對手追求更快的處理器和更華麗的螢幕時，Nintendo 專注於遊戲性和電池續航，最終以 Game Boy 壓倒了所有競爭對手（包括 Sega Game Gear 和 Atari Lynx）。

## 浮點數的統一

William Kahan 的 IEEE 754 標準雖然在 1985 年就已發布，但 1989 年它的影響力才真正顯現：

**各自為政的浮點數 (1960s-70s)** → **IEEE 754 (1985)** → **Intel 80486 FPU 整合 (1989)** → **通用浮點標準普及 (1990s)**

沒有 IEEE 754，科學計算軟體在不同的硬體上就會產生不同的結果，這讓協作研究和軟體分發變得非常困難。Kahan 的工作讓「可重複的計算」成為可能。

## 1989 年的時代脈絡

- **柏林圍牆倒塌**（11 月 9 日）：冷戰的象徵性終結，全球資訊自由流通的政治障礙開始瓦解
- **天安門事件**（6 月 4 日）：中國的民主化運動被鎮壓，影響了中國的科技發展道路
- **WWW 的搖籃**：CERN 的國際合作氛圍為 Berners-Lee 的跨國界、跨平台願景提供了理想的孕育環境
- **日本經濟高峰**：日本泡沫經濟達到頂點，日本電子公司（Sony、Nintendo、Sharp）在全球消費電子市場占主導地位
