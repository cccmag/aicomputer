# 1989 年資訊科技年鑑：前言

1989 年是全球資訊網的孕育之年。三月，在瑞士日內瓦的歐洲核子研究組織（CERN），英國科學家 Tim Berners-Lee 提交了一份題為「資訊管理：一個提案」的文件。他的主管 Mike Sendall 在備忘錄上寫下了著名的評語：「模糊，但令人興奮。」這份提案後來成為了 World Wide Web 的誕生證明。

硬體方面，Intel 推出了 80486 處理器，首次將數學協同處理器（FPU）整合在同一晶片中，大幅提升了 PC 的計算能力。Nintendo 的 Game Boy 在四月上市，憑藉其超長續航和《俄羅斯方塊》的搭配，成為史上最成功的掌上遊戲機之一。Apple 則推出了 Macintosh Portable——雖然笨重（7.2 公斤），但它是 Apple 邁向筆記型電腦的第一次嘗試。

在通訊領域，9600bps 的數據機開始普及，讓撥接上網的速度提升了一個數量級。高畫質電視（HDTV）的展示在北美引起關注，預示著電視產業的數位化轉型。

1989 年也是冷戰終結之年。柏林圍牆在十一月倒塌，全球政治格局的變化為後來的網際網路爆炸性成長創造了自由化的土壤。這一年，TCP/IP 協議在全球範圍內持續擴張，網際網路的商業化應用開始萌芽。
