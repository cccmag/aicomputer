# Microsoft Word for Windows：文書處理的統一

## 從 DOS 到 Windows

1989 年 11 月，Microsoft 發表了 Word for Windows 1.0。這不是 Microsoft 的第一個文書處理軟體——Word for DOS 早在 1983 年就已問世，Word for Mac 也在 1985 年推出。但 Word for Windows 的出現標誌著 Windows 平台文書處理的成熟。

Word for Windows 1.0 利用了 Windows 的圖形介面優勢：所見即所得（WYSIWYG）的編輯器，讓使用者在螢幕上看到的樣式與印表機輸出完全一致。這在 DOS 時代是難以實現的——DOS 的文字模式無法顯示不同的字型和格式。

## 終結對手

Word for Windows 的推出直接挑戰了當時文書處理市場的霸主 WordPerfect。WordPerfect 在 DOS 時代占據了超過 50% 的市場佔有率，但它對 Windows 的轉型反應遲緩。Word for Windows 則充分利用了圖形介面的優勢，提供了更直觀的操作方式。

到 1990 年代中期，Word for Windows 已經徹底取代了 WordPerfect 的地位。這場勝利讓 Microsoft 在辦公室軟體市場確立了主導地位。

## 與 Excel 的整合

Word for Windows 是 Microsoft Office 策略的一部分。1989 年，Excel for Windows 2.0 也已經上市。Microsoft 的策略是將文書處理、試算表和其他生產力工具整合為一套辦公室套裝軟體——這個策略後來被證明極為成功。
