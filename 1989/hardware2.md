# Nintendo Game Boy：掌上遊戲的革命

## 橫井軍平的設計哲學

1989 年 4 月 21 日，日本任天堂公司推出了 Game Boy 掌上遊戲機。它的設計師是任天堂的傳奇硬體工程師橫井軍平（Gunpei Yokoi），他信奉一套稱為「枯萎技術的水平思考」的設計哲學——使用成熟、廉價的技術來創造全新的產品體驗。

Game Boy 的硬體規格在當時並不出色：

- 8 位元 Sharp LR35902 處理器（4.19 MHz）
- 2.6 吋反射式黑白 STN 液晶螢幕（160×144 畫素）
- 4 種灰階
- 立體聲耳機輸出（但只有單聲道內建喇叭）
- 4 顆 AA 電池，續航長達 15-35 小時

## 殺手級應用

Game Boy 成功的關鍵不在於硬體，而在於軟體。隨主機同步上市的《Tetris》（俄羅斯方塊）是由 Nintendo 的 Henk Rogers 從蘇聯遊戲設計師 Alexey Pajitnov 那裡取得授權的。這款簡單但極度上癮的方塊遊戲，完美匹配了 Game Boy 隨開隨玩的特性。

《Tetris》和 Game Boy 的組合創造了一個文化現象，讓非傳統玩家（包括大量女性玩家）也加入了掌上遊戲的行列。

## 統治級的市場表現

Game Boy 在日本上市首日就售出 300,000 台。在全球，Game Boy（含後續的 Game Boy Color）最終賣出了超過 1.18 億台，成為史上最暢銷的遊戲機之一。

Game Boy 的成功說明了在消費電子產品中，使用者體驗和軟體生態比硬體規格更為重要——這個教訓至今仍然適用。
