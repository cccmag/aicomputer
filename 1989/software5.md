# Lotus 1-2-3 Release 3：試算表的新維度

## 試算表領域的霸主

Lotus 1-2-3 是 1980 年代個人電腦軟體的傳奇。1983 年問世後，它立即成為 IBM PC 的殺手級應用，讓企業用戶有理由投資購買 PC。名稱中的「1-2-3」代表了三大功能：試算表（1）、圖表（2）和資料庫（3）。

1989 年，Lotus 發表了 1-2-3 Release 3 和 Release 2.2 兩個版本。Release 3 是一次重大的技術升級。

## Release 3 的革命性功能

Release 3 引入了三維試算表的概念——使用者可以同時操作多個工作表（worksheet），並直接在儲存格中引用其他工作表中的資料。例如，`+SheetA:JAN.TOTAL` 可以直接參考另一個工作表中的儲存格。

這個版本還支援：

- **多視窗同時顯示**：可以同時查看多個工作表
- **改良的圖表功能**：更多圖表類型和更好的輸出品質
- **OS/2 和 DOS 雙平台支援**：充分利用保護模式記憶體

## 三維試算表的影響

三維試算表的概念在當時是突破性的。財務分析人員可以在一個檔案中組合多個部門的預算，或將十二個月的報表疊加在一起。這種設計後來被 Microsoft Excel 採用並發揚光大。

然而，Lotus 在 Windows 轉型上犯了致命錯誤。直到 1991 年才推出 Lotus 1-2-3 for Windows，而 Microsoft Excel 早已占據了 Windows 試算表市場的主導地位。
