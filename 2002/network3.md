# Skype 與 VoIP 革命前夕

## 點對點語音通話的概念

2002 年，Skype 的雛形正在形成。這是一家由 Niklas Zennström 和 Janus Friis 創立的公司——這兩位創業家也是 KaZaA（P2P 檔案分享軟體）的創辦人。他們在 KaZaA 的 P2P 技術基礎上，開發了全新的點對點語音通訊軟體。

Skype 的核心理念非常簡單：利用 P2P 網路技術，讓任何人在網際網路上免費撥打語音電話，不需要傳統的電話線路。

## 技術的革命

Skype 的技術架構在當時是革命性的：

**P2P 目錄服務**：與傳統 VoIP 服務依賴中央伺服器不同，Skype 像 KaZaA 一樣，使用去中心化的 P2P 網路來尋找使用者。任何節點都可以成為超級節點（Supernode），處理其他節點的查找請求。

**穿越 NAT 和防火牆**：Skype 的自動穿越技術讓它可以在複雜的網路環境中工作——這對家庭和企業使用者來說至關重要。

**高品質語音編碼**：使用 iLBC 和 iSAC 編碼器，在撥接速度下也能提供可接受的語音品質。

**端對端加密**：Skype 對所有語音通訊進行加密，提供了比傳統電話網路更好的隱私保護。

## 電信業的威脅

雖然 Skype 1.0 要到 2003 年才正式發布，但 2002 年的開發和早期測試已經讓傳統電信業者感到了威脅。VoIP 的撥打成本——免費——與電信公司長途電話的費率形成了鮮明對比。

Skype 並不是第一個 VoIP 服務（Net2Phone、Vonage 等更早出現），但它是第一個將 P2P 技術與語音通話成功結合的服務。這個技術組合讓 Skype 可以以極低的成本營運——不需要龐大的中央交換機，頻寬由使用者的電腦共同貢獻。
