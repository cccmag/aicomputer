# Mozilla 1.0：開放原始碼瀏覽器

## 從 Netscape 灰燼中重生

1998 年，Netscape 通訊公司在與 Microsoft Internet Explorer 的瀏覽器大戰中敗下陣來，決定將其瀏覽器原始碼開放，創立了 Mozilla 專案。然而，Netscape 的原始碼老舊且難以維護——Mozilla 社群基本上重寫了整個瀏覽器。

經過四年多的開發，Mozilla 1.0 終於在 2002 年 6 月 5 日正式發布。這是一個重要的里程碑——它證明了開放原始碼開發模式可以打造出一個功能完整的現代化瀏覽器。

## 技術創新

Mozilla 1.0 包含了多項重要的技術創新：

**Gecko 排版引擎**：Gecko 是 Mozilla 的核心技術——一個高效、符合標準的 HTML/CSS 渲染引擎。Gecko 比當時的 Internet Explorer 更嚴格地遵守 W3C 標準，為網頁開發者提供了一個更可預測的渲染平台。

**跨平台支援**：Mozilla 1.0 可在 Windows、Mac OS 和 Linux 上運行，提供了統一的使用者體驗——這是 IE 不具備的優勢。

**分頁瀏覽**：雖然分頁瀏覽並非 Mozilla 首創，但 Mozilla 1.0 對分頁的支援比 IE 更加完善，使用者可以在一個視窗中打開多個網頁。

**擴充套件系統**：Mozilla 的擴充套件架構讓第三方開發者可以為瀏覽器加入新功能，開創了瀏覽器生態系統的早期形式。

## 與 IE 的競爭

2002 年，Internet Explorer 6（2001 年隨 Windows XP 推出）的市佔率超過了 90%。Mozilla 1.0 的市場份額雖然微不足道（不到 5%），但它代表了對微軟壟斷的一個重要制衡。

Mozilla 1.0 為後來的 Firefox（2004 年）奠定了基礎。Firefox 從 Mozilla Suite 中獨立出來，專注於瀏覽器功能——這項策略調整最終讓 Firefox 在瀏覽器市場中獲得了可觀的佔有率。
