# 歷史穿梭：2002 年的種子

## 從 AWS 到雲端運算時代

2002 年 Amazon Web Services 概念的醞釀，是雲端運算的起點：

**AWS 概念 (2002)** → **Amazon S3 (2006)** → **Amazon EC2 (2006)** → **雲端運算主宰 IT**

貝佐斯在 2002 年的 API 命令強制 Amazon 內部的所有基礎設施都服務化——這個決策讓 Amazon 在雲端運算市場取得了十年的先發優勢。2006 年推出的 S3（儲存）和 EC2（運算）徹底改變了企業採購和部署 IT 的方式。

## 從 Nutch 到 Hadoop 到大數據

2002 年 Nutch 搜尋引擎的探索，催生了整個大數據生態系統：

**Nutch (2002)** → **Hadoop (2006)** → **Spark (2014)** → **大數據時代**

Doug Cutting 和 Mike Cafarella 在 2002 年遇到的規模化問題，引導他們發現了 Google File System 和 MapReduce 論文。這些概念最終演化為 Hadoop——大數據革命的基礎設施。

## 從 Mozilla 1.0 到 Firefox 到 Web 標準

2002 年 Mozilla 1.0 的發布，是開放原始碼瀏覽器挑戰 IE 壟斷的開始：

**Mozilla 1.0 (2002)** → **Firefox 1.0 (2004)** → **Chrome (2008)** → **現代瀏覽器時代**

Mozilla 的 Gecko 引擎對 Web 標準的嚴格遵守，推動了整個網際網路從「IE-only」的黑暗時代走向標準化。雖然 Firefox 從未擊敗 IE 的市佔率，但它的競爭壓力迫使微軟改善了 IE 的標準相容性。

## 從 iMac G4 到 Apple 設計語言的定型

2002 年 iMac G4 的燈檯設計，奠定了 Apple 接下來十年的設計方向：

**iMac G4 (2002)** → **MacBook Pro (2006)** → **iPhone (2007)** → **iPad (2010)**

白色外觀、簡潔線條、金屬與塑膠的混合——這些設計元素在 iMac G4 中確立，後來在 Apple 的每一款產品中都能看到。iMac G4 被 MoMA 永久收藏，證明了科技產品也可以是藝術品。

## 從 Cell 到異質運算

2002 年啟動的 Cell 處理器設計，預示了異質多核心運算的未來：

**Cell 設計 (2001-2002)** → **PlayStation 3 的 Cell (2006)** → **GPU 通用計算** → **SoC 時代**

Cell 的「通用核心 + 專用加速器」架構，後來被智慧型手機 SoC（如 Qualcomm Snapdragon 和 Apple A 系列）採用。GPU 通用計算（GPGPU）的概念也與 Cell 的設計理念一脈相承。

## 從 RSA 到量子安全的挑戰

2002 年圖靈獎肯定了 RSA 對網路安全的貢獻。四十年後，量子計算的發展正在挑戰 RSA 的安全性——如果量子電腦能夠高效執行 Shor 演算法，RSA 的安全性將被打破。這引發了「後量子密碼學」（Post-Quantum Cryptography）的研究浪潮——尋找能夠抵抗量子電腦攻擊的新型密碼系統。

## 2002 年的時代脈絡

- **泡沫谷底**：那斯達克指數跌至 1,100 點的谷底，但最壞的時刻正在過去
- **轉型之年**：Amazon 開始思考雲端、Google 的廣告模式成熟、Apple 慶祝重生
- **寬頻爆發**：寬頻網路超越撥接，永遠在線成為常態
- **開源之春**：Mozilla 1.0、Firebird、MySQL——開源軟體從邊緣走向主流
- **未來的種子**：Hadoop、AWS、Skype、RSS——這些始於 2002 的概念，將在未來十年改變世界

2002 年的關鍵字是「播種」——這一年看似平靜，但眾多改變未來的種子正在暗暗發芽。
