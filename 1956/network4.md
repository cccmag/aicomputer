# Teletype Corporation Model 28 電傳打字機

一九五六年，Teletype Corporation 推出了 Model 28 電傳打字機，這是當時最先進的電傳列印終端機，廣泛用於 SAGE 防空系統、軍事通訊網路以及商業電報服務。

Model 28 延續了 Teletype 公司在電傳打字機領域的長期領導地位，其前身 Model 15 已經在全球通訊網路中廣泛使用。Model 28 的設計目標是提供更可靠的機械結構與更好的列印品質，以滿足 SAGE 系統對全天候不間斷運作的要求。

Model 28 的機械設計極為堅固，採用了重型的鑄鐵底座與精密的齒輪傳動系統。它能夠以每分鐘一百個字的速率進行列印，使用五單位的 Baudot 編碼來表示字元。Model 28 不僅能夠接收並列印訊息，還能透過鍵盤發送訊息，實現雙向通訊。

在 SAGE 系統中，Model 28 被用作操作員控制台的輸出設備，用於列印系統狀態報告、雷達目標追蹤資料與警報訊息。它也被用於 SAGE 網路中各作戰中心之間的文字通訊。Model 28 的可靠性在冷戰時期的戰略防空系統中至關重要。

Model 28 代表了一個正在消逝的時代——全機械式的電傳打字機。但它的通訊協定——非同步串列傳輸、字元編碼、開始位元與停止位元——直接影響了後來電腦終端機的 RS-232 串列通訊標準。Model 28 使用的 Baudot 編碼雖然後來被 ASCII 編碼取代，但電傳打字機建立的通訊習慣與協定框架一直延續到現代網路時代。UNIX 系統中的 /dev/tty 與終端機模擬器，其根源都可以追溯到 Model 28 這樣的電傳設備。
