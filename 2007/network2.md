# Facebook 的全球化：社交網路崛起

## 從校園到世界

2007 年是 Facebook 從美國大學生網站轉變為全球社交網路的關鍵一年。2006 年 9 月，Facebook 已經向所有人開放註冊——不再需要 .edu 電子郵件。到了 2007 年，Facebook 的使用者從 1,200 萬成長到 5,800 萬，成為全球成長最快的網站之一。

## Facebook Platform

2007 年 5 月 24 日，Mark Zuckerberg 在 f8 開發者大會上宣布了 Facebook Platform——允許第三方開發者在 Facebook 內部建立應用程式的 API 平台。

這是一個大膽的平台策略：開發者可以建立小工具（Widgets）或應用程式，透過 Facebook 的社交圖譜（Social Graph）進行傳播。最成功的早期應用包括：虛擬禮物、拼字遊戲 Scrabulous、以及讓人們互相「扔綿羊」的 silly 應用。

## 社交圖譜與 API

Facebook Platform 的核心是對社交圖譜——人與人之間的關係網路——的程式化存取。開發者可以透過 API 讀取使用者的個人資料、朋友列表、相片和動態消息（須經使用者授權）。

這個概念在 2007 年是革命性的。其他社交網站（如 MySpace）主要是一個展示個人頁面的空間，而 Facebook Platform 讓社交關係本身成為了一個運算平台。這項策略為 Facebook 帶來了病毒式的成長——每個應用程式都是一個 Facebook 的廣告。
