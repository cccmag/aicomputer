# Mac OS X Leopard：Unix 桌面的成熟

## 第十版作業系統

2007 年 10 月 26 日，Apple 發布了 Mac OS X 10.5 Leopard——這是 Mac OS X 的第六個主要版本，也是第一部在 Intel Mac 與 PowerPC Mac 上同時運行的版本。自 2001 年 Mac OS X 首次發布以來，Leopard 代表了這個基於 NeXTSTEP 的作業系統的全面成熟。

## Time Machine

Leopard 最受注目的新功能是 Time Machine——一個革命性的備份系統。Time Machine 自動為系統檔案和使用者資料建立每小時、每天、每週的增量備份，並以圖形化的時間軸讓使用者可以「回到過去」。

Time Machine 的技術核心是使用硬連結（hard links）來實現增量備份的空間效率——修改過的檔案才會佔用新的空間，未修改的檔案則指向之前的副本。使用者只需連接一顆外接硬碟或設定網路磁碟，Time Machine 就會自動無縫地保護資料。

## 其他重要改進

Boot Camp 讓 Intel Mac 使用者可以安裝 Windows 作為雙系統。Spaces 提供了虛擬桌面功能。Quick Look 讓使用者可以不開啟應用程式就預覽檔案內容。而全新的 Dock 採用了 3D 反射效果。

但 Leopard 最重要的工作是在底層：它包含了對 64 位元應用程式的完整支援、更好的多核心處理器利用、以及更嚴格的記憶體保護。這些改進為 Apple 後續的 Intel 遷移和硬體升級奠定了作業系統基礎。
