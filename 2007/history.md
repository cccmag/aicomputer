# 歷史穿梭：2007 年的種子

## iPhone 十年：從手機到平台

2007 年 iPhone 的發表啟動了一條清晰的進化路線：

**iPhone (2007)** → **App Store (2008)** → **iPhone 3G (2008)** → **iPad (2010)** → **智慧型手機普及**

iPhone 不僅重新定義了手機，也重新定義了整個產業。Nokia、Motorola、BlackBerry 等曾經的霸主在五年內失去了市場主導地位。App Store 經濟創造了數百萬個工作——從 iOS 開發者到 Uber 司機——一個全新的經濟生態系統。

更根本的影響是，iPhone 將網際網路從書房和辦公室的桌面解放出來，放進了每個人的口袋。即時通訊、地圖導航、行動支付、社交媒體——這些今日習以為常的應用，都依賴於 2007 年這個觸控螢幕裝置的問世。

## Android 與開放的競爭

Google 在 11 月宣布 Android，為行動市場帶來了第二個主要平台：

**Android (2007)** → **HTC Dream (2008)** → **Android 開放成長** → **全球最大行動平台**

如果說 iPhone 開啟了智慧型手機時代，Android 則讓這個時代變得大眾化。Android 的開放策略讓三星、HTC、LG、小米等製造商可以製造各式各樣的裝置，從旗艦機到入門機，覆蓋了 iPhone 無法觸及的市場。

## 從版本控制到社交化開發

GitHub 的創立改變了軟體開發的方式：

**Git (2005)** → **GitHub (2007)** → **社交化程式設計** → **開源協作標準**

在 GitHub 之前，版本控制是一個技術細節；在 GitHub 之後，它是社會協作的基礎設施。

## 大數據的起點

**Google MapReduce (2004)** → **Hadoop (2006)** → **Cloudera (2007)** → **大數據產業**

Hadoop 的商業化讓海量資料分析從 Google 的專利變成了大眾工具。後來的 Spark、Kafka、Hive、Pig 等技術都建立在 Hadoop 生態系統之上。

## 2007 年的時代脈絡

- **金融風暴前夜**：2007 年是次貸危機醞釀的一年，科技產業相對不受影響，但即將到來的經濟衰退會改變產業格局
- **Web 2.0 高峰期**：社交網路、使用者生成內容的觀念已經成熟，Facebook、YouTube、Twitter（創立於 2006）代表了網路的新方向
- **行動化的轉捩點**：2007 年之前，行動上網是小眾功能；2007 年之後，行動上網成為標準配備
- **開源的成熟**：Linux、Apache、MySQL、Python 等開源技術已經從邊緣進入主流，GitHub 的出現加速了這一進程

2007 年最迷人特點在於：這一年沒有單一主導故事，而是多條歷史線索——行動、社交、開源、大數據——同時加速前進的交會點。
