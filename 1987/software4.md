# OS/2 1.0：IBM 與 Microsoft 的共同作業系統

## 短暫的合作

1987 年 4 月，IBM 發表 PS/2 硬體的同時，也與 Microsoft 共同發表了 OS/2 1.0 作業系統。OS/2 的設計目標是取代 MS-DOS，成為新一代 IBM PC 相容機的主要作業系統。

OS/2 1.0 是一個純文字模式的作業系統，它的主要特色是：

- **保護模式**：支援 Intel 80286 和 80386 的保護模式，允許系統保護不同程式之間的記憶體
- **多工**：真正的多工能力，一個程式當機不會拖垮整個系統
- **高記憶體容量**：突破了 DOS 的 640 KB 記憶體限制，最高支援 16 MB
- **API 相容**：向下相容 DOS 應用程式

## 圖形介面版本的延遲

OS/2 1.0 只有文字模式，沒有圖形使用者介面。它的圖形版本 OS/2 1.1（含 Presentation Manager）直到 1988 年 11 月才問世。

在這段時間裡，Microsoft 和 IBM 的合作關係出現了裂痕。Microsoft 越來越專注於 Windows 的發展，而 IBM 則堅持 OS/2 才是個人電腦的未來。兩家公司最終在 1990 年代初分道揚鑣。

## 歷史的岔路

OS/2 是個人電腦作業系統歷史上一個重要的岔路。如果 OS/2 成功了，今天的 PC 可能運行著 IBM 的作業系統而非 Microsoft 的 Windows。但 OS/2 的硬體需求過高、應用程式生態不夠健全，加上 Microsoft 的策略轉向，最終讓 Windows 3.0 和後續版本成為了市場的主宰。

OS/2 雖然在桌面市場失敗，但在 IBM 的大型企業客戶中仍有忠實用戶，直到 2006 年才正式終止支援。
