# Microsoft Windows 2.0：重疊視窗的登場

## 從平鋪到重疊

1987 年 12 月 9 日，Microsoft 發表了 Windows 2.0。與 Windows 1.0 相比，2.0 版本最重要的改進是支援重疊視窗——Windows 1.0 只能以平鋪方式排列視窗，這讓使用者感到極為不便。

Windows 2.0 還首次支援：

- 可調整大小的視窗
- 鍵盤快速鍵（如 Ctrl+C 複製、Ctrl+V 貼上）
- 16 色 VGA 圖形支援
- 桌面圖示（icon）
- 物件化的程式管理概念

## Windows/286 與 Windows/386

Windows 2.0 分為兩個版本：Windows/286 和 Windows/386。Windows/286 主要針對 80286 處理器，使用即時模式，但可以透過 EMS 記憶體來突破 640 KB 的限制。

Windows/386 更為先進：它使用 Intel 80386 的虛擬 8086 模式，允許真正的多工——多個 DOS 應用程式可以同時執行，每個程式都在自己的虛擬機器中。Windows/386 在開發完成後於 1987 年 9 月即開始出貨，比 Windows 2.0 還早。

## 法律糾紛

Windows 2.0 的圖形使用者介面與 Macintosh 系統軟體極為相似，這導致 Apple 在 1988 年 3 月對 Microsoft 提起訴訟，指控其侵犯了 Apple 的著作權。這場官司拖了數年，最終法官判決 Microsoft 勝訴，因為 Apple 的授權協議已經涵蓋了 Windows 的元素。
