# 歷史穿梭：1976 年的科技交會點

## 密碼學的節點

1976 年 Diffie-Hellman 的論文是一條技術傳承的匯流點：

**Shannon 的通訊理論 (1948)** →  
**Feistel 的 Lucifer 密碼 (1973)** →  
**Diffie-Hellman 公鑰密碼學 (1976)** →  
**RSA (1977)** →  
**SSL/TLS (1994)** → **HTTPS 與電子商務**

沒有公鑰密碼學，就不可能有安全網路交易。每一次你在瀏覽器上看到鎖頭圖示時，正在使用的是 1976 年那次思想的成果。

## 向量處理的未來

Cray-1 開創的向量處理路線：

**CDC 6600 (1964, Seymour Cray)** →  
**CDC STAR-100 (1974)** →  
**Cray-1 (1976)** →  
**Cray X-MP (1982)** →  
**Cray Y-MP (1988)** →  
**現代 GPU 中的 SIMT 架構 (2006 至今)**

今天 NVIDIA 的 GPU 使用的正是向量處理的後代概念——SIMT（單指令多執行緒）——一群浮點運算單元同時執行相同的指令。從 Cray-1 到 Tesla GPU，這條血脈已經持續了將近五十年。

## 從 Homebrew 到世界最大公司

Apple I 的故事是矽谷傳奇的縮影：

**Homebrew Computer Club (1975)** → **Apple I (1976)** → **Apple II (1977)** → **Macintosh (1984)** → **iPod (2001)** → **iPhone (2007)** → **Apple Silicon (2020)**

1976 年 4 月 1 日成立的 Apple Computer，最終成為全球市值最高的公司。而這一切的起點，只是一塊賣 666.66 美元的主機板。

## 小型化的規律

5.25 吋軟碟的誕生反映了儲存媒體的小型化趨勢：

**IBM 8 吋軟碟 (1971)** → **5.25 吋軟碟 (1976)** → **3.5 吋軟碟 (1981)** → **CD-ROM (1985)** → **USB 隨身碟 (2000)** → **雲端儲存 (2010s)**

不變的是，每一次小型化都擴大了使用者的基數，降低了存取資訊的成本。

## 物件導向的純化之路

Smalltalk-76 將 Simula 的類別概念推向極致：

**Simula 67 (1967)** → **Smalltalk-72 (1972)** → **Smalltalk-76 (1976)** → **Smalltalk-80 (1980)** → **Objective-C (1984)** → **Java (1995)** → **Swift (2014)**

Smalltalk 的純物件模型對後續語言的影響遠超過它在商業上的成就。

## 規格戰的教訓

VHS vs Betamax 是科技史上最重要的格式戰爭案例：

- **封閉 vs 開放授權**：Sony 限制 Betamax 授權，JVC 廣泛授權 VHS
- **產品功能 vs 生態系統**：技術較優的 Betamax 敗給了錄影時間較長的 VHS
- **先行者優勢並非永遠**：第一個上市並不保證最終勝利

這個教訓後來被 Blu-ray vs HD DVD（2000s）和 Type-C 統一化等案例反覆驗證。

## 1976 年的時代脈絡

- **越戰結束**：美國科技資源從戰爭轉向商業應用
- **美國建國兩百周年**：科技樂觀主義瀰漫
- **矽谷的崛起**：從車庫公司到創投生態的初步形成
- **冷戰驅動的科技投資**：Cray-1 的國防應用和 ARPANET 的軍事背景

1976 年最值得記住的是：這一年一個車庫裡誕生的公司、一篇學術論文、一台圓柱形超級電腦，共同定義了接下來五十年的科技發展方向。
