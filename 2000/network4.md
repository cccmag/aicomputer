# Mac OS X Public Beta 的 Aqua 介面

## 從 Platinum 到 Aqua

2000 年 9 月 13 日發布的 Mac OS X Public Beta 讓世界第一次看到了 Aqua 使用者介面。這個介面與 Classic Mac OS 的 Platinum 介面截然不同——如果說 Platinum 是 2D 的、平面的、功能性的，那麼 Aqua 就是 3D 的、半透明的、感性的。

Aqua 的設計靈感來自於水（Aqua 是拉丁文中「水」的意思）。視窗有類似水滴的半透明邊緣，按鈕有凝膠般的質感，選單具有柔和的陰影效果。這個介面在當時的作業系統中是獨一無二的——Windows 2000 仍然使用傳統的平面介面，Linux 的桌面環境還不夠成熟。

## 核心元件

**Dock**：這是 Aqua 最具標誌性的元件。Dock 位於螢幕底部，用於啟動應用程式和切換視窗。它可以自動隱藏、調整大小，並提供了類似放大鏡的滑鼠懸停效果——當滑鼠在 Dock 圖示上移動時，圖示會放大，形成一種動態的互動體驗。

**視窗管理**：Aqua 將視窗關閉、最小化和最大化按鈕放在視窗左上角（而不是 Windows 的右上角），並用紅黃綠三色圓點來標示。視窗最小化時會以「精靈效果」縮小到 Dock 中。

**字體渲染**：Quartz 繪圖引擎使用 CoreGraphics 進行字體渲染，提供了遠優於 Classic Mac OS 和 Windows 的螢幕字體顯示效果。

**選單欄**：選單欄在螢幕頂部（而不是在每個視窗中），這是 Mac 的傳統設計。但在 Public Beta 中，Apple 圖示被放在正中央——這個設計後來被改為放在左側。

## 評價與影響

Public Beta 的 Aqua 介面受到了廣泛的讚譽，但也引發了一些爭議。讚譽來自於其視覺上的美感和易用性——時代雜誌將其評為 2000 年最佳設計之一。爭議則圍繞著效能——G3 和早期的 G4 Mac 運行 Aqua 時明顯吃力，透明效果和動畫導致了系統反應緩慢。
