# 歷史穿梭：2000 年的種子

## 從 PS2 到雲端遊戲

2000 年 PS2 的運算能力和網路連線能力預示了遊戲產業的未來：

**PS2 (2000)** → **Xbox Live (2002)** → **PlayStation Network (2006)** → **雲端遊戲 (2020s)**

PS2 將遊戲機從單純的娛樂裝置轉變為家庭網路中心。它的 Emoti on Engine 處理器和 DVD 播放功能，為後來的多媒體遊戲平台奠定了基礎。

## 從 USB 隨身碟到雲端儲存

2000 年第一個 USB 隨身碟的誕生，是儲存技術小型化的重要里程碑：

**ThumbDrive (2000)** → **大容量隨身碟** → **Dropbox (2007)** → **雲端儲存時代**

USB 隨身碟讓資料可以輕鬆攜帶和傳輸。但更大的意義在於它預示了快閃記憶體將取代機械式儲存——這個趨勢在 SSD 的普及中達到高潮。

## 從藍牙 1.0 到物聯網

2000 年藍牙 1.0 的個人區域網路概念，是物聯網的早期藍圖：

**藍牙 1.0 (2000)** → **藍牙 4.0 BLE (2010)** → **物聯網爆炸** → **智慧家庭**

藍牙從最初的電纜替代技術，成長為物聯網的關鍵無線傳輸協定。BLE（低功耗藍牙）的出現讓感測器可以靠一顆紐扣電池運作數年。

## 從 Mac OS X 到 iOS

2000 年 Mac OS X Public Beta 的 Aqua 介面和 Unix 核心，為 Apple 的行動作業系統奠定了基礎：

**Mac OS X (2000/2001)** → **iPhone OS / iOS (2007)** → **iPadOS (2019)**

Mac OS X 的多點觸控手勢（來自 MacBook 觸控板）、Core Animation 框架和 Unix 核心，都直接影響了 iOS 的設計。而 Aqua 的視覺語言——半透明、陰影、動畫——在 iOS 中得到了繼承和發揚。

## 從達康泡沫到 Web 2.0

2000 年的達康泡沫看似一場災難，但卻為下一波網路創新清除了道路：

**達康泡沫破滅 (2000-2002)** → **Web 2.0 (2004+)** → **社群媒體** → **共享經濟**

泡沫清除了沒有商業模式的「眼球經濟」企業，留下了真正有價值的服務。Google 的廣告模式、Amazon 的電子商務基礎設施、eBay 的市集模式，都是在泡沫後才真正證明了自己的價值。

## 2000 年的時代脈絡

- **泡沫破滅**：.com 狂熱終結，科技業進入寒冬，但基礎建設沒有白費
- **微軟的統治**：Windows 2000 鞏固了微軟在企業市場的地位
- **Apple 的重生**：Mac OS X Public Beta 預示了 Apple 即將到來的復興
- **無線的萌芽**：藍牙和 Wi-Fi 同時在 2000 年開始進入消費市場
- **寬頻的黎明**：DSL 和 Cable Modem 開始取代撥接上網

2000 年的關鍵字是「過渡」——從泡沫到理性、從 9 吋到彩色、從有線到無線、從撥接到寬頻。這一年為 21 世紀的第一個十年準備了技術和教訓。
