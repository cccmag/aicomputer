# Mac OS X Public Beta：Kodiak 的試水

## 從 Copland 失敗到 NeXT 的收購

Mac OS X 的開發歷程充滿了曲折。Apple 在 1990 年代嘗試開發下一代作業系統 Copland，但以失敗告終。1996 年，Apple 收購了 NeXT Computer——Steve Jobs 在離開 Apple 後創立的公司——並以其 OPENSTEP 作業系統為基礎開發 Mac OS X。

2000 年 9 月 13 日，Apple 發布了 Mac OS X Public Beta，代號「Kodiak」。這是以 $29.95 美元向公眾出售的預覽版本，也是 Apple 首次公開提供作業系統測試版。購買 Public Beta 的使用者之後可以獲得 $30 美元折價購買正式版 Mac OS X 10.0。

## Aqua 使用者介面

Public Beta 最重要的特色是全新的 Aqua 使用者介面。這與 Classic Mac OS 的 Platinum 介面截然不同：

**Dock**：全新的應用程式啟動和管理方式，取代了傳統的應用程式選單

**半透明效果**：視窗、選單和按鈕具有半透明和外光暈效果

**大尺寸圖示**：系統圖示比以往更大、更細緻

**新的字體渲染**：採用 Quartz 繪圖引擎，提供平滑的 PostScript 字體渲染

## 技術架構

Mac OS X 基於多層架構設計：

**Darwin 核心**：底層為開放原始碼的 Unix-like 核心，基於 Mach 微核心和 FreeBSD 元件

**Quartz 繪圖引擎**：基於 PDF 的 2D 繪圖系統，提供 WYSIWYG 的精確度

**Carbon 相容層**：讓 Classic Mac OS 的應用程式可以移植到 Mac OS X

**Classic 環境**：可以在 Mac OS X 中直接運行 Mac OS 9 的應用程式

## 意義

Mac OS X Public Beta 證明了 Apple 確實有能力推出下一代作業系統——這在 Copland 失敗和 Rhapsody 探索之後，對業界觀察家來說是一個重要的信心信號。
