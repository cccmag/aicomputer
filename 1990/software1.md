# Microsoft Windows 3.0：終於成功的圖形介面

## 前兩次的失敗

Microsoft Windows 的歷史在 1990 年之前並不光彩。Windows 1.0（1985）和 Windows 2.0（1987）都未能獲得市場的廣泛接受。Windows 1.0 的平鋪式視窗設計備受批評，Windows 2.0 雖然改進了重疊視窗，但效能問題和缺乏應用程式支援使其影響有限。

1990 年 5 月 22 日，Microsoft 發表了 Windows 3.0。這一次，微軟終於做對了。

## 成功的關鍵

Windows 3.0 的成功來自幾個關鍵因素：

- **虛擬記憶體**：利用硬碟空間模擬記憶體，讓使用者可以執行比實體記憶體更大的應用程式
- **保護模式**：使用 Intel 80386 的保護模式，允許多個 DOS 應用程式同時執行
- **改良的使用者介面**：全新的圖示、可自訂的視窗外觀、更直觀的程式管理員
- **豐富的第三方應用程式**：Microsoft 在 Windows 3.0 發表前已經確保了主要應用程式的支援，包括 Microsoft Word 和 Excel 的 Windows 版本

## 商業上的巨大成功

Windows 3.0 在兩年內賣出了超過 1,000 萬套。它終於讓 IBM PC 相容機有了能夠與 Macintosh 抗衡的圖形使用者介面。

Windows 3.0 的成功也決定了作業系統市場的走向——IBM 的 OS/2 從此失去了主導地位。Microsoft 和 IBM 的合作關係也在這一年正式破裂。

## 簡單就是美好

Windows 3.0 的設計理念可以用一句話概括：「讓圖形介面在 PC 上真正可用。」它沒有發明任何全新的概念（視窗、圖示、選單在 Mac 上已經很常見），但它將這些概念以足夠好的品質帶到了 PC 平台上。
