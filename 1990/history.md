# 歷史穿梭：1990 年的種子

## 從第一個瀏覽器到現代 Web

1990 年 Tim Berners-Lee 的 WorldWideWeb 瀏覽器開啟了現代網際網路時代：

**WorldWideWeb / Nexus (1990)** → **Line Mode Browser (1991)** → **ViolaWWW (1992)** → **Mosaic (1993)** → **Netscape Navigator (1994)** → **Internet Explorer (1995)** → **Firefox (2004)** → **Chrome (2008)**

Berners-Lee 的瀏覽器既是瀏覽器也是編輯器——他設想的全球資訊網是一個可讀可寫的媒介。但後來的瀏覽器都只實作了瀏覽功能，編輯功能被遺忘了三十年，直到 Notion、Google Docs 和區塊鏈技術才重新接近這個願景。

## Windows 3.0 到 Microsoft 帝國

Windows 3.0（1990）的成功改變了個人電腦產業的權力結構：

**Windows 1.0 (1985)** → **Windows 2.0 (1987)** → **Windows 3.0 (1990)** → **Windows 3.1 (1992)** → **Windows 95 (1995)** → **Windows XP (2001)** → **Windows 10/11**

Windows 3.0 的銷售成功決定了作業系統市場的走向。IBM 的 OS/2 在 1990 年之後一蹶不振，Microsoft 從此壟斷了 PC 作業系統市場長達三十年。Windows 3.0 的 1,000 萬套銷量證明了圖形使用者介面在 PC 上的可行性。

## Photoshop 到創意雲

Photoshop 1.0（1990）開啟了數位影像編輯的時代：

**Display (1987, Thomas Knoll)** → **Photoshop 1.0 (1990)** → **Photoshop 3.0 圖層 (1994)** → **CS 系列 (2003)** → **Creative Cloud (2013)**

Photoshop 的故事說明了創新可以來自任何地方——Thomas Knoll 最初只是為了在自己的 Mac 上顯示灰階圖像而寫了這個程式。如果沒有他和哥哥 John 的堅持，今天的影像產業可能會完全不同。

## 從 ARPANET 到 ISP

1990 年 ARPANET 除役和第一個商業 ISP 的出現，象徵著網際網路從學術工具向商業基礎設施的轉變：

**ARPANET (1969)** → **NSFNET (1986)** → **The World ISP (1990)** → **NSFNET 商業化 (1991)** → **網際網路商業爆炸 (1995-)**

Vint Cerf 為 ARPANET 寫的輓詩充滿感情，但 ARPANET 的退役實際上是一件值得慶祝的事——它意味著網際網路已經足夠成熟，不再需要政府的直接運營。

## 從 386SL 到行動計算

Intel 386SL（1990）專為筆記型電腦設計，標誌著行動計算專用晶片的開端：

**386SL (1990)** → **486SL (1992)** → **Pentium MMX (1997)** → **Pentium III-M (2000)** → **Centrino (2003)** → **Core M (2014)** → **Apple Silicon M 系列 (2020)**

386SL 的 SMM（系統管理模組）成為所有現代處理器的標準功能，讓筆記型電腦的電池續航從「一兩個小時」提升到「一整天的正常工作」。

## 搜尋引擎的黎明

1990 年的 Archie 是第一個網路搜尋引擎，開啟了資訊檢索的新時代：

**Archie (1990)** → **Gopher (1991)** → **Veronica (1992)** → **WebCrawler (1994)** → **Yahoo! (1994)** → **AltaVista (1995)** → **Google (1998)**

今天，我們已經無法想像沒有搜尋引擎的網際網路。但 1990 年的人們還需要手動記錄 FTP 伺服器的位址來尋找檔案。Archie 雖然簡單到只搜尋檔案名稱，但它證明了搜尋可以在網路上自動進行。

## 1990 年的時代脈絡

- **冷戰正式結束**：兩德統一（10 月 3 日），蘇聯解體在即
- **全球資訊網誕生**：人類最重要通訊媒介的建構元年
- **Microsoft 的崛起**：Windows 3.0 的成功讓 Microsoft 從軟體公司成長為科技巨頭
- **Macintosh 的危機**：Apple 面臨邊緣化的危險，直到 1997 年 Jobs 的回歸才扭轉局勢
