# 1990 年資訊科技年鑑：前言

1990 年是個人電腦圖形介面真正成熟的一年。Microsoft 在五月推出了 Windows 3.0，這是有史以來第一個在商業上取得巨大成功的 Windows 版本。憑藉虛擬記憶體、改良的多工處理和豐富的第三方軟體支援，Windows 3.0 讓 IBM PC 相容機終於有了能與 Macintosh 抗衡的圖形使用者介面。

在全球資訊網方面，Tim Berners-Lee 在年底完成了第一個網頁瀏覽器——他稱之為 WorldWideWeb（後來更名為 Nexus）。這個在 NeXT 電腦上開發的程式不僅是瀏覽器，同時也是編輯器，實現了他對網際網路作為讀寫媒介的原始願景。同時，HTML 和 HTTP 的基本規範也在此時起草。

影像處理領域誕生了一個傳奇：Adobe 在二月發表了 Photoshop 1.0。這套由兄弟檔 John 和 Thomas Knoll 開發的軟體，最初只是 Thomas 博士研究期間的一個 side project，後來被用在電影《無底洞》的特效場景中。Photoshop 從此成為影像編輯的代名詞。

網路基礎建設方面，ARPANET 在二月正式除役，Vinton Cerf 為它寫了一首輓詩。取而代之的是 NSFNET 和其他商用網路。同一年，第一個商業網路服務供應商 The World 在美國成立，讓一般大眾也能連上網際網路。

Intel 則推出了 386SL 處理器，這是專為筆記型電腦設計的低功耗晶片，預示著行動計算時代的來臨。
