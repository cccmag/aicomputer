# Teletype Model 33 ASR：分時時代的通用終端機

## 每一個分時使用者的視窗

1965 年，幾乎每一個連接到分時系統的使用者，面前都擺著一台 Teletype Model 33 ASR（Automatic Send-Receive，自動收發）。這台由 Teletype Corporation 生產的電傳打字機，成為 1960 年代和 1970 年代初期最普遍的電腦終端機。

Model 33 ASR 在 1963 年首次推出，但隨著 1965 年分時系統的普及才真正迎來了需求高峰。它的價格約為 700 到 1,000 美元（遠低於電腦本身的價格），採用電動打字機構造，能夠以每秒 10 個字元（100 bps）的速度列印。它使用美國標準資訊交換碼（ASCII），這是 ASCII 編碼在商業產品中的最早大規模應用之一。

## 紙帶與三種模式

Model 33 ASR 最具特色的功能是它的紙帶讀取和穿孔裝置。ASR 中的「A」代表它可以自動收發（Automatic Send-Receive），這得益於其內建的紙帶系統：

鍵盤發送模式（Keyboard Send）：使用者直接在鍵盤上打字，資料即時發送到電腦。

紙帶發送模式（Tape Send）：使用者先將資料打在紙帶上，然後將紙帶送入讀取器高速發送。這個模式允許使用者在離線時準備資料，上線時以最高速度傳送。

紙帶接收模式（Tape Receive）：電腦可以將輸出資料打在紙帶上，方便使用者離線閱讀或保存。

紙帶的寬度為 7/8 英吋，可容納 8 列孔位（對應 8 位元資料）。在使用 ASCII 時，使用其中 7 列。在傳輸速度無法跟上使用者打字速度時，紙帶充當緩衝區的角色。

## 聲耦合器：從電話到電腦

Model 33 ASR 在分時環境中的使用場景通常是這樣的：使用者拿起電話聽筒，撥打大型主機所在的電話號碼。另一端傳來高頻的載波音後，使用者將聽筒插入安裝在終端機旁邊的聲耦合器（acoustic coupler）中。聲耦合器將終端機的數位訊號轉換為音頻訊號，透過電話聽筒傳送，並從聽筒接收電腦的回應音頻。

這種透過聲耦合器連接的方式雖然聽起來原始，但在沒有專用數據專線的環境中卻非常實用。使用者可以在校園的任何有電話的地方連接到 MT 的 Project MAC 大型主機，或在全國任何角落連接到公司的資料中心。

## 文化的塑造者

Teletype Model 33 不僅是一部硬體設備，它也塑造了早期電腦文化中的許多慣例。Unix 系統中「一切皆檔案」的概念在某種程度上受到電傳打字機的影響——因為終端機本身就是一個檔案裝置。CTRL 鍵的設計源自電傳打字機的控制字元輸入方式。

Model 33 那台時大聲的列印聲——喀喀喀喀的噪音——成為了早期電腦實驗室的背景音樂。當整個實驗室裡有十幾台 Model 33 同時運作時，噪音之大讓工作人員必須提高音量才能交談。到了 1970 年代，這些噪音逐漸被 CRT 終端機的無聲運作所取代，但 Model 33 的印記已經深深刻在了第一代電腦使用者的集體記憶中。
