# 1961 年資訊科技年鑑

## 目錄

### 1. 前言
- [前言](preface.md) — 1961 年資訊科技總覽：IBM 7030 Stretch、Fairchild 商用 IC、CTSS 分時系統、Project MAC 經費批准

### 2. 硬體發展
- [hardware1: IBM 7030 Stretch](hardware1.md) — IBM 第一台電晶體化超級電腦，超過 100 項創新技術，交付洛斯阿拉莫斯國家實驗室
- [hardware2: Fairchild 680 商用 IC](hardware2.md) — 第一款商用積體電路，DCTL 正反器，平面製程量產
- [hardware3: IBM 1401 普及](hardware3.md) — 取代傳統會計機，最終銷售超過 20,000 台
- [hardware4: UNIVAC 1107](hardware4.md) — Sperry Rand 第一台電晶體化主機，採用薄膜記憶體
- [hardware5: Honeywell 800](hardware5.md) — 電晶體化主機，具備平行處理能力

### 3. 軟體發展
- [software1: CTSS 分時系統](software1.md) — Corbato 在 MIT 開發的相容分時系統，運行於 IBM 709
- [software2: COBOL-61 公佈](software2.md) — CODASYL 標準化，Grace Hopper 的 FLOW-MATIC 奠基
- [software3: LISP 1.5](software3.md) — McCarthy 的 AI 語言，具備 eval、垃圾回收、符號運算
- [software4: ALGOL 60 實作](software4.md) — Burroughs B5000 堆疊架構專為 ALGOL 設計
- [software5: IBM 1401 程式系統](software5.md) — Autocoder、SPS、FARGO 組譯器

### 4. 網路發展
- [network1: Project MAC 提案](network1.md) — MIT 獲得 ARPA 經費資助分時系統研究
- [network2: Bell Labs Dataphone 103A](network2.md) — 第一款商用數據機，300 bps 電話線傳輸
- [network3: SAGE 網路運作](network3.md) — 23 個指揮中心，數位數據經由電話線傳輸
- [network4: IBM 遠端終端系統](network4.md) — 1050 數據通訊系統用於航空公司訂位
- [network5: 早期電腦網路概念](network5.md) — Licklider 的「人機共生」論文

### 5. 歷史穿梭
- [歷史穿梭](history.md) — 1961 的種子：Stretch 啟發 CDC 6600，CTSS 導向 Multics 與 Unix，IC 催生微處理器
