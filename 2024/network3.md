# Microsoft Copilot 生態系全面擴張

## 365 Copilot 的企業部署

2024 年，Microsoft 365 Copilot 從 2023 年的預覽階段進入全面商業部署。數千家企業開始將 Copilot 整合到日常工作流程中：

- **Word**：從大綱到文件的 AI 生成，已成為許多知識工作者的標準工具
- **Excel**：自然語言查詢資料、自動產生圖表與資料模型
- **PowerPoint**：從文件到簡報的一鍵生成
- **Teams**：會議摘要、行動項目自動追蹤、即時翻譯

## Copilot+ PC

2024 年 5 月，Microsoft 提出了 **Copilot+ PC** 概念——新一代 Windows PC 的硬體標準：

- **最低 40 TOPS 的 NPU**（神經處理單元）
- 支援本機 AI 處理，減少雲端依賴
- 首批搭載 Qualcomm Snapdragon X Elite/Plus 晶片
- 特色功能包括 Recall（AI 時間回溯搜尋）、Cocreator（即時 AI 繪圖）、即時翻譯

Recall 功能引發了隱私爭議——它會定期截取螢幕畫面並建立可搜尋的資料庫。Microsoft 在壓力下增加了安全措施，將 Recall 改為 opt-in 並使用 BitLocker 加密。

## Azure AI 的雲端版圖

Microsoft Azure 在 2024 年持續擴張 AI 基礎設施：

- **Azure OpenAI Service**：提供 GPT-4o、o1 等最新模型的企業級 API
- **Nvidia Blackwell 進駐**：Azure 成為首批部署 GB200 NVL72 系統的雲端服務商
- **自研晶片**：Microsoft 發表 Maia AI 加速器與 Cobalt Arm 架構 CPU
- **AI 資料中心擴建**：全球投資數百億美元新建 AI 資料中心

## GitHub Copilot 的普及

GitHub Copilot 在 2024 年達到了一個里程碑——超過 180 萬付費用戶，被整合進 VS Code、Visual Studio、JetBrains 等主流 IDE。GitHub Copilot Workspace 的推出更讓開發者可以用自然語言規劃與執行整項開發任務。
