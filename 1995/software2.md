# JavaScript：十天創造的傳奇

## 意外的誕生

1995 年 12 月，Netscape 的 Brendan Eich 在十天之內完成了 JavaScript 的第一個實作。它的故事是這樣的：

Netscape 需要一種在瀏覽器中執行腳本的方式——伺服器端處理太慢，靜態 HTML 不足以滿足日益增長的互動需求。公司最初考慮使用 Java applet，但 Java 對於非專業程式設計師來說太複雜了。

Brendan Eich 被要求設計一個「Scheme 語言的簡化版」，但同時又要讓語法看起來像 Java（因為 Java 是當時的熱潮）。結果誕生了一種既有函數式語言特性、又帶有 C 風格語法的獨特語言。

## 命名的混亂

JavaScript 最初被稱為 Mocha，然後改名為 LiveScript。1995 年 12 月，Netscape 和 Sun 達成協議，將它命名為 JavaScript——這完全是一個市場決策，目的是借 Java 的知名度。兩者除了語法表面相似之外，核心設計完全不同。

## 最初的設計

Eich 在十天內實作了 JavaScript 的核心功能：
- 動態型別
- 原型繼承（prototype-based inheritance）
- 頭等函數（first-class functions）
- 閉包（closures，雖然最初的實作不完整）
- 物件字面語法

這些特性在 1995 年看起來很奇怪——當時的主流語言（C++、Java）都是靜態型別和類別繼承的。但正是這些設計讓 JavaScript 在後來變得如此靈活。
