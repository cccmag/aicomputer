# 歷史穿梭：1995 年的種子

## Java 的傳承：從 Oak 到 Android

Java 最初是 James Gosling 在 Sun Microsystems 於 1991 年開始的內部專案，原名 Oak，目標是消費性電子產品（機上盒、洗衣機）。1995 年的正式發表將其定位轉向網際網路 applet 平台。

**Oak (1991)** → **Java 1.0 (1996)** → **Java 2 (1998)** → **Java EE (1999)** → **Android Dalvik (2008)** → **Kotlin (2016)**

Java 的「一次編寫，到處執行」理念雖然在桌面端從未完全實現，但在伺服器端和 Android 平台上取得了巨大的成功。到 2020 年代，Java 仍然是企業級開發的主要語言之一。

## JavaScript 的意外遺產

Brendan Eich 在 1995 年 12 月花了十天時間設計了 JavaScript。最初叫 Mocha，後改名 LiveScript，最終為了搭 Java 順風車而改名 JavaScript——但兩者除了語法表面相似之外，幾乎沒有任何關係。

**JavaScript (1995)** → **AJAX (2005)** → **Node.js (2009)** → **React/Vue/Angular (2010s)** → **全端 JavaScript**

JavaScript 從一個「在瀏覽器中做表單驗證的腳本語言」，成長為全世界最廣泛使用的程式語言之一，生態系統涵蓋前端、後端、行動應用和桌面應用。

## Windows 95 的桌面革命

Windows 95 引入了「開始功能表」、工作列、系統匣（system tray）等至今仍在使用的 UI 慣例。它將 MS-DOS 和 Windows 整合為單一產品，結束了長達十年的 DOS 時代。

**Windows 3.1 (1992)** → **Windows 95 (1995)** → **Windows 98 (1998)** → **Windows XP (2001)** → **Windows 7 (2009)**

Windows 95 的開始功能表設計在 Windows 8 中被移除後，因用戶強烈反彈又在 Windows 10 中以改良形式回歸——這說明了 1995 年的設計有多深植人心。

## 從商業 ISP 到全民上網

1995 年是美國主要 ISP 商業化的元年。CompuServe、America Online (AOL)、Prodigy 等服務將上百萬家庭帶上網際網路。

**校園網路 (1980s)** → **商業 ISP (1995)** → **寬頻 ADSL (1997-2000)** → **光纖 (2010s)** → **5G (2020s)**

Netscape IPO 在 8 月 9 日 的成功（市值 29 億美元），打開了科技公司上市的大門，也預示了後來 dot-com 泡沫的高峰與破裂。

## 電子商務的起點

1995 年 7 月，Amazon.com 上線，最初只賣書。Pierre Omidyar 在同一年建立了 AuctionWeb——就是後來的 eBay。

**Amazon/eBay (1995)** → **PayPal (1998)** → **Alibaba (1999)** → **淘寶 (2003)** → **行動支付 (2010s)**

Amazon 從書店出發，二十五年後成為全球最大的雲端服務供應商（AWS），這個轉折在 1995 年是沒有人能想像的。

## 1995 年的時代脈絡

- **「世紀審判」辛普森案**：電視媒體的直播審判模式成為典範，網路新聞開始崛起
- **Windows 95 上市狂熱**：全球媒體報導的科技盛事，標誌 PC 進入消費主流
- **Netscape IPO**：網路股的第一個里程碑，點燃 dot-com 熱潮
- **Java 的誕生**：從消費電子的失敗專案變成網際網路的明星語言
