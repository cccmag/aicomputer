# Windows 95：桌面革命的巔峰

## 史上最大的軟體發布

1995 年 8 月 24 日，Microsoft 在全球发表了 Windows 95。發布活動本身就是一個文化事件——微軟花費 3 億美元進行行銷，包括使用滾石樂團的「Start Me Up」作為廣告配樂。在紐約、雪梨、倫敦等地都有盛大的首發活動，消費者排隊數小時只為了第一時間買到。

Windows 95 在上市後的前五週內賣出了 700 萬套，成為史上銷售最快的作業系統。

## 關鍵創新

Windows 95 引入了多項至今仍在使用的 UI 慣例：

- **開始功能表（Start Menu）**：作為所有應用程式的統一入口
- **工作列（Taskbar）**：顯示正在執行中的應用程式和常駐圖示
- **系統匣（System Tray）**：時鐘、音量、網路狀態等系統圖示
- **檔案總管（File Explorer）**：取代了舊的檔案管理員
- **長檔名支援**：VFAT 檔案系統允許最多 255 個字元的檔名

## 技術上的重大轉折

從技術角度看，Windows 95 是一次重大的作業系統重構：

- **32 位元架構**：核心是純 32 位元的，但保留了 16 位元的應用程式相容性
- **搶先式多工**：不同於 Windows 3.1 的合作式多工，系統可以強制分配 CPU 時間
- **即插即用（Plug and Play）**：自動偵測硬體和分配資源
- **預防保護**：每個應用程式在自己的位址空間中執行，一個當機不影響整個系統
