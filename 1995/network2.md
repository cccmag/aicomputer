# Internet Explorer 的誕生：瀏覽器戰爭揭幕

## Spyglass 授權

1995 年，Microsoft 意識到網際網路的快速成長對公司未來的威脅。Bill Gates 在 1995 年 5 月的「Internet Tidal Wave」備忘錄中，將網際網路定位為 Microsoft 的最高優先級。

Microsoft 沒有從零開發瀏覽器，而是向 Spyglass 公司取得了 Mosaic 瀏覽器的原始碼授權。Spyglass Mosaic 是 NCSA Mosaic 的一個商業化版本。

## IE 1.0

1995 年 8 月 16 日（Windows 95 上市前八天），Microsoft 發布了 Internet Explorer 1.0。它其實只是 Spyglass Mosaic 的一個改名版本，功能非常基本：

- 支援 HTML 2.0（基本的文字、圖片、表格）
- 沒有框架支援
- 沒有 JavaScript
- 沒有 CSS
- 沒有 ActiveX

IE 1.0 作為 Microsoft Plus! for Windows 95 的一部分免費提供。

## IE 2.0

1995 年 11 月 22 日，Microsoft 發布了 IE 2.0。這是一個稍微改進的版本：

- 支援 HTML 表格
- 支援 SSL
- 支援 cookies
- 支援新聞群組客戶端 (NNTP)
- 同時提供 Windows 95 和 Windows NT 版本

IE 2.0 仍然遠不如 Netscape Navigator 2.0（同時期發布）。但它是免費的——這是微軟在瀏覽器戰爭中最終勝出的根本武器。
