# Java 語言的誕生：一次編寫，到處執行

## 從 Oak 到 Java

1995 年 5 月 23 日，Sun Microsystems 在 SunWorld 大會上正式發表 Java 程式語言和 HotJava 瀏覽器。James Gosling 在 1991 年開始的內部專案 Oak——原本針對消費性電子產品設計——被重新定位為網際網路的通用程式語言。

Java 的核心設計目標是「一次編寫，到處執行」（Write Once, Run Anywhere, WORA）。實現這個目標的關鍵技術是 Java Virtual Machine（JVM）——一個虛擬化的執行環境，隱藏了底層硬體和作業系統的差異。

## 語言的設計

Java 的語法繼承自 C/C++，但做了許多簡化和改進：

- **自動記憶體管理**：Garbage Collection 取代了手動的 malloc/free
- **沒有指標**：消除了 C/C++ 中最常見的記憶體錯誤來源
- **強型別**：編譯時期型別檢查
- **例外處理**：結構化的錯誤處理機制
- **多執行緒**：內建的 Thread 類別和 synchronized 關鍵字
- **豐富的類別庫**：從資料結構到網路通訊的標準 API

## 歷史性的發布

Java 的正式發表雖然在 1995 年，但第一個穩定版本 Java 1.0 在 1996 年 1 月才推出。然而，1995 年的宣布已經引發了開發社群的巨大興趣——Netscape 在 1995 年 9 月宣布 Navigator 將支援 Java，這進一步鞏固了 Java 作為 Web 程式語言的地位。
