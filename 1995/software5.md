# MySQL：開源資料庫的開始

## Michael Widenius 的夢想

1995 年，瑞典程式設計師 Michael「Monty」Widenius 開始開發一個新的關聯式資料庫管理系統。這最初是為了他所工作的公司 TcX 內部使用——尋找一個比 mSQL 更快、更穩定的資料庫引擎。

Widenius 的女兒命名為 My，所以他將這個專案命名為 MySQL——「My」的 Structured Query Language。

## 第一個版本

MySQL 的第一個內部版本在 1995 年完成，但對外發布的版本是 MySQL 3.11，在 1996 年推出。1995 年的原始版本包含：
- 一個 SQL 解析器
- ISAM（Indexed Sequential Access Method）儲存引擎
- 基本的 SELECT、INSERT、UPDATE、DELETE 支援
- 多執行緒架構

MySQL 最初的設計哲學是「速度優先於功能」——與 PostgreSQL 的「功能優先於速度」形成鮮明對比。

## 與 LAMP 的結合

MySQL 在 1995 年只是一個不起眼的開源專案。它後來的成功主要來自於：
- **與 PHP 的完美搭配**：MySQL + PHP = 最受歡迎的 Web 開發組合
- **簡單易用**：安裝和管理比 Oracle、DB2 簡單得多
- **足夠的效能**：對大多數 Web 應用來說，MySQL 的速度足夠快
- **開源授權**：免費使用，降低了初創企業的成本

MySQL 是 LAMP 堆疊（Linux + Apache + MySQL + PHP）的核心元件之一，這個組合支撐了 1990 年代末到 2000 年代中期的絕大多數網站。
