# IBM PC BASIC 與 ROM 中的語言

## 開機就能寫程式

當 IBM PC 在 1981 年推出時，它的主機板上有一個重要的元件：40 KB 的 ROM。其中一部分是 BIOS（基本輸入輸出系統），另一部分則包含了 Microsoft BASIC 直譯器——IBM Cassette BASIC。

這意味著使用者不需要磁碟機、不需要作業系統——只要開機，電腦就會自動載入 BASIC 直譯器到記憶體，使用者立即可以開始寫程式。這對教育市場和 hobbyist 社群極具吸引力。

## IBM BASIC 的版本

IBM PC 提供了三層 BASIC：
1. **Cassette BASIC**（ROM 中）：最基本的版本，支援卡帶儲存
2. **Disk BASIC**（BASIC.COM）：加入磁碟支援
3. **Advanced BASIC**（BASICA.COM）：加入圖形、音樂、通訊功能

BASICA 支援 PC 的 CGA 繪圖模式（640×200 解析度、4 色）、繪圖命令（LINE、CIRCLE、PAINT）、聲音指令（SOUND、PLAY）、以及遊戲搖桿控制。這些功能讓 BASIC 成為早期 PC 遊戲和商業軟體的開發語言。
