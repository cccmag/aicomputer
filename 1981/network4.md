# 3Com 成立：Ethernet 的商業化

## Metcalfe 的創業

1981 年，Robert Metcalfe——Ethernet 的共同發明者——離開了 Xerox PARC，創立了 3Com Corporation。公司名稱來自「Computer Communication Compatibility」。

3Com 的使命是將 Ethernet 技術商業化。當時 Ethernet 還只是在研究實驗室中使用的網路技術，3Com 需要將它轉化為可供企業購買和安裝的產品。

## 第一張 Ethernet 介面卡

3Com 的第一個產品是 3C100——適用於 Multibus 系統的 Ethernet 控制器，隨後推出了適用於 IBM PC 的 Ethernet 卡。這些網路卡讓 PC 可以連接 Ethernet 區域網路，實現檔案共享、印表機共享和電子郵件通訊。

3Com 也銷售 EtherSeries 軟體——一套基於 Ethernet 的網路作業系統，支援檔案和印表機共享。3Com 在 1980 年代成長為世界上最大的網路設備公司之一，直到 1990 年代末期被收購。
