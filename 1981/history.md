# 歷史穿梭：1981 年的種子

## IBM PC 的開放遺產

IBM PC 的開放架構決策，是個人電腦產業最重要的分水嶺：

**IBM PC 5150 (1981)** → **Compaq Portable (1982, 相容機)** → **IBM PC/XT (1983)** → **IBM PC/AT (1984)** → **PS/2 (1987, IBM 嘗試收回控制)** → **Wintel 聯盟統治**

IBM 原本希望透過開放架構快速佔領市場，但失去對平台的控制權導致 IBM 最終在 2005 年將 PC 業務賣給聯想。然而，PC 相容的生態系——由 Intel 處理器和 Microsoft Windows 驅動——主導了個人電腦 20 年。

## 從 GUI 到現代桌面

Xerox Star 雖然商業失敗，但它的設計理念啟發了整個個人電腦產業：

**Xerox Alto (1973)** → **Xerox Star (1981)** → **Apple Lisa (1983)** → **Apple Macintosh (1984)** → **Microsoft Windows (1985)** → **現代桌面環境**

Star 建立的「桌面隱喻」——圖示、視窗、滑鼠、拖曳——仍然是今天作業系統的核心互動模式。有趣的是，Xerox PARC 的 GUI 概念來自 Engelbart（1968）、Engelbart 的概念又來自 Bush 的 Memex（1945）。

## 從 SMTP 到全球電子郵件

SMTP 的去中心化設計是電子郵件成功的關鍵：

**RFC 788 SMTP (1981)** → **RFC 821 SMTP 更新 (1982)** → **MIME 擴充 (1992)** → **Webmail (1996, Hotmail)** → **Gmail (2004)**

SMTP 簡單的架構讓任何人都可以運行郵件伺服器，不需要中央權威機構。這種開放設計促成了電子郵件的全球普及——直到 Spam 問題迫使 SMTP 增加了 SPF、DKIM 等驗證機制。

## 從 BITNET 到社交網路

BITNET 和最早的 LISTSERV 預示了後來的線上社群：

**BITNET Listserv (1981)** → **Usenet Newsgroups (1980)** → **Mailing Lists (1990s)** → **Web Forums (1990s)** → **Facebook Groups (2004)** → **Discord/Slack (2015)**

LISTSERV 是第一個自動化的郵件列表管理系統——它讓學術社群可以圍繞特定主題建立討論群組。這個概念——一個主題、一群人、自動化的訊息分發——是所有現代社交網路的原型。

## 1981 年的時代脈絡

- **新冷戰緊張**：雷根總統就職（1981 年 1 月），美蘇對抗加劇，ARPANET 等國防相關技術獲得更充足的預算
- **英國的運算教育**：Sinclair ZX80/81 的低價策略讓一整代英國孩子學會了程式設計——英國後來成為遊戲開發強國與此密切相關
- **日本挑戰**：日本在 1981 年已成為世界第二大經濟體，日本半導體產業對美國構成巨大威脅
- **專家系統的夏天**：AI 領域的專家系統（Expert Systems）在 1981 年達到熱潮高峰，XCON（DEC 的電腦配置系統）每年節省數千萬美元
